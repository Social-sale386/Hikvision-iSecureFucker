# 按条件查询人脸分组

> 接口说明
> a)根据查询条件，查询人脸分组集合。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/face/group`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/face/group
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCodes`|`body`|`string[]`|否|通过唯一标识集合查询指定的人脸分组集合|
|`name`|`body`|`string`|否|人脸分组名称模糊查询|

## 请求参数示例
```json
{
  "indexCodes": [
    "5dc82633-a4cb-4107-b55e-f21bbdf952f9"
  ],
  "name": "仓库值守"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object[]`|否|人脸分组集合|
|`+indexCode`|`body`|`string`|否|人脸分组的唯一标识,由智能监控应用服务生成|
|`+name`|`body`|`string`|否|人脸分组的名称,人脸分组的名称是唯一的|
|`+description`|`body`|`string`|否|人脸分组的描述|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": [
    {
      "indexCode": "5dc82633-a4cb-4107-b55e-f21bbdf952f9",
      "name": "仓库值守人员",
      "description": "仓库值守人员是指守着仓库的人"
    }
  ]
}
```
