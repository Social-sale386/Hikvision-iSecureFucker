# 单个添加人脸分组

> 接口说明
> a)一次性添加一个人脸分组，返回结果为添加成功的人脸分组。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/face/group/single/addition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/face/group/single/addition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`name`|`body`|`string`|否|人脸分组的名称，1~32个字符；不能包含 ’ / \ : * ? " < >|
|`description`|`body`|`string`|否|分组的描述，1~128个字符|

## 请求参数示例
```json
{
  "name": "仓库值守人员",
  "description": "仓库值守人员是指守着仓库的人"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述,描述错误码的原因|
|`data`|`body`|`object`|否|人脸分组|
|`+indexCode`|`body`|`string`|否|人脸分组的唯一标识,由智能监控应用服务生成|
|`+name`|`body`|`string`|否|人脸分组的名称,人脸分组的名称是唯一的|
|`+description`|`body`|`string`|否|人脸分组的描述|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": {
    "indexCode": "5dc82633-a4cb-4107-b55e-f21bbdf952f9",
    "name": "仓库值守人员",
    "description": "仓库值守人员是指守着仓库的人"
  }
}
```
