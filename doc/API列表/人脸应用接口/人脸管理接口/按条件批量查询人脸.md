# 按条件批量查询人脸

> 接口说明
> a)根据查询条件，批量查询人脸信息，一次性最多查询1000条人脸。
> b)查询条件之间的关系为与，即所有条件同时生效。
> c)可以通过传入faceGroupIndexCode，查询指定人脸分组下的人脸。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/face`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/face
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`certificateNum`|`body`|`string`|否|人脸的证件号码模糊查询|
|`certificateType`|`body`|`string`|否|人脸的证件类型搜索|
|`faceGroupIndexCode`|`body`|`string`|否|根据人脸所属的分组搜索该分组下符合条件的人脸，可从按条件查询人脸分组接口返回报文中的indexCode字段|
|`indexCodes`|`body`|`string[]`|否|通过人脸的唯一标识集合查询指定的人脸集合|
|`name`|`body`|`string`|否|人脸名称模糊查询|
|`pageNo`|`body`|`integer`|否|分页查询条件，页码，为空时，等价于1，页码不能小于1或大于1000|
|`pageSize`|`body`|`integer`|否|分页查询条件，页尺，为空时，等价于1000，页尺不能小于1或大于1000|

## 请求参数示例
```json
{
  "certificateNum": "4202041996051",
  "certificateType": "111",
  "faceGroupIndexCode": "5dc82633-a4cb-4107-b55e-f21bbdf952f9",
  "indexCodes": [
    "5dc82633-a4cb-4107-b55e-f21bbdf952f9"
  ],
  "name": "张",
  "pageNo": 1,
  "pageSize": 20
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|查询到的人脸分页对象|
|`+total`|`body`|`number`|否|结果总数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+list`|`body`|`object[]`|否|人脸|
|`++indexCode`|`body`|`string`|否|人脸的唯一标识|
|`++faceGroupIndexCode`|`body`|`string`|否|人脸所属的人脸分组的唯一标识|
|`++faceInfo`|`body`|`object`|否|人脸信息|
|`+++name`|`body`|`string`|否|人脸的名称,1~32个字符；不能包含 ’ / \ : * ? " < >|
|`+++certificateType`|`body`|`string`|否|人脸的证件类别信息|
|`+++certificateNum`|`body`|`string`|否|人脸的证件号码信息。1~20个数字、字母|
|`++facePic`|`body`|`object`|否|人脸的图片信息|
|`+++faceUrl`|`body`|`string`|否|人脸图片的URL，查询返回时，为绝对地址|
