# 单个查询陌生人识别计划详情

> 接口说明
> a)根据陌生人识别计划的唯一标识，查询单个陌生人识别计划的下发状态和结果详情。
> b)返回的结果中包含了识别计划的所有信息和下发状态。还包含了人脸分组、人脸、通道布防的结果，其中人脸的下发结果由于过多，会以分页的方式返回。
> c)调用方可以通过定时调用该接口，获取识别计划的下发情况和具体每张人脸的下发结果。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/plan/recognition/white/detail`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/plan/recognition/white/detail
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCode`|`body`|`string`|是|识别计划的唯一标识，可从查询陌生人识别计划接口返回报文中的indexCode字段|
|`pageNo`|`body`|`integer`|否|分页查询条件，页码，为空时，等价于1。针对下发详情中的人脸条目生效|
|`pageSize`|`body`|`integer`|否|分页查询条件，页尺，为空时，等价于1000。针对下发详情中的人脸条目生效|

## 请求参数示例
```json
{
  "indexCode": "7cc0aCC2-a3c3-48fd-b432-718103e85c28",
  "pageNo": 1,
  "pageSize": 20
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|识别计划下发详情|
|`+indexCode`|`body`|`string`|否|识别计划的唯一标|
|`+name`|`body`|`string`|否|识别计划的名称|
|`+recognitionPlanType`|`body`|`string`|否|识别计划的类型， BLACK_LIST-重点人员识别计划 WHITE_LIST-陌生人识别计划|
|`+startTime`|`body`|`string`|否|识别计划开始下发的识别计划开始下发的时间,遵循ISO8601的时间格式|
|`+usingTime`|`body`|`number`|否|识别计划下发已经用的时间,单位:秒|
|`+status`|`body`|`string`|否|识别计划的状态，详见[附录A.33 识别计划状态类型](/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录A.33 识别计划状态类型)|
|`+faceGroupIssuedResults`|`body`|`object[]`|否|每个人脸分组的下发结果集合|
|`++indexCode`|`body`|`string`|否|唯一标识|
|`++dataType`|`body`|`string`|否|下发项的类型, FACE-人脸 FACE_GROUP-人脸分组 CHANNEL-监控点通道|
|`++errorCode`|`body`|`string`|否|下发项的结果|
|`+cameraControlResults`|`body`|`object[]`|否|每个通道的布防结果集合|
|`++indexCode`|`body`|`string`|否|唯一标识|
|`++dataType`|`body`|`string`|否|下发项的类型, FACE-人脸 FACE_GROUP-人脸分组 CHANNEL-监控点通道|
|`++errorCode`|`body`|`string`|否|下发项的结果|
|`+faceIssuedResultPage`|`body`|`object`|否|每张人脸的下发情况分页结果|
|`++total`|`body`|`number`|否|结果总数|
|`++pageNo`|`body`|`number`|否|当前页数|
|`++pageSize`|`body`|`number`|否|每页结果数|
|`++list`|`body`|`object[]`|否|下发项详情|
|`+++indexCode`|`body`|`string`|否|唯一标识|
|`+++dataType`|`body`|`string`|否|下发项的类型, FACE-人脸 FACE_GROUP-人脸分组 CHANNEL-监控点通道|
|`+++errorCode`|`body`|`string`|否|下发项的结果|
