# 批量添加人脸

> 接口说明
> a)本接口只支持将安保基础数据中人员的人脸图片导入到智能监控的人脸分组中。
> b)一次性添加一批人脸，添加结果由接口同步返回成功还是失败。
> c)一次最多导入1000个人脸。
> d)若添加的人脸对应的人脸分组已经配置有识别计划，则新添加的人脸会被一并下发到设备上。
> e)入参和单个添加人脸的接口不相同，批量接口只需要传入安保基础数据人员indexCode和要加入的人脸分组indexCode。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/face/batch/addition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/face/batch/addition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`Array[0]`|`body`|`object`|是|添加的人脸的集合|
|`+indexCode`|`Body`|`string`|是|人脸唯一标识，单次批量添加操作，不允许重复。范围1~1000|
|`+faceGroupIndexCode`|`Body`|`string`|是|指定人脸添加到的人脸分组的唯一标识, 可从按条件查询人脸分组接口返回报文中的indexCode字段|

## 请求参数示例
```json
[
  {
    "indexCode": "5dc82633-a4cb-4107-b55e-f21bbdf952f9",
    "faceGroupIndexCode": "5dc82633-a4cb-4107-b55e-f21bbdf952f6"
  }
]
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`boolean`|否|添加是否成功,true：成功,false：失败|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success.",
  "data": true
}
```
