# 人脸评分

> 接口说明
> a) 传入一张人脸图片，检测人脸图片是否合格，注意，人脸评分是检测人脸质量的辅助手段，只有人脸评分失败（返回错误码0x1f913016）时才认为是人脸图片不合格，其它情况如调用超时等情况，图片只是未检测，不能作为人脸不可用的依据。
> b) 注：此接口需要额外授权
> c)图片格式仅支持JGP，图片大小为128B到4M。
> d)当返回字段对应的值为空时，该字段不返回。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/frs/v1/face/picture/check`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/face/picture/check
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`facePicBinaryData`|`body`|`string`|否|人脸图片的二进制数据经过|
|`facePicUrl`|`body`|`string`|否|人脸图片url ，要求 URL 可以通过 GET 方式直接下载，该|

补充说明:
- Base64 编码后的字符串，该参数与 f acePicUrl 至少有一个
- 存在，都存在时优先使用此参数
- 参数与 facePicBinaryData 至少有一个存在

## 请求参数示例
```json
{
  "facePicUrl": "http://10.21.22.115:6120/pic?7dd43=sb6-z67194163f7=11i5m*epdt=*4pdi=*1s=i1b4i6d03*918a05cf2-88d032--6733588i273*e7ic5=s"
}
```

|`+++yaw`|`body`|`number`|否|平面外左右偏转角，范围：|
|`+++roll`|`body`|`number`|否|平面内旋转角，范围：|

|`+++colorConfidence`|`body`|`number`|否|彩色置信度,范围：[0,1]|
|`+++eyeDistance`|`body`|`number`|否|两眼间距,真实像素值|
|`+++grayMean`|`body`|`number`|否|灰阶均值,范围：[0,255]|
|`+++visibleScore`|`body`|`number`|否|可见度评分,范围：[0,1]|
|`++targetModelData`|`body`|`string`|否|目标数据模型|
|`++faceRect`|`body`|`object`|否|图片中人脸框区域|
|`+++height`|`body`|`number`|否|高度|
|`+++width`|`body`|`number`|否|宽度|
|`+++x`|`body`|`number`|否|X坐标|
|`+++y`|`body`|`number`|否|Y坐标|
|`++recommendFaceRect`|`body`|`object`|否|人脸区域推荐位置|
|`+++height`|`body`|`number`|否|高度|
|`+++width`|`body`|`number`|否|宽度|
|`+++x`|`body`|`number`|否|X坐标|
|`+++y`|`body`|`number`|否|Y坐标|
|`++faceMark`|`body`|`object`|否|图片中人脸特征位置|
|`+++leftEye`|`body`|`object`|否|左眼位置|
|`++++x`|`body`|`number`|否|X坐标|
|`++++y`|`body`|`number`|否|Y坐标|

|`++++x`|`body`|`number`|否|X坐标|
|`++++y`|`body`|`number`|否|Y坐标|
|`+++noseTip`|`body`|`object`|否|鼻尖位置|
|`++++x`|`body`|`number`|否|X坐标|
|`++++y`|`body`|`number`|否|Y坐标|
|`+++leftMouth`|`body`|`object`|否|左嘴角位置|
|`++++x`|`body`|`number`|否|X坐标|
|`++++y`|`body`|`number`|否|Y坐标|
|`+++rightMouth`|`body`|`object`|否|右嘴角位置|
|`++++x`|`body`|`number`|否|X坐标|
|`++++y`|`body`|`number`|否|Y坐标|
|`++faceScore`|`body`|`number`|否|人脸评分，范围： 0 100|

补充说明:
- false-评分不
- 合格
- 0x1f902300-人脸检测错误（图片格式不符合要求、检测不到人脸等情况时）
- 0x1f902301-人脸检测超时
- 0x1f902303-图片两眼间距过小
- 0x1f902304-图片彩色置信度过低
- 0x1f902305-图片人脸角度过大
- 0x1f902306-图片清晰度过低
- 0x1f902307-图片过曝或过暗（灰阶值不符合要求）
- 0x1f902308-图片遮挡严重
- 0x1f902309-图片分数过低
- 1-100
- 信息，用于人脸裁剪
- [-90,90]
- [-90,90]
- [-90,90]
