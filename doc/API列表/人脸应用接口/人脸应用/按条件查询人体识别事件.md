# 按条件查询人体识别事件

> 接口说明
> a)按条件查询人体识别事件。
> b)检索出来的结果是分页的集合，分页参数由接口参数传入。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/frs/v1/event/humanBody/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/event/humanBody/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|否|分页查询条件，页码，为空时，等价于1，页码不能小于1|
|`pageSize`|`body`|`integer`|否|分页查询条件，页尺，为空时，等价于1000，页尺不能小于1或大于1000|
|`startTime`|`body`|`string`|是|指定查询的开始时间，要求遵守ISO8601标准，且必须在endTime之前|
|`endTime`|`body`|`string`|是|指定查询的结束时间，要求遵守ISO8601标准，且必须在startTime之后|
|`cameraIndexCodes`|`body`|`string[]`|否|查询的监控点集合，为空则默认查询所有的监控点|
|`jacketTypes`|`body`|`string[]`|否|查询上衣类型，见数据字典上衣类型类别|
|`trousersTypes`|`body`|`string[]`|否|查询下衣类型，见数据字典下衣类型类别|
|`jacketColors`|`body`|`string[]`|否|查询上衣颜色，见数据字典上衣颜色类别|
|`trousersColors`|`body`|`string[]`|否|查询下衣颜色，见数据字典下衣颜色类别|
|`ride`|`body`|`string`|否|查询是否骑车，见数据字典是否骑车类别|
|`things`|`body`|`string`|否|查询是否拎东西，见数据字典是否拎东西类别|
|`bag`|`body`|`string`|否|查询是否背包，见数据字典是否背包类别|
|`hat`|`body`|`string`|否|查询是否戴帽子，见数据字典是否戴帽子类别|
|`mask`|`body`|`string`|否|查询是否戴口罩，见数据字典是否戴口罩类别|

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 –成功，其他- 失败|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|结果总数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+list`|`body`|`object[]`|否|查询结果集|
|`++eventTime`|`body`|`string`|否|事件时间|
|`++snapUrl`|`body`|`string`|否|抓拍图|
|`++bkgUrl`|`body`|`string`|否|背景图|
|`++jacketType`|`body`|`string`|否|上衣类型|
|`++trousersType`|`body`|`string`|否|下衣类型|
|`++jacketColor`|`body`|`string`|否|上衣颜色|
|`++trousersColor`|`body`|`string`|否|下衣颜色|
|`++ride`|`body`|`string`|否|是否骑车|
|`++things`|`body`|`string`|否|是否拎东西|
|`++bag`|`body`|`string`|否|是否背包|
|`++hat`|`body`|`string`|否|是否戴帽子|
|`++mask`|`body`|`string`|否|是否戴口罩|
|`++cameraIndexcode`|`body`|`string`|否|监控点编号|
|`++cameraName`|`body`|`string`|否|监控点名称|
