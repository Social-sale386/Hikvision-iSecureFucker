# 人脸图片1V1比对

> 接口说明
> a)将两张人脸图片进行比对，并得到两者的相似度。
> b)应用举例：将抓拍图片同身份证件上的人脸照片进行比对，以确认是否为同一个人。
> c)图片格式仅支持JGP，图片大小为128B到4M。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/application/oneToOne`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/application/oneToOne
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`srcFacePicBinaryData`|`body`|`string`|否|用于比较的人脸采集图片的二进制数据经过Base64编码后的字符串，该参数与srcFacePicUrl至少有一个存在，都存在时优先使用srcFacePicBinaryData。|
|`srcFacePicUrl`|`body`|`string`|否|用于比较的人脸采集图片，要求URL可以通过GET方式直接下载，该参数与srcFacePicBinaryData至少有一个存在，都存在时优先使用srcFacePicBinaryData。|
|`distFacePicBinaryData`|`body`|`string`|否|用于比较的人脸图片的二进制数据经过Base64编码后的字符串，该参数与distFacePicUrl至少有一个存在，都存在时优先使用|
|`distFacePicUrl`|`body`|`string`|否|用于比较的人脸图片，要求URL可以通过GET方式直接下载，该参数与distFacePicBinaryData至少有一个存在，都存在时优先使用distFacePicBinaryData。|

## 请求参数示例
```json
{
  "srcFacePicBinaryData": "45ew48r4345142f65348",
  "srcFacePicUrl": "http://10.21.22.115:6120/pic?7dd43=sb6-z67194163f7=11i5m*epdt=*4pdi=*1s=i1b4i6d03*918a05cf2-88d032--6733588i273*e7ic5=",
  "distFacePicBinaryData": "14f5214385t64f5q53",
  "distFacePicUrl": "http://10.21.22.115:6120/pic?7dd43=sb6-z67194163f7=11i5m*epdt=*4pdi=*1s=i1b4i6d03*918a05cf2-88d032--6733588i273*e7ic5="
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|1V1人脸图片比对结果|
|`+similarity`|`body`|`string`|否|两张人脸图片的相似度值|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success.",
  "data": {
    "similarity": "80"
  }
}
```
