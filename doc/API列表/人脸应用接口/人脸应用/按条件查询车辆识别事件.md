# 按条件查询车辆识别事件

> 接口说明
> a)按条件查询车辆识别事件。
> b)检索出来的结果是分页的集合，分页参数由接口参数传入。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1及以上版本|
|接口地址|`/api/frs/v1/event/vehicle/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/event/vehicle/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCodes`|`body`|`string[]`|否|查询的监控点集合，为空则默认查询所有的监控点|
|`plateNo`|`body`|`string`|否|车牌号|
|`vehicleColors`|`body`|`string[]`|否|车身颜色,见数据字典车身颜色|
|`brands`|`body`|`string[]`|否|车辆品牌|
|`startTime`|`body`|`string`|是|指定查询的开始时间，要求遵守ISO8601标准，且必须在endTime之前|
|`vehicleTypes`|`body`|`string[]`|否|车辆类型,见数据字典车辆类型|
|`endTime`|`body`|`string`|是|指定查询的结束时间，要求遵守ISO8601标准，且必须在startTime之后|
|`plateColors`|`body`|`string[]`|否|车牌颜色,见数据字典车牌颜色|
|`pageNo`|`body`|`integer`|否|页码|
|`pageSize`|`body`|`integer`|否|页尺|

```json
{
  "cameraIndexCodes": [

    "red"

    "1026"

  "vehicleTypes": [
    "vehicle"

    "red"

  "pageSize": 10
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 –成功，其他- 失败|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|结果总数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+list`|`body`|`object[]`|否|查询结果集|
|`++snapUrl`|`body`|`string`|否|抓拍图|
|`++cameraIndexcode`|`body`|`string`|否|监控点编号|
|`++cameraName`|`body`|`string`|否|监控点名称|
|`++eventTime`|`body`|`string`|否|事件时间|
|`++bkgUrl`|`body`|`string`|否|背景图|
|`++plateNo`|`body`|`string`|否|车牌号|
|`++vehicleColor`|`body`|`string`|否|车身颜色|
|`++plateColor`|`body`|`string`|否|车牌颜色|
|`++vehicleType`|`body`|`string`|否|车辆类型|
|`++brand`|`body`|`string`|否|车辆品牌|
