# 按条件查询重点目标事件

> 接口说明
> a)按条件查询重点目标事件。
> b)检索出来的结果是分页的集合，分页参数由接口参数传入。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/frs/v1/event/black/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/event/black/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|否|分页查询条件，页码，为空时，等价于1，页码不能小于1|
|`pageSize`|`body`|`integer`|否|分页查询条件，页尺，为空时，等价于1000，页尺不能小于1或大于1000|
|`startTime`|`body`|`string`|是|指定查询的开始时间，要求遵守ISO8601标准，yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：|
|`endTime`|`body`|`string`|是|指定查询的结束时间，要求遵守ISO8601标准，yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：|
|`cameraIndexCodes`|`body`|`string[]`|否|监控点唯一标志集合，若不指定，则查询所有的监控点|
|`similarity`|`body`|`integer`|是|指定查询的最小相似度, 最小为1，最大为100|
|`name`|`body`|`string`|否|指定查询的人员姓名|
|`certificateType`|`body`|`string`|否|指定查询的证件类型，ID-身份证，otherId-其他证件|
|`certificateNumber`|`body`|`string`|否|指定查询的证件号码|
|`faceGroupIndexcodes`|`body`|`string[]`|否|指定查询的所属人脸分组集合，不指定则查询所有的人脸分组|
|`orderType`|`body`|`string`|否|排序字段。结果查询排序字段，0时间，1相似度，排序支持倒序排序，默认按照时间倒叙排序|

补充说明:
- 2018-07-26T15:00:00+08:00，
- 且必须在endTime之前
- 2018-07-26T15:00:00+08:00，
- 且必须在startTime之后

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 –成功，其他- 失败|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|结果总数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+list`|`body`|`object[]`|否|查询结果集|
|`++snapInfo`|`body`|`object`|否|重点人员事件抓拍信息|
|`+++bkgUrl`|`body`|`string`|否|抓拍背景图|
|`+++snapUrl`|`body`|`string`|否|抓拍小图|
|`+++deviceIndexcode`|`body`|`string`|否|设备唯一标识|
|`+++deviceName`|`body`|`string`|否|设备名称|
|`+++cameraIndexcode`|`body`|`string`|否|监控点唯一标识|
|`+++cameraName`|`body`|`string`|否|监控点名称|
|`+++eventTime`|`body`|`string`|否|事件时间,遵守ISO8601标准，yyyy-MM-ddTHH:mm:ss.SSS+当前时区，例如北京时间：|
|`++targetInfoList`|`body`|`object[]`|否|重点人员事件目标图信息集|
|`+++name`|`body`|`string`|否|姓名|
|`+++similarity`|`body`|`string`|否|相似度|
|`+++certificateType`|`body`|`string`|否|证件类型，111-身份证，OTHER-其他证件|
|`+++certificateNumber`|`body`|`string`|否|证件号|
|`+++faceInfoIndexcode`|`body`|`string`|否|人脸唯一标识|
|`+++faceGroupIndexcode`|`body`|`string`|否|人脸分组唯一标识|
|`+++faceGroupName`|`body`|`string`|否|人脸分组名称|
|`+++faceUrl`|`body`|`string`|否|人脸图片url|

补充说明:
- 2018-07-26T15:00:00.000+08:00
