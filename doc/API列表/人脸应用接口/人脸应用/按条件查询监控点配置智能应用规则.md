# 按条件查询监控点配置智能应用规则

> 接口说明
> a)按条件查询已配置的监控点的智能应用规则。
> b)检索出来的结果是分页的集合，分页参数由接口参数传入。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/frs/v1/behavior/camera/event/query`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/behavior/camera/event/query
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`resourceIndexCodes`|`body`|`string[]`|否|服务器编号|
|`cameraIndexCodes`|`body`|`string[]`|否|监控点编号集合|
|`eventTypes`|`body`|`string[]`|否|事件类型集合|
|`pageNo`|`body`|`integer`|是|分页查询条件，页码，为空时，等价于1，页码不能小于1|
|`pageSize`|`body`|`integer`|是|分页查询条件，页尺，为空时，等价于1000，页尺不能小于1或大于1000|

## 请求参数示例
```json
{
  "cameraIndexCodes": [
    "9b2c0347a2cf43519b7de521c3630b01"
  ],
  "eventTypes": [
    "fielddetection",
    "regionEntrance"
  ],
  "pageNo": 1,
  "pageSize": 1000
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 –成功，其他- 失败|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|结果总数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|过滤|
|`+list`|`body`|`object[]`|否|查询结果集|
|`++cameraIndexCode`|`body`|`string`|否|监控点编号|
|`++resourceIndexCode`|`body`|`string`|否|服务器编号|
|`++eventType`|`body`|`string`|否|事件类型|
|`++eventParamsInfoDtos`|`body`|`object[]`|否|事件规则配置参数|
|`+++ruleCustomName`|`body`|`string`|否|规则名称|
|`+++eventParamsInfoJson`|`body`|`string`|否|透传参数|
|`++shieldRegions`|`body`|`object[]`|否|屏蔽区域集合|
|`+++polygon`|`body`|`object[]`|否|屏蔽区域坐标框|
|`++++x`|`body`|`number`|否|x坐标|
|`++++y`|`body`|`number`|否|y坐标|
|`++eventDescription`|`body`|`string`|否|事件名称|
|`++picUrl`|`body`|`string`|否|图片url|

补充说明:
- 每页记录数
