# 以图搜图

> 接口说明
> a)通过人脸图片和检索条件，检索超脑、脸谱抓拍库或智能应用服务器中符合条件的记录。
> b)以图搜图的前提是智能应用服务配置有或者曾经配置过识别计划。
> c)如果要通过cameraIndexCode搜索，请保证搜索的通道有配置或配置过超脑、脸谱或智能应用服务器的识别计划。
> d)检索出来的结果是分页的集合，为识别计划所关联的每个设备上的符合条件的前100条数据的集合

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/application/captureSearch`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/application/captureSearch
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|否|分页查询条件，页码，为空时，等价于1，页码不能小于1|
|`pageSize`|`body`|`integer`|否|分页查询条件，页尺，为空时，等价于1000，页尺不能小于1或大于1000|
|`facePicBinaryData`|`body`|`string`|否|用于搜图的图片的二进制数据经过Base64编码后的字符串，该参数与facePicUrl至少有一个存在，都存在时优先使用facePicBinaryData。|
|`facePicUrl`|`body`|`string`|否|用于搜图的图片的URL，要求URL可以通过GET方式直接下载，该参数与facePicBinaryData至少有一个存在，都存在时优先使用facePicBinaryData。|
|`cameraIndexCodes`|`body`|`string[]`|否|抓拍到人脸的抓拍机的唯一标识，若为空，则搜索全部的抓拍机|
|`searchNum`|`body`|`integer`|否|指定单个识别资源最大搜索张数，最大值为100，为空时，等价于100|
|`startTime`|`body`|`string`|是|指定搜图的开始时间，要求遵守ISO8601标准|
|`endTime`|`body`|`string`|是|指定搜图的结束时间，要求遵守ISO8601标准，且必须在startTime之后|
|`minSimilarity`|`body`|`integer`|是|指定搜图的最小相似度，最小为1，最大为100|
|`maxSimilarity`|`body`|`integer`|否|指定搜图的最大相似度，最小为1，最大为100|

补充说明:
- 建议：
- 1. 正面免冠照，露出眼睛和眉毛；
- 2. 图片文件支持jpg 格式；
- 3. 图片文件大小10 KB~200 KB。
- 建议：
- 1. 正面免冠照，露出眼睛和眉毛；
- 2. 图片文件支持jpg 格式；
- 3. 图片文件大小10 KB~200 KB。

```json
{
  "pageNo": 1,
  "pageSize": 20,
  "facePicBinaryData": "string",
  "facePicUrl": "http://10.21.22.115:6120/pic?7dd43=sb6-z67194163f7=11i5m*epdt=*4pdi=*1s=i1b4i6d03*918a05cf2-88d032--6733588i273*e7ic5=",
  "cameraIndexCodes": [
    "7cc0aCC2-a3c3-48fd-b432-718103e85c28"
  ],
  "searchNum": 50,
  "startTime": "2004-05-03T17:30:08.000+08:00",
  "endTime": "2004-06-03T17:30:08.000+08:00",
  "minSimilarity": 50,
  "maxSimilarity": 90
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|以图搜图结果分页信息|
|`+total`|`body`|`number`|否|结果总数|
|`+pageNo`|`body`|`number`|否|当前页数|
|`+pageSize`|`body`|`number`|否|每页结果数|
|`+list`|`body`|`object[]`|否|人脸检索结果|
|`++cameraIndexCode`|`body`|`string`|否|抓拍到人脸的通道的唯一标识|
|`++captureTime`|`body`|`string`|否|抓拍到人脸的绝对时标，遵守ISO8601标准|
|`++similarity`|`body`|`string`|否|抓拍到的人脸和上传人脸的相似度|
|`++bkgPicUrl`|`body`|`string`|否|抓拍到的人脸的背景图片|
|`++facePicUrl`|`body`|`string`|否|抓拍到的人脸的人脸图片|
|`++rect`|`body`|`string`|否|超脑抓拍图的坐标，rect里面四个数字，依次是height，width，x，y，可用于人脸抠图。|
