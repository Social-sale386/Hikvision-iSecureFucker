# 按条件查询识别资源

> 接口说明
> a)根据条件，查询识别资源。
> b)该接口和识别计划配置有关，识别计划配置时需要选择识别资源，通过该接口查询。
> c)查询接口之间的关系为与，若不传任何条件，为查询全部的识别资源。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/frs/v1/resource/recognition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/frs/v1/resource/recognition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCodes`|`body`|`string[]`|否|通过indexCode集合查询指定的识别资源集合|
|`name`|`body`|`string`|否|识别资源名称模糊查询|
|`recognitionResourceType`|`body`|`string`|否|根据识别资源的类型搜索，SUPER_BRAIN 超脑，FACE_RECOGNITION_SERVER 脸谱，COMPARISON 海康深眸|

## 请求参数示例
```json
{
  "indexCodes": [
    "string"
  ],
  "name": "脸谱",
  "recognitionResourceType": "COMPARISON"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.5 智能监控错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object[]`|否|识别资源的集合|
|`+indexCode`|`body`|`string`|否|识别计划的唯一标识|
|`+name`|`body`|`string`|否|识别计划的名称|
|`+recognitionResourceType`|`body`|`string`|否|识别计划的类型，SUPER_BRAIN 超脑，FACE_RECOGNITION_SERVER 脸谱，COMPARISON 海康深眸|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success.",
  "data": [
    {
      "indexCode": "7cc0aCC2-a3c3-48fd-b432-718103e85c28",
      "name": "这是一台海康深眸",
      "recognitionResourceType": "COMPARISON"
    }
  ]
}
```
