# 查询测温记录

> 接口说明
> 支持分页查询接收到的测温记录信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/gptas/v1/alarmInfo/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/gptas/v1/alarmInfo/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageSize`|`body`|`integer`|是|分页大小；0<pageSize≤1000|
|`pageNo`|`body`|`integer`|是|当前页码；pageNo≥1|
|`startTime`|`body`|`string`|是|开始时间（事件开始时间，采用ISO8601时间格式，与endTime配对使用，不能单独使用，时间范围最大不能超过3个月）|
|`endTime`|`body`|`string`|是|结束时间（事件结束时间，采用ISO8601时间格式，最大长度32个字符，与startTime配对使用，不能单独使用，时间范围最大不能超过3个月）|
|`alarmId`|`body`|`string`|否|测温记录id|
|`resourceName`|`body`|`string`|否|测温点名称，支持模糊查询|
|`groupIds`|`body`|`string[]`|否|测温分组id数组，从查询测温分组接口获取返回报文中的groupId字段|
|`resourceCodes`|`body`|`string[]`|否|测温点编号数组，从查询测温点接口获取返回报文中的resourceCode字段|
|`alarmType`|`body`|`integer`|否|报警类型，参考附录A.90 测温报警类型|
|`sort`|`body`|`string`|否|排序字段（支持resourceName、alarmTime填写排序的字段名称，例如：”alarmTime”）alarmTime：报警触发时间|
|`order`|`body`|`string`|否|升/降序（指定排序字段是使用升序（asc）还是降序（desc），例如：”asc”）|

## 请求参数示例
```json
{
  "pageSize": 10,
  "pageNo": 1,
  "startTime": "2020-03-09T18:00:00+08:00",
  "endTime": "2020-03-18T20:00:00+08:00",
  "alarmId": "49be8df4-b220-4b94-b328-ee275b7c81da",
  "resourceName": "Camera 02",
  "groupIds": [
    "fff65664-d2a7-4590-bc55-82715c9f19e6"
  ],
  "resourceCodes": [
    "f34dCC23b46540d2969c1ff36daba0fb"
  ],
  "alarmType": 192517,
  "sort": "alarmTime",
  "order": "asc"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.6 测温应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|单页展示数据数目|
|`+total`|`body`|`number`|否|总结果数|
|`+list`|`body`|`object[]`|否|返回数据集合|
|`++alarmId`|`body`|`string`|否|测温记录id|
|`++alarmType`|`body`|`number`|否|报警类型，参考附录A.90 测温报警类型|
|`++alarmName`|`body`|`string`|否|记录名称|
|`++groupId`|`body`|`string`|否|分组id|
|`++groupName`|`body`|`string`|否|分组名称|
|`++resourceCode`|`body`|`string`|否|资源点code|
|`++resourceName`|`body`|`string`|否|资源点名称|
|`++visiblePictureUri`|`body`|`string`|否|可见光图片相对地址|
|`++thermalPictureUri`|`body`|`string`|否|热成像图片相对地址|
|`++faceUrl`|`body`|`string`|否|人脸采集图片相对地址|
|`++svrIndexCode`|`body`|`string`|否|图片服务器code|
|`++alarmTime`|`body`|`string`|否|记录时间，采用ISO8601时间格式，最大长度32个字符，|
|`++temp`|`body`|`string`|否|体温|
|`++overTemp`|`body`|`number`|否|是否异常，0：正常，1：异常|
|`++registered`|`body`|`number`|否|是否登记，0:登记，1未登记|
|`++personName`|`body`|`string`|否|人员名称（身份证，匹配对应数据）|
|`++certNo`|`body`|`string`|否|身份证号，门禁数据字段（身份证，匹配对应数据）|
|`++personId`|`body`|`string`|否|人员id|
|`++jobNo`|`body`|`string`|否|工号|
|`++orgIndexCode`|`body`|`string`|否|组织code，|
|`++cardNo`|`body`|`string`|否|卡号|
