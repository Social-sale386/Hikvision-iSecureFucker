# 查询测温点

> 接口说明
> 支持分页查询添加到服务的测温点信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/gptas/v1/thermodetector/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/gptas/v1/thermodetector/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码；pageNo≥1|
|`pageSize`|`body`|`integer`|是|分页大小；0<pageSize≤1000|
|`resourceName`|`body`|`string`|否|资源名称，支持模糊查询|
|`groupIds`|`body`|`string[]`|否|测温分组id数组，从查询测温分组接口获取返回报文中的groupId字段|
|`resourceCodes`|`body`|`string[]`|否|测温点code数组|
|`resourceType`|`body`|`integer`|否|资源类型，1：视频，2门禁,不填则查询所有资源类型|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 1000,
  "resourceName": "门禁设备",
  "groupIds": [
    "fff65664-d2a7-4590-bc55-82715c9f19e6"
  ],
  "resourceCodes": [
    "881f45c91d7b423992b4CCd33fa16691"
  ],
  "resourceType": 2
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.6 测温应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|单页展示数据数目|
|`+total`|`body`|`number`|否|总结果数|
|`+list`|`body`|`object[]`|否|返回数据集合|
|`++resourceCode`|`body`|`string`|否|测温点code|
|`++resourceName`|`body`|`string`|否|测温点名称|
|`++resourceType`|`body`|`number`|否|测温点类型，1：视频，2门禁|
|`++deviceCode`|`body`|`string`|否|设备code|
|`++deviceName`|`body`|`string`|否|设备名称|
|`++groupId`|`body`|`string`|否|分组id|
|`++tmp`|`body`|`string`|否|温度阈值，单位：摄氏度|
|`++alarmTmp`|`body`|`string`|否|报警阈值，单位：摄氏度|
|`++openDoor`|`body`|`number`|否|体温异常允许开门，1：允许开门 0：不开门|
|`++extendDevIp`|`body`|`string`|否|热成像设备ip|
|`++deviceType`|`body`|`string`|否|设备型号|
|`++channelNum`|`body`|`string`|否|通道号|
|`++updateTime`|`body`|`string`|否|更新时间（采用ISO8601时间格式）|
