# 查询测温分组

> 接口说明
> 支持分页查询创建的测温分组信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/gptas/v1/group/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/gptas/v1/group/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码；pageNo≥1|
|`pageSize`|`body`|`integer`|是|分页大小；0<pageSize≤1000|
|`groupName`|`body`|`string`|否|测温分组名称，支持模糊查询|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 1000,
  "groupName": "测温"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.6 测温应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|单页展示数据数目|
|`+total`|`body`|`number`|否|总结果数|
|`+list`|`body`|`object[]`|否|返回数据集合|
|`++groupId`|`body`|`string`|否|测温分组id|
|`++groupName`|`body`|`string`|否|测温分组名称|
|`++updateTime`|`body`|`string`|否|更新时间（采用ISO8601时间格式）|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "pageNo": 1,
    "pageSize": 1000,
    "total": 1,
    "list": [
      {
        "groupId": "3f4974ae-c137-486b-a05f-e0323685137c",
        "groupName": "测温分组",
        "updateTime": "2020-03-10T16:06:33+08:00"
      }
    ]
  }
}
```
