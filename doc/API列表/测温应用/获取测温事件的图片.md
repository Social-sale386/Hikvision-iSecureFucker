# 获取测温事件的图片

> 接口说明
> 通过该接口根据输入的图片的相对Uri地址，可以获取到具体的数据数据。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/gptas/v1/events/picture`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/gptas/v1/events/picture
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`svrIndexCode`|`body`|`string`|否|存储接入服务（图片服务器唯一标识）|
|`picUri`|`body`|`string`|是|图片uri|

## 请求参数示例
```json
{
  "svrIndexCode": "ac1ae588-dd0e-47c5-b73e-6e389765101b",
  "picUri": "/pic?3ddc89ife-e*58a10433ee3a--b377e0dd9a67di6b1*=sd*=5dpi*=1d1i6t6pe*m5i18=a-86011e60z553s=8i02="
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.6 测温应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+picUrl`|`body`|`string`|否|图片URL|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "picUrl": "https://10.19.170.134:6113/pic?3ddc89ife-e*58a10433ee3a--b377e0dd9a67di6b1*=sd*=5dpi*=1d1i6t6pe*m5i18=a-86011e60z553s=8i02=&AccessKeyId=mTpmO2Qrp+FZXDEh&Expires=1583837976&Signature=hTPMhsN2moIT96mvBGftNJMC5EA="
  }
}
```
