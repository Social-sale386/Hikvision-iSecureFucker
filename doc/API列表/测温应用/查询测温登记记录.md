# 查询测温登记记录

> 接口说明
> 支持分页查询登记人员信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/gptas/v1/register/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/gptas/v1/register/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageSize`|`body`|`integer`|是|分页大小；0<pageSize≤1000|
|`pageNo`|`body`|`integer`|是|当前页码；pageNo≥1|
|`startTime`|`body`|`string`|是|开始时间（采用ISO8601时间格式，与endTime配对使用，不能单独使用，时间范围最大不能超过3个月）|
|`endTime`|`body`|`string`|是|结束时间（采用ISO8601时间格式，最大长度32个字符，与startTime配对使用，不能单独使用，时间范围最大不能超过3个月）|
|`alarmIds`|`body`|`string[]`|否|测温记录id数组，从查询测温记录接口获取返回报文中的alarmId字段|
|`resourceCodes`|`body`|`string[]`|否|测温点编号数组，从查询测温点接口获取返回报文中的resourceCode字段|
|`groupIds`|`body`|`string[]`|否|测温分组id数组，从查询测温分组接口获取返回报文中的groupId字段|
|`personName`|`body`|`string`|否|人员姓名|

```json
{
  "pageSize": 1000,
  "pageNo": 1,
  "startTime": "2020-03-01T12:00:00+08:00",
  "endTime": "2020-03-30T12:00:00+08:00",

  ],

    "835833826a194ab8b5a0fc7423738f11"
  ],
  "groupIds": [
    "52414a93-0fb0-48ce-bab0-278519b6d566"
  ],
  "personName": "中"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.6 测温应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|单页展示数据数目|
|`+total`|`body`|`number`|否|总结果数|
|`+list`|`body`|`object[]`|否|返回数据集合|
|`++registerId`|`body`|`string`|否|登记记录id|
|`++personName`|`body`|`string`|否|人员姓名|
|`++phone`|`body`|`string`|否|联系电话|
|`++identityType`|`body`|`number`|否|证件类型，参考附录A.11 证件类型|
|`++identityId`|`body`|`string`|否|证件号|
|`++fromArea`|`body`|`number`|否|是否来自疫区，0：否；1：是|
|`++organisation`|`body`|`string`|否|所属组织|
|`++address`|`body`|`string`|否|现住址|
|`++remarks`|`body`|`string`|否|描述|
|`++relatedAlarmId`|`body`|`string`|否|关联测温记录id|
|`++updateTime`|`body`|`string`|否|记录时间，采用ISO8601时间格式，最大长度32个字符，|
