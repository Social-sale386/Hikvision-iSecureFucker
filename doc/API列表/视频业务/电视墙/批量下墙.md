# 批量下墙

> 接口说明
> 1、在电视墙客户端上添加解码器和电视墙，并关联解码器的输出通道至电视墙上。
> 2、在电视墙上进行开窗上墙操作或调用窗口上墙接口。
> 3、调用窗口下墙接口（支持批量下墙）。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/tvms/v1/tvwall/realplay/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/tvwall/realplay/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`realplay_list`|`body`|`object[]`|是|下墙的窗口信息|
|`+wnd_uri`|`Body`|`string`|是|窗口Uri,作为窗口唯一标识，从获取电视墙窗口信息列表接口获取返回参数wnd_uri|

## 请求参数示例
```json
{
  "realplay_list": [
    {
      "wnd_uri": "api/tvwall/1/device/2/floatwnd/16/subwnd/1"
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+realplay_list`|`body`|`object[]`|否|窗口信息|
|`++wnd_id`|`body`|`number`|否|窗口编号|
|`++wnd_uri`|`body`|`string`|否|窗口uri|
|`++status`|`body`|`number`|否|窗口上墙状态：0-空闲，1-上墙，2-轮巡，3-报警|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "realplay_list": [
      {
        "wnd_id": 100860930,
        "wnd_uri": "api/tvwall/1/device/6/floatwnd/1/subwnd/1",
        "status": 0
      }
    ]
  }
}
```
