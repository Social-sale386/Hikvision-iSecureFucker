# 窗口批量创建

> 接口说明
> 1.添加解码器和电视墙。
> 2.关联输出通道至电视墙上
> 3.调用该接口进行窗口创建

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/tvms/v1/public/tvwall/floatWnds/addition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/public/tvwall/floatWnds/addition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`floatwnd_list`|`body`|`object[]`|是|窗口信息列表|
|`+dlp_id`|`Body`|`number`|是|电视墙id，从获取电视墙大屏信息接口获取返回参数dlp_id|
|`+wnd_pos`|`Body`|`number`|是|添加电视墙时每个窗口的编号，从0开始|
|`+wnd_left`|`Body`|`number`|是|坐标left|
|`+wnd_top`|`Body`|`number`|是|坐标top|
|`+wnd_width`|`Body`|`number`|是|宽（单位：LCD是按电视墙比例，LED是像素），注意LCD和LED的区别：|
|`+wnd_height`|`Body`|`number`|是|高（单位：LCD是按电视墙比例，LED是像素，注意LCD和LED的区别：|

补充说明:
- LCD：宽为1~128电视墙列数
- LED：宽为输出通道分辨率电视墙列数
- LCD：高为1~128行电视墙数
- LED：高为输出通道分辨率电视墙行数

## 请求参数示例
```json
{
  "floatwnd_list": [
    {
      "dlp_id": 1,
      "wnd_pos": 1,
      "wnd_left": 0,
      "wnd_top": 0,
      "wnd_width": 128,
      "wnd_height": 128
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+floatwnd_list`|`body`|`object[]`|否|图层列表信息|
|`++id`|`body`|`number`|否|图层编号|
|`++left`|`body`|`number`|否|左坐标|
|`++top`|`body`|`number`|否|顶坐标|
|`++width`|`body`|`number`|否|图层宽度|
|`++height`|`body`|`number`|否|图层高度|
|`++layer`|`body`|`number`|否|图层叠放序号，图层和图层之间可以重叠，值大的图层叠在值小的图层上。|
|`++dev_id`|`body`|`number`|否|设备中记录的图层编号|
|`++subwnd_num`|`body`|`number`|否|子窗口数|
|`++decoder_id`|`body`|`number`|否|所属解码器的编号|
|`++wndpos`|`body`|`number`|否|开窗所属查看器pos|
|`++wnd_id`|`body`|`number`|否|本地缓存的图层编号|
|`++uri`|`body`|`string`|否|绑定窗口uri,与电视墙相关，例：”api/tvwall/1/dlp/3/floatwnd/33624065”|
|`++attached_uri`|`body`|`string`|否|绑定设备uri,与设备相关，例：“api/tvwall/1/device/2/floatwnd/33624065”|
|`++dlp_id`|`body`|`number`|否|电视墙编号|
|`++openwnd_mode`|`body`|`number`|否|开窗模式: 0–统一坐标 1–分辨率开窗|
|`++dlp_row`|`body`|`number`|否|电视墙行数|
|`++dlp_col`|`body`|`number`|否|电视墙列数|
|`++device_wall_no`|`body`|`number`|否|设备墙号|
