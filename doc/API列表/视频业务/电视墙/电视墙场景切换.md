# 电视墙场景切换

> 接口说明
> 1、在电视墙客户端配置电视墙的场景信息。
> 2、获取电视墙所有的场景列表信息。
> 3、进行场景切换。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/tvms/v1/tvwall/scene/switch`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/tvwall/scene/switch
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`scene_id`|`body`|`integer`|是|场景id，从获取电视墙场景列表信息接口获取返回参数scene_id|

## 请求参数示例
```json
{
  "scene_id": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+scene_id`|`body`|`number`|否|场景id|
|`+scene_name`|`body`|`string`|否|场景名称|
|`+using`|`body`|`boolean`|否|是否正在使用|
|`+scene_order`|`body`|`number`|否|场景序号|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "scene_id": 1,
    "scene_name": "开窗",
    "using": true,
    "scene_order": 1
  }
}
```
