# 获取电视墙窗口信息列表

> 接口说明
> 1.在电视墙客户端上添加解码器和电视墙，并关联解码器的输出通道至电视墙上。
> 2.在电视墙上进行开窗操作。
> 3.调用获取窗口列表信息接口。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/tvms/v1/tvwall/wnds/get`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/tvwall/wnds/get
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`dlp_id`|`body`|`integer`|否|电视墙id，若为空，则返回该服务下所有窗口列表。|

补充说明:
- 从获取电视墙大屏信息接口获取返回参数dlp_id

## 请求参数示例
```json
{
  "dlp_id": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+wnd_list`|`body`|`object[]`|否|窗口信息列表|
|`++wnd_id`|`body`|`number`|否|窗口编号|
|`++wnd_uri`|`body`|`string`|否|窗口uri|
|`++status`|`body`|`number`|否|窗口上墙状态：0-空闲，1-上墙，2-轮巡，3-报警|
|`++camera`|`body`|`object`|否|监控点信息|
|`+++index_code`|`body`|`string`|否|监控点唯一标识|
|`+++stream_url`|`body`|`string`|否|取流url|
|`+++name`|`body`|`string`|否|监控点名称|
|`+++channel`|`body`|`number`|否|通道号|
|`+++stream_type`|`body`|`number`|否|码流类型，-1-平台码流 0-主码流 1-子码流 2-第三码流|
|`++decoder`|`body`|`object`|否|解码资源信息|
|`+++id`|`body`|`number`|否|解码器编号|
|`+++name`|`body`|`string`|否|解码器名称|
|`+++index_code`|`body`|`string`|否|解码器indexCode|
|`+++ip_addr`|`body`|`string`|否|解码器Ip|
|`+++port`|`body`|`number`|否|解码器port|
|`+++offline`|`body`|`boolean`|否|true-离线，false-在线|
