# 非开窗设备窗口分割

> 接口说明
> 1.添加解码器和电视墙。
> 2.关联输出通道至电视墙上。
> 3.调用该接口进行分割

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/tvms/v1/public/tvwall/monitor/division`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/public/tvwall/monitor/division
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`dlp_id`|`body`|`integer`|是|电视墙id，从获取电视墙大屏信息接口获取返回参数dlp_id|
|`monitor_pos`|`body`|`integer`|是|查看器pos,从获取电视墙大屏信息接口获取返回参数pos|
|`div_num`|`body`|`integer`|是|分割数，支持：|

补充说明:
- 1、2、4、6、8、9、12、16、25、36

## 请求参数示例
```json
{
  "dlp_id": 1,
  "monitor_pos": 1,
  "div_num": 4
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+output_id`|`body`|`number`|否|输出通道编号|
|`+index`|`body`|`number`|否|输出通道indexCode|
|`+subwnd_num`|`body`|`number`|否|子窗口数|
|`+pos`|`body`|`number`|否|输出在大屏上的位置，按从左到右，从上到下，0开始的编号|
|`+monitor_name`|`body`|`string`|否|输出通道的名称|
|`+joint`|`body`|`boolean`|否|标记是否已拼接|
|`+dlp_id`|`body`|`number`|否|关联的电视墙编号|
|`+dlp_row`|`body`|`number`|否|电视墙行数|
|`+dlp_col`|`body`|`number`|否|电视墙列数|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "output_id": 1,
    "index": 22,
    "subwnd_num": 4,
    "pos": 1,
    "monitor_name": "VGA_1",
    "joint": false,
    "dlp_id": 1,
    "dlp_row": 2,
    "dlp_col": 2
  }
}
```
