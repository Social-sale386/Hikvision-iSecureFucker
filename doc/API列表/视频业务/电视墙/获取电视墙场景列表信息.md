# 获取电视墙场景列表信息

> 接口说明
> 1、在电视墙客户端配置电视墙的场景信息。
> 2、获取电视墙所有的场景列表信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/tvms/v1/tvwall/scenes`|
|请求方法|`POST`|

## 快速调用
```text
POST /api/tvms/v1/tvwall/scenes
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+dlp_id`|`body`|`number`|否|电视墙编号|
|`+scene_list`|`body`|`object[]`|否|场景信息列表|
|`++tvwall_id`|`body`|`number`|否|电视墙编号|
|`++scene_id`|`body`|`number`|否|场景id|
|`++scene_name`|`body`|`string`|否|场景名称|
|`++default`|`body`|`boolean`|否|是否为默认场景|
|`++using`|`body`|`boolean`|否|场景是否正在使用|
|`++scene_order`|`body`|`number`|否|场景排序序号|
|`++belong_dlp_id`|`body`|`number`|否|所属于电视墙id|
|`++device_scene`|`body`|`boolean`|否|是否离线|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "dlp_id": 0,
    "scene_list": [
      {
        "tvwall_id": 1,
        "scene_id": 30,
        "scene_name": "系统测试",
        "default": true,
        "using": true,
        "scene_order": 30,
        "belong_dlp_id": 16,
        "device_scene": false
      }
    ]
  }
}
```
