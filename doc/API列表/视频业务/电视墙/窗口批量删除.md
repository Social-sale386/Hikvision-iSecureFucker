# 窗口批量删除

> 接口说明
> 1.添加解码器和电视墙。
> 2.关联输出通道至电视墙上。
> 3.窗口已存在
> 4.调用该接口删除窗口

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/tvms/v1/public/tvwall/floatWnds/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/public/tvwall/floatWnds/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`floatwnd_list`|`body`|`object[]`|是|窗口信息列表|
|`+dlp_id`|`Body`|`number`|是|电视墙id,从获取电视墙大屏信息接口获取返回参数dlp_id|
|`+floatwnd_id`|`Body`|`number`|是|窗口id，从获取电视墙大屏信息接口获取返回参数id（图层编号）|

## 请求参数示例
```json
{
  "floatwnd_list": [
    {
      "dlp_id": 1,
      "floatwnd_id": 1
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+floatwnd_list`|`body`|`object[]`|否|删除窗口结果列表|
|`++error_code`|`body`|`number`|否|删除结果码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`++error_info`|`body`|`string`|否|删除结果信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "floatwnd_list": [
      {
        "error_code": 0,
        "error_info": "OK"
      }
    ]
  }
}
```
