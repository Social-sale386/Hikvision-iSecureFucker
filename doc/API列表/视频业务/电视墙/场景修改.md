# 场景修改

> 接口说明
> 1.添加解码器和电视墙。
> 2.关联输出通道至电视墙上。
> 3.场景已存在
> 4.调用该接口修改或者保存场景

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/tvms/v1/public/tvwall/scene/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/public/tvwall/scene/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`scene_id`|`body`|`integer`|是|场景id，从获取电视墙场景列表信息接口获取返回参数scene_id|
|`name`|`body`|`string`|是|场景名称,不支持的特殊字符在ASCII码中的位置：1-8位，10-31位和127位。|

## 请求参数示例
```json
{
  "scene_id": 1,
  "name": "场景1"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+tvwall_id`|`body`|`number`|否|电视墙编号|
|`+scene_id`|`body`|`number`|否|场景id|
|`+scene_name`|`body`|`string`|否|场景名称|
|`+default`|`body`|`boolean`|否|是否为默认场景|
|`+using`|`body`|`boolean`|否|场景是否正在使用|
|`+scene_order`|`body`|`number`|否|场景排序序号|
|`+belong_dlp_id`|`body`|`number`|否|所属于电视墙id|
|`+device_scene`|`body`|`boolean`|否|是否离线|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "tvwall_id": 1,
    "scene_id": 30,
    "scene_name": "系统测试",
    "default": true,
    "using": true,
    "scene_order": 30,
    "belong_dlp_id": 16,
    "device_scene": false
  }
}
```
