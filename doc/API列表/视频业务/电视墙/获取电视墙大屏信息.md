# 获取电视墙大屏信息

> 接口说明
> 1.在电视墙客户端配置相关电视墙所需的信息：如添加电视墙、添加解码器、关联输出通道，配置场景信息。
> 2.调用获取电视墙所有的资源数据的接口。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/tvms/v1/tvwall/allResources`|
|请求方法|`POST`|

## 快速调用
```text
POST /api/tvms/v1/tvwall/allResources
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+tvwall_list`|`body`|`object[]`|否|电视墙信息列表|
|`++tvwall_id`|`body`|`number`|否|Tvwall编号|
|`++tvwall_name`|`body`|`string`|否|Tvwall名称|
|`++index_code`|`body`|`string`|否|Tvwall索引值|
|`++dlp_list`|`body`|`object[]`|否|大屏信息列表，实际客户端上操作的电视墙|
|`+++dlp_id`|`body`|`number`|否|大屏（实际客户端上操作的电视墙）id|
|`+++dlp_name`|`body`|`string`|否|大屏（实际客户端上操作的电视墙）名称|
|`+++index_code`|`body`|`string`|否|大屏（实际客户端上操作的电视墙）indexcode|
|`+++row`|`body`|`number`|否|电视墙行数|
|`+++col`|`body`|`number`|否|电视墙列数|
|`+++belong_tvwall_indexcode`|`body`|`string`|否|所属的电视墙indexcode|
|`+++dlp_type`|`body`|`number`|否|电视墙显示屏类型，电视墙显示屏类型分为两种：1-LCD，2-LED|
|`+++decoder_id`|`body`|`number`|否|关联到这个电视墙解码器id，如果有多个解码器关联一个电视墙，该值为首个关联到该电视墙的解码器id|
|`+++attached_device_num`|`body`|`number`|否|关联到该电视墙解码器数量|
|`+++cur_scene_id`|`body`|`number`|否|当前执行的场景id|
|`+++cur_scene_name`|`body`|`string`|否|当前执行的场景name|
|`+++layout_modified`|`body`|`boolean`|否|布局是否改变，包括窗口分割、拼接、上墙等|
|`+++order`|`body`|`number`|否|客户端上展示的电视墙排序序号|
|`+++floatwmd_list`|`body`|`object[]`|否|图层列表信息|
|`++++id`|`body`|`number`|否|图层编号|
|`++++left`|`body`|`number`|否|左坐标|
|`++++top`|`body`|`number`|否|顶坐标|
|`++++width`|`body`|`number`|否|图层宽度|
|`++++height`|`body`|`number`|否|图层高度|
|`++++layer`|`body`|`number`|否|图层叠放序号，图层和图层之间可以重叠，值大的图层叠在值小的图层上。|
|`++++dev_id`|`body`|`number`|否|设备中记录的图层编号|
|`++++decoder_id`|`body`|`number`|否|所属解码器的编号|
|`++++wndpos`|`body`|`number`|否|开窗所属查看器pos|
|`++++wnd_id`|`body`|`number`|否|本地缓存的图层编号|
|`++++uri`|`body`|`string`|否|绑定窗口uri,与电视墙相关，例：”api/tvwall/1/dlp/3/floatwnd/33624065”|
|`++++attached_uri`|`body`|`string`|否|绑定设备uri,与设备相关，例：“api/tvwall/1/device/2/floatwnd/33624065”|
|`++++dlp_id`|`body`|`number`|否|大屏（实际客户端上操作的电视墙）id|
|`++++cur_ability`|`body`|`number`|否|当前的解码能力集， 1-CIF 2-4CIF 3-D1 4-720P 5-1080P|
|`++++enlarged`|`body`|`boolean`|否|否处于放大状态|
|`++++alarm_link_div`|`body`|`boolean`|否|是否进行报警联动分割|
|`++++openwnd_mode`|`body`|`number`|否|开窗模式: 0–统一坐标 1–分辨率开窗|
|`++++dlp_row`|`body`|`number`|否|电视墙行数|
|`++++dlp_col`|`body`|`number`|否|电视墙列数|
|`++++device_wall_no`|`body`|`number`|否|设备墙号|
|`++++wnd_list`|`body`|`object[]`|否|窗口信息列表|
|`+++++wnd_id`|`body`|`number`|否|窗口编号|
|`+++++wnd_uri`|`body`|`string`|否|与设备相关窗口uri，例：api/tvwall/1/device/2/floatwnd/33624065/subwnd/1|
|`+++++status`|`body`|`number`|否|窗口上墙状态：0-空闲，1-上墙，2-轮巡，3-报警|
|`+++++decode_channel`|`body`|`number`|否|解码通道id|
|`+++++virtual_id`|`body`|`number`|否|虚拟id|
|`+++monitor_list`|`body`|`object[]`|否|显示器信息列表|
|`++++output_id`|`body`|`number`|否|输出通道编号|
|`++++index`|`body`|`number`|否|输出通道indexCode|
|`++++pos`|`body`|`number`|否|输出在大屏上的位置，按从左到右，从上到下，0开始的编号|
|`++++monitor_name`|`body`|`string`|否|输出通道的名称|
|`++++joint`|`body`|`boolean`|否|标记是否已拼接|
|`++++dlp_id`|`body`|`number`|否|关联的电视墙编号|
|`++++dlp_row`|`body`|`number`|否|电视墙行数|
|`++++dlp_col`|`body`|`number`|否|电视墙列数|
