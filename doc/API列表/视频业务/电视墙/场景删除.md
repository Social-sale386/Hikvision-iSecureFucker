# 场景删除

> 接口说明
> 1.添加解码器和电视墙。
> 2.关联输出通道至电视墙上。
> 3.场景已存在
> 4.调用该接口删除场景

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/tvms/v1/public/tvwall/scene/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/public/tvwall/scene/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`scene_id`|`body`|`integer`|是|场景id，从获取电视墙场景列表信息接口获取返回参数scene_id|

## 请求参数示例
```json
{
  "scene_id": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+error_code`|`body`|`number`|否|删除结果码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`+error_info`|`body`|`string`|否|删除结果信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "error_code": 0,
    "error_info": "success"
  }
}
```
