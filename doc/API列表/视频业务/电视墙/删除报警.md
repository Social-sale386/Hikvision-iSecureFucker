# 删除报警

> 接口说明
> 1.添加解码器和电视墙。
> 2.关联输出通道至电视墙上。
> 3.窗口已存在且设置为报警窗口
> 4.当前报警正在报警上墙
> 5.调用该接口停止报警上墙

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/tvms/v1/public/tvwall/alarm/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/tvms/v1/public/tvwall/alarm/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`event_index`|`body`|`string`|是|报警事件编号|

## 请求参数示例
```json
{
  "event_index": "677BF56FA1AE4A9BB1308F57CA93C346"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.2 电视墙错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {}
}
```
