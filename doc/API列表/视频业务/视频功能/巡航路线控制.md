# 巡航路线控制

> 接口说明
> 本接口用于进行巡航路线控制，包括开始、停止巡航两种操作。
> 进行巡航路线控制操作的用户需要配置相应监控点的云台控制权限，本接口会根据合作方的userId进行权限过滤。
> 级联点位不支持该功能。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/video/v1/cruiseRoutes/controlling`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/cruiseRoutes/controlling
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识,从分页获取监控点资源接口获取返回参数cameraIndexCode|
|`command`|`body`|`string`|是|控制命令，不区分大小写，开始巡航：START_CRUISE；停止巡航：STOP_CRUISE|
|`cruiseIndex`|`body`|`integer`|是|巡航路线序号，范围：[1,8]，可通过查询巡航路线接口获取返回参数cruiseIndex|

## 请求参数示例
```json
{
  "cameraIndexCode": "4750e3a4a5bbad3cda5bbad3cd4af9ed5101",
  "command": "START_CRUISE",
  "cruiseIndex": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```
