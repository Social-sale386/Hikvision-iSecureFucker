# 删除预置点信息

> 接口说明
> 该接口用于删除监控点的预置点信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/video/v1/presets/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/presets/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识，分页获取监控点资源接口获取返回参数cameraIndexCode|
|`presetIndex`|`body`|`integer`|是|预置点编号，查询预置点信息接口获取返回参数presetPointIndex|

## 请求参数示例
```json
{
  "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb",
  "presetIndex": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否||
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|

补充说明:
- 返回码，0-成功，其它-失败，参考附录E.1.1 视频应用错误码

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": ""
}
```
