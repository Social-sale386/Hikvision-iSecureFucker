# 查询手动录像编号

> 接口说明
> 该接口用于根据用户id获取当前用户创建的手动录像任务ID，用于获取手动录像状态和停止手动录像。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/video/v1/manualRecord/taskId/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/manualRecord/taskId/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCodes`|`body`|`string[]`|是|监控点编号集合，一次性最多传入1000个监控点编号进行查询|

## 请求参数示例
```json
{
  "cameraIndexCodes": [
    "fee3d641f0e44ecCC9b9def2b423b7eb"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+userId`|`body`|`string`|否|下发手动录像的用户ID|
|`+manualRecordTask`|`body`|`object[]`|否|手动录像编号集合，只查询未结束的手动录像编号信息|
|`++cameraIndexCode`|`body`|`string`|否|监控点唯一标识|
|`++taskID`|`body`|`string`|否|手动录像编号|
|`++type`|`body`|`number`|否|下发的存储类型。0-中心存储，1-设备存储。如果不传入，默认为0-中心存储|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "userId": "admin",
    "manualRecordTask": [
      {
        "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb",
        "taskID": "753187000009",
        "type": 0
      }
    ]
  }
}
```
