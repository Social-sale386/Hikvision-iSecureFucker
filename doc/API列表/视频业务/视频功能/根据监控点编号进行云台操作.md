# 根据监控点编号进行云台操作

> 接口说明
> 根据监控点编号进行云台操作接口

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/video/v1/ptzs/controlling`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/ptzs/controlling
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识，分页获取监控点资源接口获取返回参数cameraIndexCode|
|`action`|`body`|`integer`|是|0-开始|
|`command`|`body`|`string`|是|不区分大小写|
|`speed`|`body`|`integer`|否|云台速度，取值范围为1-100，默认50。|
|`presetIndex`|`body`|`integer`|否|预置点编号，整数，通常在300以内|

补充说明:
- 1-停止
- 注：GOTO_PRESET命令下填任意值均可转到预置点,建议填0即可
- 说明：
- LEFT 左转
- RIGHT右转
- UP 上转
- DOWN 下转
- ZOOM_IN 焦距变大
- ZOOM_OUT 焦距变小
- LEFT_UP 左上
- LEFT_DOWN 左下
- RIGHT_UP 右上
- RIGHT_DOWN 右下 FOCUS_NEAR 焦点前移
- FOCUS_FAR 焦点后移 IRIS_ENLARGE 光圈扩大
- IRIS_REDUCE 光圈缩小
- WIPER_SWITCH 接通雨刷开关
- START_RECORD_TRACK开始记录路线
- STOP_RECORD_TRACK 停止记录路线
- START_TRACK 开始路线
- STOP_TRACK 停止路线
- 以下命令presetIndex不可
- 为空：
- GOTO_PRESET到预置点
- 若遇到设备不会转动，建议云台控制时传入speed字段。

## 请求参数示例
```json
{
  "cameraIndexCode": "748d84750e3a4a5bbad3cd4af9ed5101",
  "action": 1,
  "command": "GOTO_PRESET",
  "speed": 4,
  "presetIndex": 20
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0-成功，其它-失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```
