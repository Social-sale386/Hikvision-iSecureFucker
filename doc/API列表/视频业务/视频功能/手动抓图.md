# 手动抓图

> 接口说明
> 该接口用于手动触发设备抓图，返回图片的地址，抓图前请确保平台上已配置图片存储信息。抓图时间为触发手动抓图命令的时间。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/video/v1/manualCapture`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/manualCapture
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识，分页获取监控点资源接口获取返回参数cameraIndexCode|

## 请求参数示例
```json
{
  "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+picUrl`|`body`|`string`|否|图片url信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "picUrl": "http://10.33.47.126:6040/pic?=d4=i535z4165s08d-=64idm*ep=t6p7i=d1s*i7d9i*d2=*9b8d0eaa5-16c2d72-4a555f-88i4e6*e2e28d4"
  }
}
```
