# 查询预置点信息

> 接口说明
> 该接口用于查询监控点的预置点信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/video/v1/presets/searches`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/presets/searches
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识，分页获取监控点资源接口获取返回参数cameraIndexCode|

## 请求参数示例
```json
{
  "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0-成功，其它-失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|结果信息|
|`+total`|`body`|`number`|否|结果总数|
|`+list`|`body`|`object[]`|否|预置点信息列表|
|`++presetPointName`|`body`|`string`|否|预置点名称|
|`++presetPointIndex`|`body`|`number`|否|预置点编号|
|`++cameraIndexCode`|`body`|`string`|否|监控点编号|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "list": [
      {
        "presetPointName": "预置点1",
        "presetPointIndex": 1,
        "cameraIndexCode": "9b388a28080c448a873b711aafda144c"
      }
    ]
  }
}
```
