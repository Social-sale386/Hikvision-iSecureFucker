# 添加或修改巡航路线

> 接口说明
> 本接口用于增加或修改巡航路线，存储巡航路线信息的同时下发到设备。
> 每台前端设备最多设置8个巡航路线，每条路线最多32个巡航点。
> 进行增加或修改巡航路线操作的用户需要配置相应监控点的云台控制权限。
> 级联点位不支持该功能。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/video/v1/cruiseRoutes/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/cruiseRoutes/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识,从分页获取监控点资源接口获取返回参数cameraIndexCode|
|`cruiseIndex`|`body`|`integer`|是|巡航路线序号，范围：[1,8]，可通过 查询巡航路线接口获取返回参数cruiseIndex|
|`cruiseName`|`body`|`string`|否|巡航路线名称，增加时，必须指定；修改时，如果为null或为空时不进行修改|
|`points`|`body`|`object[]`|否|巡航路线的路线点数据列表|
|`+presetPointIndex`|`Body`|`number`|是|预置点序号，范围：[1,255]|
|`+speed`|`Body`|`number`|是|速度，无具体单位，数值越大，速度越快，范围：[1, 40]|
|`+standingTime`|`Body`|`number`|是|停留时间，单位为秒，范围：[15, 100]|

## 请求参数示例
```json
{
  "cameraIndexCode": "3bc68712640a49209fcfef896e54849f",
  "cruiseIndex": 1,
  "cruiseName": "巡航名称",
  "points": [
    {
      "presetPointIndex": 1,
      "speed": 20,
      "standingTime": 20
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```
