# 查询巡航路线

> 接口说明
> 查询巡航路线信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/video/v1/cruiseRoutes/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/cruiseRoutes/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识,从分页获取监控点资源接口获取返回参数cameraIndexCode|
|`nameLike`|`body`|`string`|否|巡航路线名称，名称模糊搜索，为空或者为null时忽略此条件|

## 请求参数示例
```json
{
  "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb",
  "nameLike": "巡航1"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总数|
|`+list`|`body`|`object[]`|否|巡航路线对象列表|
|`++cruiseName`|`body`|`string`|否|巡航路线名称|
|`++cameraIndexCode`|`body`|`string`|否|巡航路线所属监控点编号|
|`++cruiseIndex`|`body`|`number`|否|巡航路线序号，范围：[1,8]|
|`++points`|`body`|`object[]`|否|巡航路线的路线点数据列表|
|`+++presetPointIndex`|`body`|`number`|否|关联的预置点序号|
|`+++speed`|`body`|`number`|否|速度，无具体单位，数值越大，速度越快，范围：[1, 40]|
|`+++standingTime`|`body`|`number`|否|停留时间，单位为秒，范围：[15,100]|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 10,
    "list": [
      {
        "cruiseName": "巡航名称",
        "cameraIndexCode": "3bc68712640a49209fcfef896e54849f",
        "cruiseIndex": 1,
        "points": [
          {
            "presetPointIndex": 1,
            "speed": 20,
            "standingTime": 20
          }
        ]
      }
    ]
  }
}
```
