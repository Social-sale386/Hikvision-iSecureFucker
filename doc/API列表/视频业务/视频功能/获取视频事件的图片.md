# 获取视频事件的图片

> 接口说明
> 通过该接口根据输入的图片的相对Uri地址，可以获取到具体的图片数据。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/video/v1/events/picture`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/events/picture
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`svrIndexCode`|`body`|`string`|是|图片存储服务器唯一标识，提供picUri处会提供此字段|
|`picUri`|`body`|`string`|是|图片的相对地址|
|`netProtocol`|`body`|`string`|否|网络协议。“http”:HTTP协议、“https”:HTTPS协议。参数不填，优先返回https协议|

## 请求参数示例
```json
{
  "svrIndexCode": "1f276203e5234bdca08f7d99e1097bba",
  "picUri": "/pic?9dd351ia3-e*8634503d1165m3ep=t=i1p*i=d1s*i=d3b*i8da0*c74818542-9445fb–0874a79z189s=7ie4=",
  "netProtocol": "http"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0–成功，其他-失败|
|`msg`|`body`|`string`|否|接口调用的结果说明|
|`data`|`body`|`object`|否|返回数据详细信息|
|`+picUrl`|`body`|`string`|否|图片url|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "picUrl": "http://10.33.47.126:6040/pic?=d4=i535z4165s08d-=64idm*ep=t6p7i=d1s*i7d9i*d2=*9b8d0eaa5-16c2d72-4a555f-88i4e6*e2e28d4"
  }
}
```
