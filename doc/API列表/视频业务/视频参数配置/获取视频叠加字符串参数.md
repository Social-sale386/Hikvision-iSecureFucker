# 获取视频叠加字符串参数

> 接口说明
> 本接口用于根据监控点编号获取视频叠加字符串参数配置。
> 仅支持海康SDK、ehome两种协议的监控点，且级联点位不支持该功能。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/video/v1/showStringParams/get`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/showStringParams/get
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识,从分页获取监控点资源接口获取返回参数cameraIndexCode|

## 请求参数示例
```json
{
  "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+cameraIndexCode`|`body`|`string`|否|监控点唯一标识|
|`+showStringParams`|`body`|`object[]`|否|叠加字符串信息列表|
|`++showString`|`body`|`string`|否|叠加字符串文本内容；注意海康SDK限制其数组长度为44，故最大允许设置43个字符|
|`++isShow`|`body`|`number`|否|是否显示叠加内容，0-不显示；1-显示|
|`++xPos`|`body`|`number`|否|显示X坐标|
|`++yPos`|`body`|`number`|否|显示Y坐标|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb",
    "showStringParams": [
      {
        "showString": "camera 01",
        "isShow": 1,
        "xPos": 16,
        "yPos": 16
      }
    ]
  }
}
```
