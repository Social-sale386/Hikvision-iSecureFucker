# 获取预览画面参数配置

> 接口说明
> 本接口用于查询监控点的预览画面参数配置，包括亮度、灰度、对比度、饱和度。
> 仅支持海康SDK、ehome两种协议的监控点，且级联点位不支持该功能。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/video/v1/videoParams/get`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/video/v1/videoParams/get
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点唯一标识,从分页获取监控点资源接口获取返回参数cameraIndexCode|

## 请求参数示例
```json
{
  "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.1.1 视频应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+cameraIndexCode`|`body`|`string`|否|监控点唯一标识|
|`+brightness`|`body`|`number`|否|亮度，范围：[1,10]|
|`+hue`|`body`|`number`|否|灰度，范围：[1,10]|
|`+contrast`|`body`|`number`|否|对比度，范围：[1,10]|
|`+saturation`|`body`|`number`|否|饱和度，范围：[1,10]|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "cameraIndexCode": "fee3d641f0e44ecCC9b9def2b423b7eb",
    "brightness": 4,
    "hue": 2,
    "contrast": 1,
    "saturation": 1
  }
}
```
