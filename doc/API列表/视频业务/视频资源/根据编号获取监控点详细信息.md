# 根据编号获取监控点详细信息

> 接口说明
> 获取单个监控点信息是指根据监控点唯一标识来获取指定的监控点信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/cameras/indexCode`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/cameras/indexCode
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cameraIndexCode`|`body`|`string`|是|监控点编号，可通过【查询监控点列表】接口获取。|

## 请求参数示例
```json
{
  "cameraIndexCode": "44080000081314000001"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|监控点信息结构体|
|`+altitude`|`body`|`string`|否|海拔|
|`+cameraIndexCode`|`body`|`string`|否|监控点唯一标识|
|`+cameraName`|`body`|`string`|否|监控点名称|
|`+cameraType`|`body`|`number`|否|监控点类型|
|`+cameraTypeName`|`body`|`string`|否|监控点类型说明|
|`+capabilitySet`|`body`|`string`|否|能力集|
|`+capabilitySetName`|`body`|`string`|否|能力集说明|
|`+intelligentSet`|`body`|`string`|否|智能分析能力集|
|`+intelligentSetName`|`body`|`string`|否|智能分析能力集说明|
|`+channelNo`|`body`|`string`|否|通道编号|
|`+channelType`|`body`|`string`|否|通道类型|
|`+channelTypeName`|`body`|`string`|否|通道子类型说明|
|`+createTime`|`body`|`string`|否|创建时间|
|`+encodeDevIndexCode`|`body`|`string`|否|所属编码设备唯一标识|
|`+encodeDevResourceType`|`body`|`string`|否|所属设备类型|
|`+encodeDevResourceTypeName`|`body`|`string`|否|所属设备类型说明|
|`+gbIndexCode`|`body`|`string`|否|监控点国标编号|
|`+installLocation`|`body`|`string`|否|安装位置|
|`+keyBoardCode`|`body`|`string`|否|键盘控制码|
|`+latitude`|`body`|`string`|否|纬度|
|`+longitude`|`body`|`string`|否|经度|
|`+pixel`|`body`|`string`|否|摄像机像素（1-普通像素，2-130万高清，3-200万高清，4-300万高清）|
|`+ptz`|`body`|`string`|否|云镜类型（1-全方位云台（带转动和变焦），2-只有变焦,不带转动，3-只有转动，不带变焦，4-无云台，无变焦）|
|`+ptzName`|`body`|`string`|否|云镜类型说明|
|`+ptzController`|`body`|`string`|否|云台控制(1-DVR，2-模拟矩阵，3-MU4000，4-NC600)|
|`+ptzControllerName`|`body`|`string`|否|云台控制说明|
|`+recordLocation`|`body`|`string`|否|录像存储位置|
|`+recordLocationName`|`body`|`string`|否|录像存储位置说明|
|`+regionIndexCode`|`body`|`string`|否|所属区域唯一标识|
|`+status`|`body`|`string`|否|在线状态（0-未知，1-在线，2-离线）|
|`+statusName`|`body`|`string`|否|状态说明|
|`+transType`|`body`|`number`|否|传输协议（0-UDP，1-TCP）|
|`+transTypeName`|`body`|`string`|否|传输协议类型说明|
|`+treatyType`|`body`|`string`|否|接入协议|
|`+treatyTypeName`|`body`|`string`|否|接入协议类型说明|
|`+viewshed`|`body`|`string`|否|可视域相关（JSON格式）|
|`+updateTime`|`body`|`string`|否|更新时间|
