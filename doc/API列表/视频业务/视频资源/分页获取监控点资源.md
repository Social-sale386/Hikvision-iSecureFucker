# 分页获取监控点资源

> 接口说明
> 获取监控点列表接口可用来全量同步监控点信息，返回结果分页展示。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/cameras`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/cameras
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|范围 ( 0 , ~ )，若是范围内数字超过实际最大页码值，返回最后一页记录|
|`pageSize`|`body`|`number`|是|范围 ( 0 , 1000 ]|
|`treeCode`|`body`|`string`|否|树编号。综合安防平台当前未使用该字段。该字段预留。|

补充说明:
- 注：0 < pageNo
- 注：0 < pageSize≤1000

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 2
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|区域信息结构体|

|`+pageSize`|`body`|`number`|否|每页记录总数|

|`++altitude`|`body`|`string`|否|海拔|
|`++cameraIndexCode`|`body`|`string`|否|监控点唯一标识|
|`++cameraName`|`body`|`string`|否|监控点名称|
|`++cameraType`|`body`|`number`|否|监控点类型|
|`++cameraTypeName`|`body`|`string`|否|监控点类型说明|
|`++capabilitySet`|`body`|`string`|否|能力集（详见数据字典，typeCode为xresmgr.capability_set）|
|`++capabilitySetName`|`body`|`string`|否|能力集说明|
|`++intelligentSet`|`body`|`string`|否|智能分析能力集|
|`++intelligentSetName`|`body`|`string`|否|智能分析能力集说明|
|`++channelNo`|`body`|`string`|否|通道编号|
|`++channelType`|`body`|`string`|否|通道类型|
|`++channelTypeName`|`body`|`string`|否|通道子类型说明|
|`++createTime`|`body`|`string`|否|创建时间|
|`++encodeDevIndexCode`|`body`|`string`|否|所属编码设备唯一标识|
|`++encodeDevResourceType`|`body`|`string`|否|所属设备类型（详见数据字典，typeCode为xresmgr.resource_type）|
|`++encodeDevResourceTypeName`|`body`|`string`|否|所属设备类型说明|
|`++gbIndexCode`|`body`|`string`|否|监控点国标编号|
|`++installLocation`|`body`|`string`|否|安装位置|
|`++keyBoardCode`|`body`|`string`|否|键盘控制码|
|`++latitude`|`body`|`string`|否|纬度|
|`++longitude`|`body`|`string`|否|经度|
|`++pixel`|`body`|`number`|否|摄像机像素（1-普通像素，2-130万高清，3-200万高清，4-300万高清，取值参考数据字典，typeCode为xresmgr.piexl）|
|`++ptz`|`body`|`number`|否|云镜类型（1-全方位云台（带转动和变焦），2-只有变焦,不带转动，3-只有转动，不带变焦，4-无云台，无变焦，取值参考数据字典，typeCode为xresmgr.ptz_type）|
|`++ptzController`|`body`|`number`|否|云台控制(1-DVR，2-模拟矩阵，3-MU4000，4-NC600，取值参考数据字典，typeCode为xresmgr.ptz_control_type)|
|`++ptzControllerName`|`body`|`string`|否|云台控制说明|
|`++ptzName`|`body`|`string`|否|云镜类型说明|
|`++recordLocation`|`body`|`string`|否|录像存储位置（0-中心存储，1-设备存储，取值参考数据字典，typeCode为xresmgr.record_location）|
|`++recordLocationName`|`body`|`string`|否|录像存储位置说明|
|`++regionIndexCode`|`body`|`string`|否|所属区域唯一标识|
|`++status`|`body`|`number`|否|在线状态（0-不在线，1-在线，取值参考数据字典，typeCode为xresmgr.status）|
|`++statusName`|`body`|`string`|否|状态说明|
|`++transType`|`body`|`number`|否|传输协议（0-UDP，1-TCP）|
|`++transTypeName`|`body`|`string`|否|传输协议类型说明|
|`++treatyType`|`body`|`string`|否|接入协议（详见数据字典，typeCode为xresmgr.protocol_type）|
|`++treatyTypeName`|`body`|`string`|否|接入协议类型说明|
|`++viewshed`|`body`|`string`|否|可视域相关（JSON格式）|
|`++updateTime`|`body`|`string`|否|更新时间|

补充说明:
- 采用ISO8601标准，如2018-07-26T21:30:08+08:00
- 表示北京时间2017年7月26日21时30分08秒
