# 获取编码设备在线状态

> 接口说明
> 根据条件获取编码设备在线状态接口

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/nms/v1/online/encode_device/get`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/nms/v1/online/encode_device/get
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`regionId`|`body`|`string`|否|区域唯一标志，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`ip`|`body`|`string`|否|设备ip|
|`indexCodes`|`body`|`string[]`|否|设备编码列表，最大500，查询编码设备列表v2接口获取返回参数indexCode|
|`status`|`body`|`string`|否|状态，1-在线，0-离线，-1-未检测|
|`pageNo`|`body`|`integer`|否|页码，从1开始，不填默认为1|
|`pageSize`|`body`|`integer`|否|页大小，正整数，不填默认为10|
|`includeSubNode`|`body`|`string`|否|是否包含下级，1包含，0不包含（若regionId为空，则该参数不起作用）|

## 请求参数示例
```json
{
  "regionId": "root000000",
  "ip": "10.33.43.135",
  "indexCodes": [
    "f98y8c28y85y7y213c082yu95yu"
  ],
  "status": "1",
  "pageNo": 1,
  "pageSize": 20,
  "includeSubNode": "1"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.4 网管错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|数据信息|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+totalPage`|`body`|`number`|否|总页数|
|`+total`|`body`|`number`|否|总记录数|
|`+list`|`body`|`object[]`|否|资源信息集|
|`++deviceType`|`body`|`string`|否|设备型号|
|`++deviceIndexCode`|`body`|`string`|否|设备唯一编码|
|`++regionIndexCode`|`body`|`string`|否|区域编码|
|`++collectTime`|`body`|`string`|否|采集时间|
|`++regionName`|`body`|`string`|否|区域名字|
|`++indexCode`|`body`|`string`|否|资源唯一编码|
|`++cn`|`body`|`string`|否|设备名称|
|`++treatyType`|`body`|`string`|否|协议类型，|
|`++manufacturer`|`body`|`string`|否|厂商，hikvision-海康，dahua-大华|
|`++ip`|`body`|`string`|否|ip地址|
|`++port`|`body`|`number`|否|端口|
|`++online`|`body`|`number`|否|在线状态，0离线，1在线|

补充说明:
- 海康私有协议：hiksdk_net，
- GB/T28181：
- gb_reg,
- eHome协议：ehome_reg，
- 大华私有协议：dhsdk_net，
- ONVIF协议：onvif_net，
- 萤石协议：
- ezviz_net，
- 级联：
- cascade
