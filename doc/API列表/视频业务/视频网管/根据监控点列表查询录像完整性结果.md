# 根据监控点列表查询录像完整性结果

> 接口说明
> 根据监控点编号、开始时间、结束时间
> 分页获取录像录像完整性记录

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/nms/v1/record/list`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/nms/v1/record/list
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码，0 < pageNo|
|`pageSize`|`body`|`integer`|是|分页大小，0<pageSize≤1000|
|`beginTime`|`body`|`string`|是|录像开始日期，IOS8601格式|
|`endTime`|`body`|`string`|是|录像结束日期，IOS8601格式|
|`indexCodes`|`body`|`string[]`|是|监控点编号集，|

补充说明:
- 查询监控点列表接口获取返回参数cameraIndexCode

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 20,
  "beginTime": "2017-06-15T00:00:00.000+08:00",
  "endTime": "2017-06-17T00:00:00.000+08:00",
  "indexCodes": [
    "90ad77d8057c43dab140b77361606927"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.4 网管错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总条数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|当前分页记录数|
|`+list`|`body`|`object[]`|否|录像完整性数据列表|
|`++indexCode`|`body`|`string`|否|监控点编号|
|`++date`|`body`|`string`|否|录像日期（格式：yyyy-MM-dd）|
|`++collectTime`|`body`|`number`|否|录像巡检时间（具体的巡检结果更新时间，时间戳，单位：毫秒）|
|`++inspectionDate`|`body`|`string`|否|巡检日期（格式：yyyy-MM-dd）|
|`++result`|`body`|`number`|否|巡检结果(31-正常，32-异常，34-未配置)|
|`++videoClipstimeSegment`|`body`|`object[]`|否|实际巡检录像的片段|
|`+++from`|`body`|`number`|否|片段开始时间（单位：秒）|
|`+++to`|`body`|`number`|否|片段结束时间（单位：秒）|
|`+++type`|`body`|`number`|否|片段结果(31-正常，32-异常，34-未配置)|
|`++intactDuration`|`body`|`number`|否|巡检录像时长（单位：秒|
|`++planDuration`|`body`|`number`|否|计划录像时长（单位：秒）|

```json
            "to": 86400,
            "type": 32
          }
        ],

      }
    ]
  }

```
