# 根据监控点列表查询视频质量诊断结果

> 接口说明
> 根据监控点编号查询视频质量诊断结果，此接口可用作检测监控点位预览画面是否正常。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/nms/v1/vqd/list`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/nms/v1/vqd/list
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码，0 < pageNo|
|`pageSize`|`body`|`integer`|是|分页大小，0<pageSize≤500|
|`indexCodes`|`body`|`string[]`|是|监控点编号集， 查询监控点列表接口获取返回参数cameraIndexCode|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 20,
  "indexCodes": [
    "90ad77d8057c43dab140b77361606927"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.4 网管错误码|
|`msg`|`body`|`string`|否|返回信息|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总条数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|当前分页记录数|
|`+totalPage`|`body`|`number`|否|总页数|
|`+list`|`body`|`object[]`|否|视频质量诊断结果列表|
|`++scChromaResult`|`body`|`number`|否|偏色检测结果(2-诊断失败，-1-未知，1-正常，0-异常)|
|`++scSnowResult`|`body`|`number`|否|雪花点干扰结果(2-诊断失败，-1-未知，1-正常，0-异常)|
|`++scDarkResult`|`body`|`number`|否|图像过暗结果(2-诊断失败，-1-未知，1-正常，0-异常)|
|`++inspectDate`|`body`|`string`|否|巡检日期（格式：yyyy-MM-dd）|
|`++indexCode`|`body`|`string`|否|监控点编号|
|`++scLumaResult`|`body`|`number`|否|亮度异常结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scFreezeResult`|`body`|`number`|否|画面冻结结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scShakeResult`|`body`|`number`|否|视频抖动结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++resolution`|`body`|`number`|否|分辨率（720为基准 1-高清，0-标清，-1-未知）|
|`++scContrastResult`|`body`|`number`|否|对比度检测结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scStreakResult`|`body`|`number`|否|条纹干扰结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++picUrl`|`body`|`string`|否|图片url地址|
|`++scFrameResult`|`body`|`number`|否|视频丢帧结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scCoverResult`|`body`|`number`|否|视频遮挡结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scSignalResult`|`body`|`number`|否|视频丢失结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scMonoResult`|`body`|`number`|否|黑白图像结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++inspectResult`|`body`|`number`|否|视频巡检结果（0或-1-未检测，1-正常，2-异常，3-登录失败，4-取流失败，5-解码失败，6-码流时延）|
|`++scBlurResult`|`body`|`number`|否|图像模糊结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scSceneResult`|`body`|`number`|否|场景变更结果（2-诊断失败，-1-未知，1-正常，0-异常）|
|`++scFlashResult`|`body`|`number`|否|视频剧变结果（2-诊断失败，-1-未知，1-正常，0-异常）|
