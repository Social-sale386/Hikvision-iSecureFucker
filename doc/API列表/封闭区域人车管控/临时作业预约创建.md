# 临时作业预约创建

> 接口说明
> 该接口可以创建区域临时作业，指定作业人员，平台管理员审批后可以对作业人员下发区域权限。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center 1.3.0及以上版本|
|接口地址|`/api/acsopac/v1/temporary/work/create`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acsopac/v1/temporary/work/create
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`areaCode`|`body`|`string`|是|区域编号，可通过查询封闭区域列表接口获取。|
|`workName`|`body`|`string`|是|临时作业名称，长度为1~32个字符,支持中英文字符、数字，不能包含 /😗?<>|
|`workDesc`|`body`|`string`|否|临时作业内容描述，长度为1~32个字符,支持中英文字符、数字，不能包含 /😗?<>|
|`principalPersonId`|`body`|`string`|是|作业负责人personId，可通过人员信息查询接口，如查询人员列表v2接口获取。|
|`peerPersonId`|`body`|`string[]`|否|作业随行人，最多可添加10人|
|`workStartTime`|`body`|`string`|是|作业开始时间，时间参数需满足ISO8601格式如：“2022-10-24T17:30:08.000+08:00”|
|`workEndTime`|`body`|`string`|是|作业结束时间，时间参数需满足ISO8601格式如：“2022-10-24T17:30:08.000+08:00”|

## 请求参数示例
```json
{
  "areaCode": "28790ece-fd64-4c86-b800-f100be0CC3e3",
  "workName": "临时作业api接口测试",
  "workDesc": "临时作业api接口测试描述",
  "principalPersonId": "bb1188a70694402fa10fedff0463051f",
  "peerPersonId": [
    "f53908c665f0454a978feb69ac5a054f"
  ],
  "workStartTime": "2022-10-15T17:30:08.000+08:00",
  "workEndTime": "2022-10-15T18:30:08.000+08:00"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|返回码|
|`msg`|`body`|`string`|是|返回消息|
|`data`|`body`|`object`|是|返回数据|
|`+workId`|`body`|`number`|是|临时作业id|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "workId": 8
  }
}
```
