# 查询区域实时数据详情

> 接口说明
> 该接口主要是应用于查询区域实时数据详情。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/acsopac/v1/area/detail`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acsopac/v1/area/detail
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`areaCode`|`body`|`string`|是|区域编号，可通过查询封闭区域列表接口获取。|

## 请求参数示例
```json
{
  "areaCode": "051a8888-d65a-43ac-bc81-84a4ac0ec4c9"
}
```
