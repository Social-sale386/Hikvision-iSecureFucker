# 查询封闭区域列表

> 接口说明
> 查询封闭区域列表

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/acsopac/v1/area/page`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acsopac/v1/area/page
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码（0<pageNo<=1000）|
|`pageSize`|`body`|`integer`|是|每页展示数目（0<pageSize<=1000）|
|`areaCode`|`body`|`string`|否|区域编号|
|`areaName`|`body`|`string`|否|区域名称，模糊查询，长度为1~32个字符,支持中英文字符、数字，不能包含 ?😕*<>#@$%^&|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 1000,
  "areaCode": "dsgrytryutuy",
  "areaName": "***区域"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败；|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总结果数|
|`+pageSize`|`body`|`number`|否|单页展示数据数目|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+list`|`body`|`object[]`|否|返回数据集合|
|`++areaId`|`body`|`number`|否|区域ID|
|`++areaCode`|`body`|`string`|否|区域编号|
|`++areaName`|`body`|`string`|否|区域名称|
|`++readerNum`|`body`|`number`|否|区域关联读卡器数量|
|`++cameraNum`|`body`|`number`|否|区域关联监控点数量|
|`++parkingNum`|`body`|`number`|否|区域关联停车场数量|
|`++netledDeviceNum`|`body`|`number`|否|区域关联LED设备数量|
|`++areaPersonNum`|`body`|`number`|否|区域实时人数|
|`++areaPersonInNum`|`body`|`number`|否|区域实时进入人次|
|`++areaPersonOutNum`|`body`|`number`|否|区域实时离开人次|
|`++areaCarNum`|`body`|`number`|否|区域实时车辆数|
|`++areaCarInNum`|`body`|`number`|否|区域实时进入车次|
|`++areaCarOutNum`|`body`|`number`|否|区域实时离开车次|
|`++areaAlarmNum`|`body`|`number`|否|区域实时报警数量|
|`++delayNumMax`|`body`|`string`|否|区域人数上限|
|`++delayTimeMax`|`body`|`string`|否|滞留时长上限|
|`++delayNumMin`|`body`|`string`|否|区域人数下限|
|`++timeToClear`|`body`|`string`|否|区域人数重置时间|
|`++hourToClear`|`body`|`number`|否|区域人数重置时间，小时数|
|`++minuteToClear`|`body`|`number`|否|区域人数重置时间，分钟数|
|`++invalidClearTime`|`body`|`string`|否|无效滞留清除时间上限|
|`++delayTimeMaxOfCar`|`body`|`string`|否|车辆滞留时长上限|
|`++maxPasstimesOfSingleday`|`body`|`string`|否|单日累计进入次数上限|
|`++maxInsideTimeOfSingleday`|`body`|`string`|否|单日累计进入时长上限|
|`++visitorSignOutTime`|`body`|`string`|否|访客最晚离场事件|
|`++visitorDelayTimeMax`|`body`|`string`|否|访客滞留时长上限|
|`++touristSignOutTime`|`body`|`string`|否|游客最晚离场事件|
|`++touristDelayTimeMax`|`body`|`string`|否|游客滞留时长上限|
|`++onWorkTime`|`body`|`string`|否|上班时段|
|`++offWorkTime`|`body`|`string`|否|下班时段|
|`++areaType`|`body`|`string`|否|区域类型，0：自建区域，1：同步区域|
|`++parentCode`|`body`|`string`|否|父区域编号|
|`++isParentArea`|`body`|`string`|否|是否包含子区域字段,0:否，1：是|
|`++timeToClearAlarm`|`body`|`string`|否|报警清零时间|
|`++forbiddenPlanCode`|`body`|`string`|否|禁入时段计划模板编号|
|`++workPlanCode`|`body`|`string`|否|工作时段计划模板编号|
|`++minuteOfOutLeave`|`body`|`string`|否|允许最大外出时间，分钟|
|`++createTime`|`body`|`number`|否|创建时间|
|`++updateTime`|`body`|`number`|否|更新时间|
|`++isDelete`|`body`|`number`|否|是否删除:0-未删除；1-已删除|
|`++choosedProperty`|`body`|`string`|否|已选属性|
|`++isMain`|`body`|`number`|否|是否默认区域，主区域，0-否；1-是|
|`+lastPage`|`body`|`boolean`|否|是否是最后一页|
