# 临时作业预约分页查询

> 接口说明
> 该接口主要是用于查询已创建的临时作业列表。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center 1.3.0及以上版本|
|接口地址|`/api/acsopac/v1/temporary/work/pageQuery`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acsopac/v1/temporary/work/pageQuery
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码（0<pageNo<=1000）|
|`pageSize`|`body`|`integer`|是|每页展示数目（0<pageSize<=1000）|
|`workName`|`body`|`string`|否|临时作业名称，长度为1~32个字符,支持中英文字符、数字，不能包含 /😗?<>|
|`areaCode`|`body`|`string`|否|区域编号，可通过查询封闭区域列表接口获取。|
|`principalPersonId`|`body`|`string`|否|作业负责人personId，可通过人员信息查询接口，如查询人员列表v2接口获取。|
|`workStartTime`|`body`|`string`|否|作业开始时间，时间参数需满足ISO8601格式如：“2022-10-24T17:30:08.000+08:00”|
|`workEndTime`|`body`|`string`|否|作业结束时间，时间参数需满足ISO8601格式如："2022-10-24T17:30:08.000+08:00"|

补充说明:
- 注：workStartTime和workEndTime要么同时为空，要么同时不为空

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 20,
  "workName": "临时作业测试8",
  "areaCode": "28790ece-fd64-4c86-b800-f100be0CC3e3",
  "principalPersonId": "bb1188a70694402fa10fedff0463051f",
  "workStartTime": "2022-10-15T17:30:08.000+08:00",
  "workEndTime": "2022-10-17T18:30:08.000+08:00"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|返回码|
|`msg`|`body`|`string`|是|返回消息|
|`data`|`body`|`object`|是|返回数据|
|`+total`|`body`|`number`|是|总结果数|
|`+pageSize`|`body`|`number`|是|单页展示数据数目|
|`+pageNo`|`body`|`number`|是|当前页|
|`+list`|`body`|`object[]`|是|数据集合|
|`++workId`|`body`|`number`|否|临时作业编号|
|`++areaCode`|`body`|`string`|否|区域编号|
|`++areaName`|`body`|`string`|否|区域名称|
|`++workName`|`body`|`string`|否|作业名称|
|`++workDesc`|`body`|`string`|否|作业描述|
|`++principalPersonId`|`body`|`string`|否|负责人personId|
|`++principalPersonName`|`body`|`string`|否|负责人姓名|
|`++peerPersonId`|`body`|`string`|否|随行人personId|
|`++peerPersonNames`|`body`|`string`|否|随行人姓名|
|`++workStartTime`|`body`|`string`|否|作业开始时间|
|`++workEndTime`|`body`|`string`|否|作业结束时间|
|`++reserveSource`|`body`|`string`|否|预约来源（web/api）|
|`++creater`|`body`|`string`|否|创建人personId|
|`++createrName`|`body`|`string`|否|创建人姓名|
|`++authStatus`|`body`|`number`|否|权限状态，0:待配置 1:待下载 2:已下载 3:已失效 4:已删除 5:更新待下发|
|`++workStatus`|`body`|`number`|否|作业状态，0:未开始 1:作业中 2:已结束|
|`++createTime`|`body`|`string`|否|创建时间|
|`++updateTime`|`body`|`string`|否|更新时间|
|`+lastPage`|`body`|`boolean`|是|是否最后一页|
