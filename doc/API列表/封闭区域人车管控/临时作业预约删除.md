# 临时作业预约删除

> 接口说明
> 该接口主要是应用于删除已创建的临时作业。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center 1.3.0及以上版本|
|接口地址|`/api/acsopac/v1/temporary/work/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acsopac/v1/temporary/work/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`workId`|`body`|`integer`|是|临时作业编号，临时作业创建接口返回，也可以通过临时作业预约分页查询接口获取。|

## 请求参数示例
```json
{
  "workId": 8
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码|
|`msg`|`body`|`string`|否|返回消息|
|`data`|`body`|`string`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```
