# 获取出入事件的图片

> 接口说明
> 通过该接口根据输入的图片的相对Uri地址，可以获取到具体的数据数据。
> 此接口返回Http Status 302 重定向，所以参数返回与其他接口有些差异。本接口的应答描述是图片存储服务器的返回信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/vis/v1/accessEvent/eventPictures`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/vis/v1/accessEvent/eventPictures
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`svrIndexCode`|`body`|`string`|是|存储接入服务（图片服务器唯一标识），获取picUri处会提供此参数|
|`picUri`|`body`|`string`|是|图片地址|

## 请求参数示例
```json
{
  "svrIndexCode": "a517ab45-09bf-423c-83fe-fea3b6332292",
  "picUri": "/pic?=d62i7f6e*6a7i125-c838b9--a8c67dea96e65icb1*=sd*=5dpi*=1dpi*m2i1t=4ed35444bb4s=-39z422d3"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，附录E.2.4 可视对讲管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|
