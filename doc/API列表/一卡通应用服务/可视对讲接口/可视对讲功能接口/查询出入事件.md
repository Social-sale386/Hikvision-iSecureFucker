# 查询出入事件

> 接口说明
> 该接口可以查询发生在门口机、围墙机上的人员出入事件，支持多个维度来查询，支持按时间、人员、设备、卡号、事件类型五个维度来查询；其中按事件类型来查询的方式，如果查询不到事件，存在两种情况，一种是该类型的事件没有发生过，所以查询不到，还有一种情况，该类型的事件发生过，但是由于可视对讲管理组件对该事件类型订阅配置处于关闭状态，所以不会存储该类型的事件，导致查询不到，对于这种情况，需要到可视对讲管理组件中，将该事件类型的订阅配置打开。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/vis/v1/accessEvent/events`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/vis/v1/accessEvent/events
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`startTime`|`body`|`string`|是|开始时间（事件开始时间，采用ISO8601时间格式，与endTime配对使用，不能单独使用）|
|`endTime`|`body`|`string`|是|结束时间（事件结束时间，采用ISO8601时间格式，与startTime配对使用，不能单独使用）|
|`eventTypes`|`body`|`string`|否|事件类型（10进制，需根据事件类型定义表中类型进行转换。参考[附录D.2.2.1 可视对讲事件类型](/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录D%20事件列表-附录D2 一卡通事件-附录D2.2 可视对讲事件#df39f22a)|
|`personIds`|`body`|`string`|否|人员ID集（最大支持10个人员搜索），获取人员列表v2接口返回报文中的personId字段|
|`cardNos`|`body`|`string`|否|卡号集（最大支持10张卡片搜索），多个以逗号隔开， 获取卡片列表接口获取返回参数cardNo|
|`deviceIds`|`body`|`string`|否|设备编号集（最大支持10个设备集搜索），多个以逗号隔开， 查询可视对讲设备列表v2接口获取返回参数indexCode|
|`sort`|`body`|`string`|否|排序字段（支持personName、deviceName、eventTime填写排序的字段名称，与order配对使用，不能单独使用，例如：”personName”）|
|`order`|`body`|`string`|否|升/降序（指定排序字段是使用升序（asc）还是降序（desc），与sort配对使用，不能单独使用，例如：”asc”）|
|`pageNo`|`body`|`integer`|是|起始页（pageNo>0）|
|`pageSize`|`body`|`integer`|是|当前页大小（0<pageSize<=1000）|

{

  "personIds": "e9b94466-4808-440b-9af0-79CC4e52893b,e9b94466-4808-440b-9af0-79CC4e52893c",
  "cardNos": "11111111111,22222222222",

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0：成功，其它参考附录E.2.4 可视对讲管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据（具体接口返回信息结构，此处为事件分页数据）|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+pageNo`|`body`|`number`|否|当前页|
|`+total`|`body`|`number`|否|结果总数|
|`+totalPage`|`body`|`number`|否|总页数|
|`+list`|`body`|`object[]`|否|出入事件|
|`++eventId`|`body`|`string`|否|事件唯一编号|
|`++eventType`|`body`|`number`|否|事件类型，参考[附录D.2.2.1 可视对讲事件类型](/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录D%20事件列表-附录D2 一卡通事件-附录D2.2 可视对讲事件#df39f22a)|
|`++eventName`|`body`|`string`|否|事件名称|
|`++eventTime`|`body`|`string`|否|事件产生时间|
|`++personId`|`body`|`string`|否|人员唯一编码|
|`++cardNo`|`body`|`string`|否|卡号|
|`++personName`|`body`|`string`|否|人员名称|
|`++orgIndexCode`|`body`|`string`|否|组织编码|
|`++orgName`|`body`|`string`|否|组织名称|
|`++deviceName`|`body`|`string`|否|设备名称|
|`++deviceIndexCode`|`body`|`string`|否|设备编码|
|`++svrIndexCode`|`body`|`string`|否|图片存储服务的唯一标识|
|`++picUrl`|`body`|`string`|否|抓拍图片地址|
