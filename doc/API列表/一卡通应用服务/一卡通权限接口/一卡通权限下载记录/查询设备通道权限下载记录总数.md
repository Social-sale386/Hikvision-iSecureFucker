# 查询设备通道权限下载记录总数

> 接口说明
> 根据查询条件查询设备通道的下载记录总数。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/acps/v1/download_record/channel/total/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acps/v1/download_record/channel/total/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`taskId`|`body`|`string`|否|下载任务唯一标识，创建任务时获得|
|`resourceInfos`|`body`|`object[]`|否|设备通道对象列表|
|`+resourceIndexCode`|`Body`|`string`|是|资源的唯一标识，资源为设备，查询资源列表v2接口获取返回报文中的personId字段|
|`+resourceType`|`Body`|`string`|是|资源类型，详见附录A.50 设备通道类型|
|`+channelNos`|`Body`|`number[]`|否|通道号|
|`taskTypes`|`body`|`integer[]`|否|下载任务类型|
|`downloadResult`|`body`|`integer`|否|下载结果，|
|`downloadStartTime`|`body`|`object`|否|下载开始时间查询对象|
|`+startTime`|`Body`|`string`|否|开始日期，采用ISO8601时间格式，最大长度32个字符，如2018-09-03T17:30:08+08:00|
|`+endTime`|`Body`|`string`|否|结束日期，采用ISO8601时间格式，最大长度32个字符，如2018-09-03T17:30:08+08:00|
|`downloadEndTime`|`body`|`object`|否|下载结束时间查询对象|
|`+startTime`|`Body`|`string`|否|结束日期，采用ISO8601时间格式，最大长度32个字符，如2018-09-03T17:30:08+08:00|
|`+endTime`|`Body`|`string`|否|结束日期，采用ISO8601时间格式，最大长度32个字符，如2018-09-03T17:30:08+08:00|

补充说明:
- 0成功，
- 1失败，
- 2部分成功

## 请求参数示例
```json
{
  "taskId": "02e66ac47a03481fa0f304fa3acfa773",
  "resourceInfos": [
    {
      "resourceIndexCode": "ef72fd5e7ec341a99a37ecd99f90af6e",
      "resourceType": "acsDevice",
      "channelNos": [
        1
      ]
    }
  ],
  "taskTypes": [
    1
  ],
  "downloadResult": 1,
  "downloadStartTime": {
    "startTime": "2018-09-03T17:30:08+08:00",
    "endTime": "2018-09-03T18:30:08+08:00"
  },
  "downloadEndTime": {
    "startTime": "2018-09-04T17:30:08+08:00",
    "endTime": "2018-09-04T17:30:08+08:00"
  }
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.3 出入控制权限错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`integer`|否|下载记录总数|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": 2
}
```
