# 查询正在下载的任务列表

> 接口说明
> 该接口用于查询正在下载的任务编号列表，只能查询由组件创建的任务.

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/acps/v1/authDownload/task/list`|
|请求方法|`POST`|

## 快速调用
```text
POST /api/acps/v1/authDownload/task/list
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.3 出入控制权限错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回结果|
|`+total`|`body`|`number`|否|总条数|
|`+list`|`body`|`string[]`|否|任务Id列表|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 2,
    "list": [
      "8df26960016e4a15b6347894fc19826d"
    ]
  }
}
```
