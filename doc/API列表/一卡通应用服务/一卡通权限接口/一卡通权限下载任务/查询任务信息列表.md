# 查询任务信息列表

> 接口说明
> 该接口用于分页查询任务的详细信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/acps/v1/authDownload/task/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acps/v1/authDownload/task/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`authTaskIds`|`body`|`string[]`|否|下载任务唯一标志，创建任务时获取|
|`authTaskStatusList`|`body`|`integer[]`|否|任务状态，0-未下载，1-下载中，2-下载完成|
|`operationTypeList`|`body`|`integer[]`|否|下载任务操作类型，详见附录A.82 任务操作类型|
|`taskTypeList`|`body`|`integer[]`|否|下载任务类型列表，详见附录A.53 任务下载类型说明|
|`pageNo`|`body`|`integer`|是|页码，pageNo>0|
|`pageSize`|`body`|`integer`|是|分页大小，0<pageSize<=1000|

## 请求参数示例
```json
{
  "authTaskIds": [
    "46f2328384064eec905fcd77d64c7bb0"
  ],
  "authTaskStatusList": [
    1
  ],
  "operationTypeList": [
    1
  ],
  "taskTypeList": [
    4
  ],
  "pageNo": 1,
  "pageSize": 10
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.3 出入控制权限错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总条数|
|`+pageSize`|`body`|`number`|否|分页大小|
|`+pageNo`|`body`|`number`|否|页码|
|`+list`|`body`|`object[]`|否|任务详情列表|
|`++authTaskId`|`body`|`string`|否|任务ID|
|`++createTime`|`body`|`string`|否|任务创建时间，采用ISO8601时间格式，如2019-09-03T17:30:08.000+08:00|
|`++startTime`|`body`|`string`|否|任务开始时间，采用ISO8601时间格式，如2019-09-03T17:30:08.000+08:00|
|`++endTime`|`body`|`string`|否|任务结束时间，采用ISO8601时间格式，如2019-09-03T17:30:08.000+08:00|
|`++taskType`|`body`|`number`|否|下载任务类型，详见附录A.53 任务下载类型说明|
|`++taskStatus`|`body`|`number`|否|任务状态，0-未下载，1-下载中，2-下载完成|
|`++operationType`|`body`|`number`|否|下载任务操作类型，详见附录A.82 任务操作类型|
|`++callbackUrl`|`body`|`string`|否|回调地址|

```json
        "operationType": 1,

      }

  }

```
