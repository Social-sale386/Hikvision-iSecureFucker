# 删除未开始的下载任务

> 接口说明
> 该接口用于删除创建的下载任务，已经开始下载的任务不能删除，只能由创建任务的组件触发。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/acps/v1/authDownload/task/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acps/v1/authDownload/task/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`taskId`|`body`|`string`|是|下载任务唯一标识，创建任务时获取|

## 请求参数示例
```json
{
  "taskId": "02e66ac47a03481fa0f304fa3acfa773"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.3 出入控制权限错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|无返回结果|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
