# 查询下载任务进度

> 接口说明
> 根据任务ID查询任务的下载进度，只能查询由组件创建的任务，进度信息包含任务总体下载进度及各个资源的下载进度。
> 建议该接口调用频率每3-5秒查询一次任务的下载进度。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/acps/v1/authDownload/task/progress`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acps/v1/authDownload/task/progress
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`taskId`|`body`|`string`|是|下载任务唯一标识，创建任务时获取|

## 请求参数示例
```json
{
  "taskId": "02e66ac47a03481fa0f304fa3acfa773"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.3 出入控制权限错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回结果|
|`+tagId`|`body`|`string`|否|标签,用于区分不同业务组件，建议使用组件标识。只支持1-32个数字和小写字母。|
|`+taskId`|`body`|`string`|否|下载任务唯一标识|
|`+taskType`|`body`|`number`|否|下载任务类型 1：卡片 4：人脸 其他参考附录A.53 任务下载类型说明|
|`+taskOptType`|`body`|`number`|否|任务操作类型|
|`+startTime`|`body`|`string`|否|下载开始时间|
|`+totalPercent`|`body`|`number`|否|下载总进度|
|`+leftTime`|`body`|`number`|否|下载剩余时间|
|`+consumeTime`|`body`|`number`|否|下载总用时，单位：秒|
|`+isDownloadFinished`|`body`|`boolean`|否|下载是否结束|
|`+resourceDownloadProgress`|`body`|`object[]`|否|资源下载详情|
|`++resourceInfo`|`body`|`object`|否|资源对象|
|`+++resourceIndexCode`|`body`|`string`|否|资源的唯一标识|
|`+++resourceType`|`body`|`string`|否|资源类型，详见附录A.50 设备通道类型|
|`+++channelNos`|`body`|`number[]`|否|资源通道号|
|`+++channelIndexCodes`|`body`|`string[]`|否|资源通道唯一标识（在权限下载和配置接口入参时对应查询接口会返回，不作为查询接口的查询条件）|
|`++startTime`|`body`|`string`|否|下载开始时间|
|`++updateTime`|`body`|`string`|否|下载进度更新时间|
|`++endTime`|`body`|`string`|否|下载结束时间|
|`++leftTime`|`body`|`number`|否|下载剩余时间，单位秒|
|`++errorCode`|`body`|`string`|否|下载错误码，参考附录E.2.3 出入控制权限错误码|
|`++downloadResult`|`body`|`number`|否|下载结果（当下载结束时有值，0:成功,1:失败，2：部分失败）|
|`++consumeTime`|`body`|`number`|否|下载用时，单位秒|
|`++downloadStatus`|`body`|`string`|否|下载状态|
|`++isDownloadFinished`|`body`|`boolean`|否|是否下载结束|
|`++downloadPercent`|`body`|`number`|否|下载百分比|
|`++totalPersonCount`|`body`|`number`|否|下载总人数|
|`++downloadPersonCount`|`body`|`number`|否|已下发人员数量|
|`++successedPersonCount`|`body`|`number`|否|下载成功人员数量|
|`++failedPersonCount`|`body`|`number`|否|下载失败人员数量|

补充说明:
- 0:初始化下载,
- 1:异动下载,
- 2:指定下载-普通任务,
- 3:指定下载-快速任务,
- 4:自动下载,
- 5:同步下载
