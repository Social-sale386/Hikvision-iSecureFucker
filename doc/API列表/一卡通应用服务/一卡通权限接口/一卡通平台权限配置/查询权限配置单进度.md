# 查询权限配置单进度

> 接口说明
> 根据配置单编号查询配置单的配置进度，只能查询组件自身创建的配置单。
> 建议该接口调用频率每3-5秒查询一次进度。
> V1.5.0版本返回字段新增isConfigFinished 表示配置数据入库成功

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/acps/v1/auth_config/rate/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acps/v1/auth_config/rate/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`taskId`|`body`|`string`|是|权限配置单编号，从添加权限配置接口，或删除权限配置接口获取返回参数orgIndexCode|

## 请求参数示例
```json
{
  "taskId": "02e66ac47a03481fa0f304fa3acfa773"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.3 出入控制权限错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+tagId`|`body`|`string`|否|标签|
|`+taskId`|`body`|`string`|否|配置单编号|
|`+startTime`|`body`|`string`|否|配置开始时间，采用ISO8601时间格式，最大长度32个字符，如2019-09-03T17:30:08.000+08:00|
|`+endTime`|`body`|`string`|否|配置结束时间，采用ISO8601时间格式，最大长度32个字符，如2019-09-03T17:30:08.000+08:00|
|`+percent`|`body`|`number`|否|任务进度，范围0~100|
|`+leftTime`|`body`|`number`|否|剩余时间，单位秒|
|`+isFinished`|`body`|`boolean`|否|下载是否结束|
|`+isConfigFinished`|`body`|`boolean`|否|是否配置数据入库结束,该字段仅表示配置数据入库结束，还需进行人员权限计算|
|`+successedNum`|`body`|`number`|否|配置成功数量|
|`+totalNum`|`body`|`number`|否|配置单总数据量|
|`+failedNum`|`body`|`number`|否|配置失败数量|
|`+repeatedNum`|`body`|`number`|否|重复配置数量，指相同的配置数据重复提交的情况|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "tagId": "acs",
    "taskId": "02e66ac47a03481fa0f304fa3acfa773",
    "startTime": "2019-09-03T17:30:08.000+08:00",
    "endTime": "2019-09-03T17:30:18.000+08:00",
    "percent": 100,
    "leftTime": 0,
    "isFinished": true,
    "totalNum": 500,
    "successedNum": 400,
    "failedNum": 0,
    "repeatedNum": 100
  }
}
```
