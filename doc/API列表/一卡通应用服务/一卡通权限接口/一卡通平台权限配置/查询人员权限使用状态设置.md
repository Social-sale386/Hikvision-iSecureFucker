# 查询人员权限使用状态设置

> 接口说明
> 支持按人员查询人员权限使用状态

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5.0及以上版本|
|接口地址|`/api/acps/v1/authConfig/personAuth/useStatus/list`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acps/v1/authConfig/personAuth/useStatus/list
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`personIds`|`body`|`string[]`|否|人员ID列表，查询人员列表v2接口获取返回报文中的personId字段|
|`useStatus`|`body`|`integer`|否|权限使用状态，|
|`pageNo`|`body`|`integer`|是|页码,pageNo>0|
|`pageSize`|`body`|`integer`|是|页大小,0<pageSize<=1000|

补充说明:
- 0-权限有效，1-权限禁用

## 请求参数示例
```json
{
  "personIds": [
    "37835310-f786-451a-8eb5-5d6c43b827cc"
  ],
  "useStatus": 1,
  "pageNo": 1,
  "pageSize": 20
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.3 出入控制权限错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总条数|
|`+pageSize`|`body`|`number`|否|页大小|
|`+pageNo`|`body`|`number`|否|页码|
|`+list`|`body`|`object[]`|否|人员权限全局状态列表|
|`++personId`|`body`|`string`|否|人员ID，查询人员列表v2接口获取返回报文中的personId字段|
|`++useStatus`|`body`|`number`|否|权限使用状态(0-权限有效，1-权限禁用)|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "pageSize": 20,
    "pageNo": 1,
    "list": [
      {
        "personId": "624 ce380 - d9b4 - 4 d97 - 9654 - 9 a49c1a58f1c",
        "useStatus": 1
      }
    ]
  }
}
```
