# 查询消费记录

> 接口说明
> 该接口可以查询人员消费的记录，支持按照人员唯一标示、卡号、消费模式、上报时间段查询人员消费记录信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/cems/v1/consume/log/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/cems/v1/consume/log/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`startTime`|`body`|`string`|是|记录开始时间，ISO8601格式yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：|
|`endTime`|`body`|`string`|是|记录结束时间，ISO8601格式yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：|
|`chargeTypes`|`body`|`string`|否|消费模式：|
|`personId`|`body`|`string`|否|人员唯一ID|
|`cardNo`|`body`|`string`|否|人员卡号|
|`pageNo`|`body`|`integer`|是|当前页码，pageNo>0|
|`pageSize`|`body`|`integer`|是|页大小，0<pageSize<=1000|

补充说明:
- 2018-07-26T15:00:00 + 08:00
- 开始时间和结束时间最大跨度控制在30天
- 2018-07-26T15:00:00 + 08:00
- 开始时间和结束时间最大跨度控制在30天
- 1：计次模式
- 3：金额模式
- 多个用“,”隔开

## 请求参数示例
```json
{
  "startTime": "2018-07-26T15:00:00+08:00",
  "endTime": "2018-07-26T15:00:00+08:00",
  "chargeTypes": "1,3",
  "personId": "HZ12345",
  "cardNo": "12333355554",
  "pageNo": 1,
  "pageSize": 20
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询总数|
|`+page`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|页大小|
|`+firstPage`|`body`|`boolean`|否|是否为第一页|
|`+lastPage`|`body`|`boolean`|否|是否为最后一页|
|`+hasPreviousPage`|`body`|`boolean`|否|是否有上一页|
|`+hasNextPage`|`body`|`boolean`|否|是否有下一页|
|`+totalPage`|`body`|`number`|否|总页数|
|`+rows`|`body`|`object[]`|否|返回对象列表|
|`++chargeNo`|`body`|`string`|否|账单流水号|
|`++cardNo`|`body`|`string`|否|卡号，由刷卡终端设备触发读取的卡号|
|`++personId`|`body`|`string`|否|人员唯一ID|
|`++preAccount`|`body`|`number`|否|扣款前余额，单位：分|
|`++merchantName`|`body`|`string`|否|消费商户|
|`++deduction`|`body`|`number`|否|扣款金额，单位：分|
|`++balance`|`body`|`number`|否|当前余额，单位：分|
|`++debitTime`|`body`|`string`|否|交易时间，ISO8601格式：|
|`++personName`|`body`|`string`|否|人员姓名|
|`++orgName`|`body`|`string`|否|人员所属组织名|
|`++consumeTerminalNo`|`body`|`string`|否|消费终端的终端号|
|`++times`|`body`|`number`|否|消费次数|
|`++chargeType`|`body`|`number`|否|消费模式：|

补充说明:
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：2018-07-26T15:00:00 + 08:00
- 1：计次模式
- 3：金额模式
