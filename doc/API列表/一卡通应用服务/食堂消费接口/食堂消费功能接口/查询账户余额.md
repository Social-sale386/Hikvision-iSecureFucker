# 查询账户余额

> 接口说明
> 该接口可以通过卡号或人员唯一ID查询账户余额信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/cems/v1/account/balance/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/cems/v1/account/balance/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`personId`|`body`|`string`|是|人员唯一Id， personId 和 cardNo 只要填一个即可，都填的话，以personId为准，根据查询人员列表v2接口获取返回报文中的personId字段|
|`cardNo`|`body`|`string`|是|卡号， personId 和 cardNo 只要填一个即可，都填的话，以personId为准，根据获取卡片列表接口获取返回报文中的cardId字段|

## 请求参数示例
```json
{
  "personId": "2eb8382f-ced5-4b4f-83ba-c4b64646aa91"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.6 食堂消费错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+personName`|`body`|`string`|否|账户所属人员名称|
|`+genAccount`|`body`|`number`|否|普通账户金额，单位：分|
|`+subAccount`|`body`|`number`|否|补贴账户金额，单位：分|
|`+totalAccount`|`body`|`number`|否|账户总金额，单位：分|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "personName": "张三",
    "genAccount": 100,
    "subAccount": 200,
    "totalAccount": 300
  }
}
```
