# 查询访客权限组

> 接口说明
> 该接口用于查询访客管理应用服务中已配置的权限组信息。访客登记时，本接口的结果可以做为访客登记时的入参使用。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/visitor/v1/privilege/group`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/privilege/group
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`name`|`body`|`string`|否|访客权限组名称|
|`pageNo`|`body`|`integer`|是|当前页码，要求大于0|
|`pageSize`|`body`|`integer`|是|每页记录总数，要求大于0且不超过1000|

## 请求参数示例
```json
{
  "name": "123",
  "pageNo": 1,
  "pageSize": 10
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+list`|`body`|`object[]`|否|权限组对象列表|
|`++privilegeGroupId`|`body`|`string`|否|权限组ID,用于访客登记时对访客授权|
|`++privilegeGroupName`|`body`|`string`|否|权限组名称|
|`++remark`|`body`|`string`|否|描述|
|`++isDefault`|`body`|`number`|否|是否是默认权限组，0-是，1-否|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "pageSize": 24,
    "pageNo": 1,
    "list": [
      {
        "privilegeGroupId": "xxd-00e-kkk-553",
        "privilegeGroupName": "访客权限组1",
        "remark": "访客权限组描述",
        "isDefault": 0
      }
    ]
  }
}
```
