# 获取访客权限组权限项

> 接口说明
> 获取访客权限组权限项

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1及以上版本|
|接口地址|`/api/v1/privilege/itemInfo`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/v1/privilege/itemInfo
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`privilegeGroupId`|`body`|`string`|是|访客权限组ID|

## 请求参数示例
```json
{
  "privilegeGroupId": "abcd"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|

|`++privilegeItemResType`|`body`|`string`|否|权限项目类型:1-门禁，2-梯控，3-门口机，4-停车场，5-人脸监控，6-智能柜,7-停车场车道|
|`++privilegeItemResRegion`|`body`|`string`|否|权限项目所属区域ID|
|`++privilegeItemIndexCode`|`body`|`string`|否|权限项资源indexCode|
