# 查询来访单位

> 接口说明
> 查询来访单位

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1及以上版本|
|接口地址|`/api/v1/visitorData/workUnitList`|
|请求方法|`GET`|

## 快速调用
```text
GET /api/v1/visitorData/workUnitList
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`object[]`|否|返回数据(具体接口返回信息结构)|
|`+workUnitId`|`body`|`string`|否|来访单位Id|
|`+workUnitName`|`body`|`string`|否|来访单位名称|

## 返回参数示例
```json
{
  "code": "1",
  "msg": "ERROR",
  "data": [
    {
      "workUnitId": "111",
      "workUnitName": "CS"
    }
  ]
}
```
