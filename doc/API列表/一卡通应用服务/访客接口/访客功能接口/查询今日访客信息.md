# 查询今日访客信息

> 接口说明
> 获取今日访客信息包含：今日来访总人数（已签离人数，未签离人数），预约人数

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1及以上版本|
|接口地址|`/api/v1/visitorInfo/count`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/v1/visitorInfo/count
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkId`|`body`|`string`|否|园区编号,参数不填则查询所有园区|

## 请求参数示例
```json
{
  "parkId": "1232132"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`object`|否|返回数据(具体接口返回信息结构)|
|`+appointmentTotal`|`body`|`number`|否|预约人数（兼容低版本的字段）|
|`+notSignOutTotal`|`body`|`number`|否|未签离人数 （兼容低版本的字段）|
|`+signOutTotal`|`body`|`number`|否|已签离人数（兼容低版本的字段）|
|`+signTotal`|`body`|`number`|否|今日来访总人数（兼容低版本的字段）|
|`+orderCount`|`body`|`string`|否|预约未到访人数（正常+迟到（1,2））|
|`+visitCount`|`body`|`string`|否|在访人数（来访中(8)）|
|`+visitCountForTemp`|`body`|`string`|否|预约到访人数|
|`+visitCountForOrder`|`body`|`string`|否|未预约到访人数|
|`+signOutCount`|`body`|`string`|否|已签离人数（已签离(5,6)）|
|`+notSignOutCount`|`body`|`string`|否|滞留人数 （7）|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "error",
  "data": {
    "appointmentTotal": 1,
    "notSignOutTotal": 1,
    "signOutTotal": 1,
    "signTotal": 1,
    "orderCount": "1",
    "visitCount": "1",
    "visitCountForTemp": "1",
    "visitCountForOrder": "1",
    "signOutCount": "1",
    "notSignOutCount": "1"
  }
}
```
