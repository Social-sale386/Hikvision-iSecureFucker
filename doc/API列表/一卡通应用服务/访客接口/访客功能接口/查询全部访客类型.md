# 查询全部访客类型

> 接口说明
> 查询全部访客类型

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1及以上版本|
|接口地址|`/api/v1/visitorData/visitorTypeList`|
|请求方法|`GET`|

## 快速调用
```text
GET /api/v1/visitorData/visitorTypeList
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`object[]`|否|返回数据(具体接口返回信息结构)|
|`+parkId`|`body`|`string`|否|园区id|
|`+remitRegister`|`body`|`string`|否|是否开启预约确认免登记：1:关闭 2：开启|
|`+visitorTypeId`|`body`|`string`|否|访客类型Id|
|`+visitorTypeName`|`body`|`string`|否|访客类型名称|

## 返回参数示例
```json
{
  "code": "1",
  "msg": "3333",
  "data": [
    {
      "parkId": "1232131",
      "remitRegister": "1",
      "visitorTypeId": "12333",
      "visitorTypeName": "cs"
    }
  ]
}
```
