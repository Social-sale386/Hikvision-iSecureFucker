# 查询园区列表

> 接口说明
> 查询平台已配置的园区信息，包括入园须知。
> 注意：此接口从访客管理组件V1.10.0起支持。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center v1及以上版本|
|接口地址|`/api/v1/visitorData/parkList`|
|请求方法|`GET`|

## 快速调用
```text
GET /api/v1/visitorData/parkList
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`object[]`|否|返回数据(具体接口返回信息结构)|
|`+latitude`|`body`|`string`|否|纬度|
|`+longitude`|`body`|`string`|否|经度|
|`+parkId`|`body`|`string`|否|园区编号|
|`+parkName`|`body`|`string`|否|园区名称|
|`+parkDesc`|`body`|`string`|否|备注说明|
|`+regionCode`|`body`|`string`|否|关联区域编号|
|`+regionName`|`body`|`string`|否|关联区域名称|
|`+navImgSvrCode`|`body`|`string`|否|导航地图图片服务器编号|
|`+navigationMapUrl`|`body`|`string`|否|导航地图URL|
|`+isDefault`|`body`|`number`|否|是否默认园区 1=默认，0=不是默认园区|
|`+noteAdmissionId`|`body`|`string`|否|入园须知编号|
|`+noteAdmissionTitle`|`body`|`string`|否|入园须知标题|
|`+noteAdmissionContent`|`body`|`string`|否|入园须知内容|
|`+noteAdmissionStatus`|`body`|`number`|否|是否启用入园须知 1=启用 0=关闭|
|`+parkArea`|`body`|`string`|否|园区行政区划|
|`+parkAddr`|`body`|`string`|否|园区详细地址|
|`+h5ImgSvrCode`|`body`|`string`|否|H5背景图片图片服务器编号|
|`+h5BgUrl`|`body`|`string`|否|H5背景图片URL|
