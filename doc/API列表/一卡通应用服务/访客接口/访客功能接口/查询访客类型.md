# 查询访客类型

> 接口说明
> 查询访客类型

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.2.0及以上版本|
|接口地址|`/api/visitor/v1/visitorTypeInfo`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/visitorTypeInfo
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码|
|`pageSize`|`body`|`integer`|是|每页记录总数,0<pageSize<=1000|
|`parkId`|`body`|`string`|否|园区id|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 20,
  "parkId": "abcde12345xyz"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|访客类型记录|
|`+total`|`body`|`number`|否|总记录数|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|页大小|
|`+list`|`body`|`object[]`|否|访客类型记录列表|
|`++visitorTypeId`|`body`|`string`|否|访客类型Id|
|`++visitorTypeName`|`body`|`string`|否|访客类型名称|
|`++createTime`|`body`|`string`|否|创建时间|
|`++status`|`body`|`number`|否|状态|
|`++parkId`|`body`|`string`|否|园区id|
|`++remitRegister`|`body`|`string`|否|是否开启预约确认免登记：1:关闭 2：开启|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "pageNo": 1,
    "pageSize": 20,
    "list": [
      {
        "visitorTypeId": "321654987",
        "visitorTypeName": "VIP",
        "createTime": "2020-10-18 15:00:00",
        "status": 0,
        "parkId": "aaaaaa",
        "remitRegister": "1"
      }
    ]
  }
}
```
