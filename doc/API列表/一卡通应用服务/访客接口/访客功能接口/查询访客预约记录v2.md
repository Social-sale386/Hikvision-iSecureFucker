# 查询访客预约记录v2

> 接口说明
> 该接口用于查询预约记录，可以根据查询条件：被访人唯一标识、预计来访时间范围、预计离开时间范围、访客联系电话、预约记录ID、访客验证码等条件进行组合查询。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/visitor/v2/appointment/records`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v2/appointment/records
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`receptionistId`|`body`|`string`|否|被访人唯一标识,查询人员列表v2接口获取返回报文中的personId字段|
|`visitorName`|`body`|`string`|否|访客姓名，支持中英文字符，不能包含 ’ / \ : * ? " < >|
|`phoneNo`|`body`|`string`|否|联系电话建议填写手机号码|
|`visitStartTimeBegin`|`body`|`string`|否|时间参数需满足ISO8601格式：|
|`visitStartTimeEnd`|`body`|`string`|否|时间参数需满足ISO8601格式：|
|`visitEndTimeBegin`|`body`|`string`|否|时间参数需满足ISO8601格式：|
|`visitEndTimeEnd`|`body`|`string`|否|时间参数需满足ISO8601格式：|
|`pageNo`|`body`|`integer`|是|pageNo要求大于0不超过2147483647|
|`pageSize`|`body`|`integer`|是|pageSize要求大于0且不超过1000|
|`verificationCode`|`body`|`string`|否|访客验证码为访客管理组件提供，用于访客在访客机登记的凭证|
|`visitorStatusList`|`body`|`integer[]`|否|访客状态集合,-1:待提交、0:待审核、1：正常、2：迟到、3：失效、4：审核退回、5：超期自动签离,6：已签离,7：超期未签离,8：已到达、9：审核失效、10：邀约中、11:邀约失效、15：邀约拒绝|
|`QRCode`|`body`|`string`|否|访客二维码内容,支持动态二维码或者静态二维码|
|`orderId`|`body`|`string`|否|预约记录ID，可从访客预约v2接口获取返回报文中的appointRecordId字段|
|`cardNo`|`body`|`string`|否|卡号(支持IC卡号码/二维码号码/身份证号码/身份证序列号等不同类型卡号查询)|
|`identityNum`|`body`|`string`|否|证件号码|
|`parkId`|`body`|`string`|否|园区编号|
|`plateNo`|`body`|`string`|否|访客车牌号|
|`receptionistCode`|`body`|`string`|否|被访人所属组织编号|
|`visitUpdateTimeBegin`|`body`|`string`|否|更新时间查询时间段条件的开始时间|
|`visitUpdateTimeEnd`|`body`|`string`|否|更新来访时间查询时间段条件的结束时间|
|`visitorInfoId`|`body`|`string`|否|访客信息唯一标识|

补充说明:
- yyyy-MM-ddTHH:mm:ss+当前时区，
- 例如北京时间：
- 2018-07-26T15:00:00 + 08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，
- 例如北京时间：
- 2018-07-26T15:00:00 + 08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，
- 例如北京时间：
- 2018-07-26T15:00:00 + 08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，
- 例如北京时间：
- 2018-07-26T15:00:00 + 08:00

|---|---|---|---|---|

|`++visitStartTime`|`body`|`string`|否|时间参数需满足ISO8601格式：|

|`++visitorId`|`body`|`string`|否|访客id|
|`++verificationCode`|`body`|`string`|否|访客验证码，用于访客在访客机登记的凭证|

|`++birthplace`|`body`|`string`|否|籍贯|
