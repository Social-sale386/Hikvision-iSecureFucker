# 获取访客基础信息字段

> 接口说明
> 获取访客基础信息字段（包含自定义字段）信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/visitor/v1/visitorInfo/field`|
|请求方法|`POST`|

## 快速调用
```text
POST /api/visitor/v1/visitorInfo/field
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object[]`|否|返回信息|
|`+name`|`body`|`string`|否|参数名|
|`+id`|`body`|`string`|否|参数key|
|`+showType`|`body`|`string`|否|参数类型 文本框-1、下拉框-2、时间框-3、图片框-6、单选框-7、文本框-8、多选框-10|
|`+hint`|`body`|`string`|否|表单项的提示信息|
|`+regular`|`body`|`string`|否|表单项的校验正则|
|`+dataValue`|`body`|`object[]`|否|取值范围|
|`++text`|`body`|`string`|否|参数值说明|
|`++value`|`body`|`number`|否|参数值|
|`++selected`|`body`|`boolean`|否|是否默认|
|`+required`|`body`|`boolean`|否|是否必填|
|`+display`|`body`|`boolean`|否|是否可见|
|`+visitorFieldType`|`body`|`string`|否|字段类型,自定义字段-1、基础信息字段-0|
|`+fieldCatalog`|`body`|`string`|否|字段类别，common=常用信息，custom=自定义，antiepidemic=防疫|
|`+elementType`|`body`|`string`|否|前端显示控件类型 input=文本输入框，select=下拉选择，date_picker=日期选择，picture=图片框，radio=单选框，textarea=文本框，checkbox=多选框|
