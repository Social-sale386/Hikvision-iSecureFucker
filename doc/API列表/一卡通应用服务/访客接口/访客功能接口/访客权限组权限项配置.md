# 访客权限组权限项配置

> 接口说明
> 配置权限组中门禁，梯控，门口机，停车场,人脸监控,智能柜等权限项。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1及以上版本|
|接口地址|`/api/v1/privilege/item/add`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/v1/privilege/item/add
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`privilegeGroupId`|`body`|`string`|是|访客权限组ID|
|`privilegeItemType`|`body`|`string`|是|权限项的Type，1-门禁，2-梯控，3-门口机，4-停车场,5-人脸监控,6-智能柜,7-停车场车道|
|`privilegeItemIndexCodes`|`body`|`string[]`|是|权限项资源indexCode集合即：设备资源点code(门禁点id,梯控楼层id,门口机设备id,停车场id,人脸分组id,智能柜设备id,停车场车道id)|

## 请求参数示例
```json
{
  "privilegeGroupId": "abcd",
  "privilegeItemType": "1",
  "privilegeItemIndexCodes": [
    "aaaa"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`string`|否|返回数据(具体接口返回信息结构)|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "aaaa"
}
```
