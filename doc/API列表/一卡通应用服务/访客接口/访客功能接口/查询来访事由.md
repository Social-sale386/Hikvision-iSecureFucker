# 查询来访事由

> 接口说明
> 查询平台已配置的来访事由

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center v1及以上版本|
|接口地址|`/api/v1/visitorData/purposeList`|
|请求方法|`GET`|

## 快速调用
```text
GET /api/v1/visitorData/purposeList
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`object[]`|否|返回数据(具体接口返回信息结构)|
|`+purposeId`|`body`|`string`|否|来访事由Id|
|`+purposeName`|`body`|`string`|否|来访事由名称|

## 返回参数示例
```json
{
  "code": "1",
  "msg": "aaa",
  "data": [
    {
      "purposeId": "a111",
      "purposeName": "cs"
    }
  ]
}
```
