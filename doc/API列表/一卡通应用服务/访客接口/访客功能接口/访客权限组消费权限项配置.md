# 访客权限组消费权限项配置

> 接口说明
> 该接口用于配置权限组中消费权限项配置。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1及以上版本|
|接口地址|`/api/v1/privilege/cemsItem/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/v1/privilege/cemsItem/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`privilegeGroupId`|`body`|`string`|是|权限组ID|
|`cemsAuthEnable`|`body`|`integer`|是|是否开启消费权限。1：开启；0：关闭|
|`amountLimit`|`body`|`integer`|否|访客账户消费金额限制，单位为元,空时表示不进行限制，取值范围[0,100000]|
|`timesLimit`|`body`|`integer`|否|访客账户消费次数限制，空表示不进行限制，取值范围[0,9999]|

## 请求参数示例
```json
{
  "privilegeGroupId": "731bb582-fc49-11e7-83de-2ffbc13dcaba",
  "cemsAuthEnable": 1,
  "amountLimit": 1,
  "timesLimit": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`string`|否|返回数据(具体接口返回信息结构)|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "aaa"
}
```
