# 访客受限名单分组中配置人员数据

> 接口说明
> 向受限名单分组中添加或删除人员数据。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/visitor/v1/blocklist/person/set`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/blocklist/person/set
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`delPersonList`|`body`|`string[]`|否|待删除受限名单人员id集合，可从查询访客受限名单分组中人员数据接口获取返回报文中的personId字段|
|`groupId`|`body`|`string`|是|访客受限名单分组Id集合，可从查询访客受限名单分组接口获取返回报文中的groupId字段|
|`personList`|`body`|`object[]`|否|待添加受限名单人员信息集合|
|`+identityId`|`Body`|`number`|否|证件类型:111身份证，335驾驶证，131工作证，133学生证，414护照，990其他|
|`+identityNum`|`Body`|`string`|否|证件号码(身份证号码需符合18位字符，其他证件号码需符合1～20位数字、字母，手机号和证件号码至少有一个存在)|
|`+personName`|`Body`|`string`|是|访客受限名单人员姓名|
|`+phoneNum`|`Body`|`string`|否|访客受限名单人员手机号码（1-20个数字，手机号和证件号码至少有一个存在）|
|`+remark`|`Body`|`string`|否|描述(0-128个字符)|

## 请求参数示例
```json
{
  "delPersonList": [
    "sdfsd-sadfdsf"
  ],
  "groupId": "asdfsd-sadfdsf-sadf",
  "personList": [
    {
      "identityId": 111,
      "identityNum": "320282199307100390",
      "personName": "小红",
      "phoneNum": "18100122222",
      "remark": "测试人员"
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，全部成功时返回：0 – 成功，有一条失败时返回：其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回结果，为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```
