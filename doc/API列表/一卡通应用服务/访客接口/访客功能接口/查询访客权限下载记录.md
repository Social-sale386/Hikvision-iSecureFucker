# 查询访客权限下载记录

> 接口说明
> 该接口用于查询访客的权限下载记录。当访客登记完成后或者完成签离后，可以通过该接口查询访客权限的下载（或者取消）结果，如果存在权限下载失败的记录，可以通过“访客权限重新下发”接口实现对该访客的权限进行重新下发。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/visitor/v1/auth/result`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/auth/result
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`orderId`|`body`|`string`|否|访客记录id，登记时获取，或从查询访客来访记录v2接口获取返回报文中的orderId字段|
|`pageNo`|`body`|`integer`|是|当前页码，要求大于0|
|`pageSize`|`body`|`integer`|是|每页记录总数，要求大于0且不超过1000|

## 请求参数示例
```json
{
  "orderId": "123",
  "pageNo": 1,
  "pageSize": 10
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+list`|`body`|`object[]`|否|下载记录列表|
|`++authNo`|`body`|`string`|否|授权号码|
|`++authStatus`|`body`|`number`|否|下载结果,0:待下发,1:正在下发,2:下发成功,3:下发失败,4:正在签离,5:签离失败,6:已失效|
|`++authType`|`body`|`number`|否|授权类型,0：身份证；1：身份证序列号；2:IC卡号；3:二维码；4:车牌号；5:人脸|
|`++deviceDesc`|`body`|`string`|否|设备名称|
|`++deviceId`|`body`|`string`|否|设备id|
|`++deviceType`|`body`|`string`|否|设备类型，1: 门禁;2: 梯控;3: 可视对讲;4: 停车场;5人脸监控|
|`++doorDesc`|`body`|`string`|否|设备资源点名称|
|`++doorId`|`body`|`string`|否|设备资源点id|
|`++orderId`|`body`|`string`|否|访客id|
|`++visitorName`|`body`|`string`|否|访客姓名|
