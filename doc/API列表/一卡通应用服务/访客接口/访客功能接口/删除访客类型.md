# 删除访客类型

> 接口说明
> 删除访客类型

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.2.0及以上版本|
|接口地址|`/api/visitor/v1/visitorTypeInfo/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/visitorTypeInfo/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`visitorTypeIds`|`body`|`array`|是|访客类型id|

## 请求参数示例
```json
{
  "visitorTypeIds": []
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|无|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
