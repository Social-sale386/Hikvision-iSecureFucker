# 对待审核的记录进行审核

> 接口说明
> 对待审核的记录进行审核

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/v1/aprovalRecord/approval`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/v1/aprovalRecord/approval
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`approvalInfoList`|`body`|`object[]`|否|审核结果的集合，最大长度不超过10|
|`+orderIdList`|`Body`|`string[]`|是|访客记录id集合，最大长度不超过1000|
|`+privilegeGroupIdList`|`Body`|`string[]`|否|权限组Id集合，最多不超过20个|
|`+approvalResult`|`Body`|`string`|是|审核结果: 1(通过) 0(退回)|
|`+remark`|`Body`|`string`|否|审核退回填写的原因,最大128个字符|

## 请求参数示例
```json
{
  "approvalInfoList": [
    {
      "orderIdList": [
        "aaaa"
      ],
      "privilegeGroupIdList": [
        "aaaaaa"
      ],
      "approvalResult": "1",
      "remark": "cs"
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`object[]`|否|返回数据(具体接口返回信息结构)|
|`+personNum`|`body`|`string`|否|来访人数|
|`+visitEndTime`|`body`|`string`|否|预计离开时间ISO8601格式|
|`+visitStartTime`|`body`|`string`|否|预计来访时间ISO8601格式|
|`+visitorInfoList`|`body`|`object[]`|否|访客详细数据集合|
|`++verificationCode`|`body`|`string`|否|访客验证码,6位数字字符串|
|`++visitorOrderId`|`body`|`string`|否|访客记录id|
|`++certificateType`|`body`|`string`|否|证件类型：111、身份证|
|`++visitorName`|`body`|`string`|否|访客姓名|
|`++certificateNo`|`body`|`string`|否|证件号码|
|`++phoneNo`|`body`|`string`|否|访客联系电话|
|`++plateNo`|`body`|`string`|否|访客车牌号|
|`++picUri`|`body`|`string`|否|访客头像,图片的相对URL|
|`++svrIndexCode`|`body`|`string`|否|图片存储服务的唯一标识|
|`++identityUri`|`body`|`string`|否|证件照,图片的相对URL|
|`++identitySvrCode`|`body`|`string`|否|证件照对应图片服务器serviceNodes|
|`++scanningPhoto`|`body`|`string`|否|证件扫描照,图片的相对URL|
|`++scanningSvrCode`|`body`|`string`|否|证件扫描照对应图片服务器serviceNodes|
|`++visitorStatus`|`body`|`string`|否|访客状态,0:待审核、1：正常、2：迟到、3：失效、4：审核退回、9：审核失效、10：邀约中、11:邀约失效|
|`+orderId`|`body`|`string`|否|唯一标识|
|`+receptionistId`|`body`|`string`|否|被访人ID|
|`+receptionistNo`|`body`|`string`|否|被访人工号|
|`+receptionistName`|`body`|`string`|否|被访人名称|
|`+receptionistFactory`|`body`|`string`|否|被访人厂区|
|`+receptionistCode`|`body`|`string`|否|被访人所属组织编码|
|`+visitPurpose`|`body`|`string`|否|来访事由|
|`+visitorWorkUnit`|`body`|`string`|否|来访单位|
