# 添加访客权限组

> 接口说明
> 该接口用于添加权限组信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1及以上版本|
|接口地址|`/api/v1/privilege/groups/add`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/v1/privilege/groups/add
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`privilegeGroupName`|`body`|`string`|是|权限组名称(1-16位，且非竖线等以下特殊字符’ / \ : * ? " < >)|
|`remark`|`body`|`string`|否|描述(1-16位字符)|
|`parkId`|`body`|`string`|是|园区编号|

## 请求参数示例
```json
{
  "privilegeGroupName": "CS",
  "remark": "111",
  "parkId": "AADAD"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`object`|否|返回数据(具体接口返回信息结构)|
|`+parkId`|`body`|`string`|否|园区id|
|`+privilegeGroupId`|`body`|`string`|否|权限组ID|
|`+privilegeGroupName`|`body`|`string`|否|权限组名称|
|`+remark`|`body`|`string`|否|描述|
|`+isDefault`|`body`|`number`|否|是否是默认权限组，0-是，1-否|
|`+cemsAuthEnable`|`body`|`number`|否|是否开启消费权限。0：开启；1：关闭|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "parkId": "6546sdafs56adf",
    "privilegeGroupId": "731bb582-fc49-11e7-83de-2ffbc13dcaba",
    "privilegeGroupName": "访客基本权限组",
    "remark": "访客仅可进入大厅区域",
    "isDefault": 1,
    "cemsAuthEnable": 1
  }
}
```
