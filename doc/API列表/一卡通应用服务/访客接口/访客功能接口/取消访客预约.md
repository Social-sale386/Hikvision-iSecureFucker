# 取消访客预约

> 接口说明
> 该接口用于对访客预约记录进行取消操作。可用于取消访客预约的记录状态有：正常、迟到。失效的预约记录不能取消。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/visitor/v1/appointment/cancel`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/appointment/cancel
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`Array[0]`|`body`|`string`|是|预约记录ID|

## 请求参数示例
```json
[
  "321654987"
]
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|空值|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
