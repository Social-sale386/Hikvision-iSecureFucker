# 查询访客受限名单分组中人员数据

> 接口说明
> 查询已添加的受限名单分组中的人员数据，返回受限名单人员相关信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/visitor/v1/blocklist/person/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/blocklist/person/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`groupIdList`|`body`|`string[]`|否|访客受限名单分组Id集合，可从查询访客受限名单分组接口获取返回报文中的groupId字段|
|`groupName`|`body`|`string`|否|访客受限名单分组名称,支持模糊查询|
|`identityNumList`|`body`|`string[]`|否|受限名单人员证件号码集合|
|`phoneNumList`|`body`|`string[]`|否|受限名单人员手机号码集合|
|`personName`|`body`|`string`|否|受限名单人员名称,支持模糊查询|
|`pageNo`|`body`|`integer`|是|页码,pageNo>0|
|`pageSize`|`body`|`integer`|是|页大小,0<pageSize<=1000|
|`parkId`|`body`|`string`|否|园区id|

## 请求参数示例
```json
{
  "groupIdList": [
    "sdfsd-sadfdsf"
  ],
  "groupName": "小红",
  "identityNumList": [
    "sdfsd-sadfdsf"
  ],
  "phoneNumList": [
    "18100100001"
  ],
  "personName": "1253566554",
  "pageNo": 3,
  "pageSize": 50,
  "parkId": "abcde12345xyz"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+firstPage`|`body`|`boolean`|否|是否为第一页|
|`+hasNextPage`|`body`|`boolean`|否|是否有下一页|
|`+hasPreviousPage`|`body`|`boolean`|否|是否有上一页|
|`+lastPage`|`body`|`boolean`|否|是否为最后一页|
|`+page`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|每页结果数|
|`+total`|`body`|`number`|否|总结果数|
|`+totalPage`|`body`|`number`|否|总页数|
|`+rows`|`body`|`object[]`|否|人员数据列表|
|`++groupId`|`body`|`string`|否|访客受限名单分组Id|
|`++groupName`|`body`|`string`|否|访客受限名单分组名称|
|`++identityId`|`body`|`number`|否|证件类型:111身份证，335驾驶证，131工作证，133学生证，414护照，990其他|
|`++identityNum`|`body`|`string`|否|证件号码|
|`++personName`|`body`|`string`|否|访客受限名单人员姓名|
|`++personId`|`body`|`string`|否|访客受限名单人员ID|
|`++phoneNum`|`body`|`string`|否|手机号码|
|`++remark`|`body`|`string`|否|描述|
