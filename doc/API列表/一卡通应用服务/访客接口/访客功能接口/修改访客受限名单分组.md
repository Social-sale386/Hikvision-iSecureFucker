# 修改访客受限名单分组

> 接口说明
> 修改已添加的受限名单分组数据，支持修改受限名单分组名称，注意修改后的分组名称不能与已存在的分组名称重复。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/visitor/v1/blocklist/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/blocklist/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`groupId`|`body`|`string`|是|待修改受限名单人员id，可从查询访客受限名单分组接口获取返回报文中的groupId字段|
|`groupName`|`body`|`string`|是|访客受限名单分组名称(1-32位，且非以下特殊字符’ / \ : * ? " < >|

## 请求参数示例
```json
{
  "groupId": "asdfsdfwsf23f3df",
  "groupName": "VIP组"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+groupId`|`body`|`string`|否|访客受限名单分组ID|
|`+groupName`|`body`|`string`|否|访客受限名单分组名称|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "groupId": "asdfsd-sadfdsf-sadf",
    "groupName": "VIP组"
  }
}
```
