# 更新访客权限组信息

> 接口说明
> 更新访客权限组信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center v1及以上版本|
|接口地址|`/api/v1/privilege/groups/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/v1/privilege/groups/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`privilegeGroupId`|`body`|`string`|是|访客权限组ID|
|`privilegeGroupName`|`body`|`string`|否|权限组名称(1-16位，且非竖线等以下特殊字符’ / \ : * ? " < >)|
|`remark`|`body`|`string`|否|描述(1-16位)|

## 请求参数示例
```json
{
  "privilegeGroupId": "abcd",
  "privilegeGroupName": "访客基本权限组",
  "remark": "访客仅可进入大厅区域"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码(错误码16进制，成功返回"0")|
|`msg`|`body`|`string`|否|错误提示信息(记录接口执行情况说明信息)|
|`data`|`body`|`string`|否|返回数据(具体接口返回信息结构)|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "aaaa"
}
```
