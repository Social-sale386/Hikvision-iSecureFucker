# 查询访客出入事件

> 接口说明
> 本接口支持通过访客id,姓名,事件时间,来访时间，门禁点等资源信息查询访客出入事件,并且返回数据中补全访客信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/visitor/v1/event/turnover/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/visitor/v1/event/turnover/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`visitorId`|`body`|`string`|否|访客人员id，查询访客预约记录V2接口获取返回报文中的visitorId字段|
|`visitorName`|`body`|`string`|否|访客人员名称，长度不超过32，支持中英文字符，不能包含 ’ / \ : * ? " < >|
|`resourceCode`|`body`|`string`|否|设备资源code,可以是门禁设备，梯控设备，可视对讲设备，停车场等资源信息）|
|`eventTimeBegin`|`body`|`string`|否|事件开始时间ISO格式|
|`eventTimeEnd`|`body`|`string`|否|事件结束时间ISO格式|
|`visitStartTimeBegin`|`body`|`string`|否|访客预计来访开始时间ISO格式|
|`visitStartTimeEnd`|`body`|`string`|否|访客预计来访结束时间ISO格式|
|`visitEndTimeBegin`|`body`|`string`|否|访客预计离开开始时间ISO格式|
|`visitEndTimeEnd`|`body`|`string`|否|访客预计离开结束时间ISO格式|
|`pageNo`|`body`|`integer`|是|页码，大于0|
|`pageSize`|`body`|`integer`|是|页面大小,大于0，不能超过1000|
|`parkId`|`body`|`string`|否|园区id|

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.2.2 访客管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+firstPage`|`body`|`boolean`|否|是否为第一页|
|`+hasNextPage`|`body`|`boolean`|否|是否存在下一页|
|`+hasPreviousPage`|`body`|`boolean`|否|是否存在前一页|
|`+lastPage`|`body`|`boolean`|否|是否为最后一页|
|`+page`|`body`|`number`|否|当前页|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+rows`|`body`|`object[]`|否|当前分页查询数据结果集|
|`++orderId`|`body`|`string`|否|访客记录id|
|`++visitorName`|`body`|`string`|否|访客姓名|
|`++visitorSex`|`body`|`number`|否|性别,1男,2女|
|`++visitorGroupId`|`body`|`string`|否|访客分组id|
|`++beVisitPersonId`|`body`|`string`|否|被访人id|
|`++beVisitPersonName`|`body`|`string`|否|被访人姓名|
|`++cardNum`|`body`|`string`|否|访客卡号|
|`++cardType`|`body`|`number`|否|卡类型|
|`++deviceIndexCode`|`body`|`string`|否|设备indexCode|
|`++deviceType`|`body`|`number`|否|设备类型|
|`++deviceDesc`|`body`|`string`|否|设备描述|
|`++eventName`|`body`|`string`|否|事件名称(事件类型描述)|
|`++eventId`|`body`|`string`|否|事件消息唯一标识|
|`++phoneNum`|`body`|`string`|否|手机号|
|`++carNumber`|`body`|`string`|否|车牌号|
|`++identityId`|`body`|`number`|否|证件类型：111身份证，335驾驶证，131工作证，133学生证，414护照，990其他|
|`++identityNum`|`body`|`string`|否|证件号码|
|`++visitorPhotoUri`|`body`|`string`|否|访客抓拍头像照片,图片的相对URL|
|`++identityPhotoUri`|`body`|`string`|否|访客证件照片,图片的相对URL|
|`++visitorSvrCode`|`body`|`string`|否|访客头像对应图片服务器serviceNodes|
|`++identitySvrCode`|`body`|`string`|否|证件照对应图片服务器serviceNodes|
|`++eventTime`|`body`|`string`|否|事件时间ISO格式|
|`++visitStartTime`|`body`|`string`|否|访客预计来访时间ISO格式|
|`++visitEndTime`|`body`|`string`|否|访客预计离开时间ISO格式|
|`++visitRegisterTime`|`body`|`string`|否|访客登记时间ISO格式|
|`++visitRealEndTime`|`body`|`string`|否|访客签离时间ISO格式|
|`+total`|`body`|`number`|否|总结果数|
|`+totalPage`|`body`|`number`|否|总页数|
