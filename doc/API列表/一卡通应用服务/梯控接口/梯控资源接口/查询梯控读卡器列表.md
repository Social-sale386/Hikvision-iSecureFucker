# 查询梯控读卡器列表

> 接口说明
> 查询目录下有权限的梯控读卡器列表。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center v1.3及以上版本|
|接口地址|`/api/resource/v1/ladderCardReader/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/ladderCardReader/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`name`|`body`|`string`|否|名称，模糊搜索 若包含中文，最大长度指不超过按照指定编码的字节长度，即getBytes(“utf-8”).length 最大长度：32|
|`regionIndexCodes`|`body`|`string[]`|否|区域编号,可以为空;|
|`isSubRegion`|`body`|`boolean`|否|true时，搜索regionIndexCodes及其子孙区域的资源；|
|`pageNo`|`body`|`integer`|是|当前页码，0 < pageNo|
|`pageSize`|`body`|`integer`|是|分页大小，0<pageSize≤1000|
|`orderBy`|`body`|`string`|否|排序字段,注意：排序字段必须是查询条件，否则返回参数错误|
|`orderType`|`body`|`string`|否|降序升序,降序：desc，升序：asc|
|`expressions`|`body`|`object[]`|否|查询表达式|
|`+key`|`Body`|`string`|否|资源属性名，key传updateTime，operator传between可以查询特定时间段更新的数据，考虑到校时和夏令时，建议值查询过去一天的数据变更|
|`+operator`|`Body`|`number`|否|操作运算符， 0 ：=， 1 ：>=， 2 ：<=， 3 ：in， 4 ：not in， 5 ：between， 6 ：like， 7 ：pre like， 8 ：suffix like|
|`+values`|`Body`|`string[]`|否|资源属性值，=、>=、<=、like、values数组长度只能是1； in、not in，values数组长度大于1，最大不超时20； in_array用于查询key值有多个value的情况，例如行车监控添加的设备类型为encodeDevice、encodeDeviceMss两个类型，使用encodeDevice或者encodeDeviceMss都可以查询到； between只能用于整形、日期 ； like只能用于字符串。|

补充说明:
- String[]，支持根据区域批量查询； 区域编号个数<=1000个； 可从查询区域列表v2接口获取返回参数indexCode
- false时，只搜索 regionIndexCodes的资源

## 请求参数示例
```json
{
  "name": "Test",
  "regionIndexCodes": [
    "fb4f65c-18c7-4073-9314-5402234b38"
  ],
  "isSubRegion": true,
  "pageNo": 1,
  "pageSize": 1,
  "orderBy": "name",
  "orderType": "desc",
  "expressions": [
    {
      "key": "indexCode",
      "operator": 0,
      "values": [
        "ayd8y80y1y09078979882ye01y2e8"
      ]
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|分页大小|
|`+list`|`body`|`object[]`|否|返回数据|
|`++resourceType`|`body`|`string`|否|资源类型，详见附录A.2 资源类型|
|`++indexCode`|`body`|`string`|否|资源唯一标志|
|`++name`|`body`|`string`|否|资源名称|
|`++regionIndexCode`|`body`|`string`|否|所属区域编号|
|`++regionPath`|`body`|`string`|否|所属区域目录，由唯一标示组成，最大10级，格式：|
|`++ip`|`body`|`string`|否|设备IP|
|`++port`|`body`|`string`|否|设备端口|
|`++deviceCode`|`body`|`string`|否|主动设备编号|
|`++deviceKey`|`body`|`string`|否|设备驱动|
|`++deviceModel`|`body`|`string`|否|设备型号|
|`++deviceType`|`body`|`string`|否|设备系列|
|`++netZoneId`|`body`|`string`|否|所属网域|
|`++deployId`|`body`|`string`|否|拨码|
|`++communicationMode`|`body`|`string`|否|通信方式，0：韦根;|
|`++parentIndexCode`|`body`|`string`|否|父级资源编号|
|`++sort`|`body`|`number`|否|显示顺序|
|`++createTime`|`body`|`string`|否|创建时间，ISO8601标准|
|`++updateTime`|`body`|`string`|否|更新时间，ISO8601标准|
|`++description`|`body`|`string`|否|描述|
|`++regionName`|`body`|`string`|否|区域名称|
|`++regionPathName`|`body`|`string`|否|所属区域目录名，以"/"分隔|

补充说明:
- 根节点\子区域1\子区域2
- 1：RS232;
- 2：RS485
