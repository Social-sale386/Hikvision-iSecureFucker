# 获取门禁事件的图片

> 接口说明
> 获取门禁事件的图片，重定向后返回图片流。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/acs/v1/event/pictures`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/acs/v1/event/pictures
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`svrIndexCode`|`body`|`string`|是|提供picUri处会提供此字段|
|`picUri`|`body`|`string`|是|图片相对地址|

## 请求参数示例
```json
{
  "svrIndexCode": "1f276203e5234bdca08f7d99e1097bba",
  "picUri": "/pic?9dd351ia3-e*8634503d1165m3ep=t=i1p*i=d1s*i=d3b*i8da0*c74818542-9445fb--0874a79z189s=7ie4="
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其它参考附录E.2.1 门禁管理错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|返回数据|
