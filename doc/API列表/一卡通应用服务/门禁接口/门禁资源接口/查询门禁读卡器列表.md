# 查询门禁读卡器列表

> 接口说明
> 查询目录下有权限的门禁读卡器列表。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center v1.3及以上版本|
|接口地址|`/api/resource/v1/reader/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/reader/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`name`|`body`|`string`|否|名称，模糊搜索 若包含中文，最大长度指不超过按照指定编码的字节长度，即getBytes(“utf-8”).length 最大长度：32|
|`regionIndexCodes`|`body`|`string[]`|否|区域编号,可以为空;|
|`isSubRegion`|`body`|`boolean`|否|是否包含下级区域，注： 1、参数isSubRegion和regionIndexCodes必须同时存在，且regionIndexCodes只能指定一个区域编码|
|`pageNo`|`body`|`integer`|是|当前页码，0 < pageNo|
|`pageSize`|`body`|`integer`|是|分页大小，0<pageSize≤1000|
|`capabilitySet`|`body`|`string[]`|否|设备能力集(含设备上的智能能力)，详见附录A .44，|
|`intelligentSet`|`body`|`string[]`|否|智能后分析能力集，详见附录A.45，|
|`logicalRelationEnum`|`body`|`string`|否|设备能力集和后分析智能分析能力关系，默认为AND，AND：capabilitySet和intelligentSet都匹配|
|`orderBy`|`body`|`string`|否|排序字段,注意：排序字段必须是查询条件，否则返回参数错误|
|`orderType`|`body`|`string`|否|降序升序,降序：desc，升序：asc|

补充说明:
- String[]，支持根据区域批量查询； 区域编号个数<=1000个； 单个长度<=64Byte
- 多个以@分隔
- 能力集个数20个，单个长度40
- 多个以@分隔
- 能力集个数20个，单个长度40
- OR：capabilitySet或intelligentSet匹配

## 请求参数示例
```json
{
  "name": "Test",
  "regionIndexCodes": [
    "fb4f65c-18c7-4073-9314-5402234b38"
  ],
  "isSubRegion": true,
  "pageNo": 1,
  "pageSize": 1,
  "capabilitySet": [
    "support_card"
  ],
  "intelligentSet": [
    "body"
  ],
  "logicalRelationEnum": "AND",
  "orderBy": "name",
  "orderType": "desc"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|分页大小|
|`+list`|`body`|`object[]`|否|返回数据|
|`++resourceType`|`body`|`string`|否|资源类型，reader：门禁读卡器|
|`++indexCode`|`body`|`string`|否|资源唯一标志|
|`++name`|`body`|`string`|否|资源名称|
|`++regionIndexCode`|`body`|`string`|否|所属区域编号|
|`++regionPath`|`body`|`string`|否|所属区域目录，由唯一标示组成，最大10级，格式：|
|`++ip`|`body`|`string`|否|设备IP|
|`++port`|`body`|`string`|否|设备端口|
|`++deviceCode`|`body`|`string`|否|主动设备编号|
|`++deviceKey`|`body`|`string`|否|设备驱动|
|`++deviceModel`|`body`|`string`|否|设备型号|
|`++deviceType`|`body`|`string`|否|设备系列|
|`++dataVersion`|`body`|`string`|否|版本号|
|`++netZoneId`|`body`|`string`|否|所属网域|
|`++deployId`|`body`|`string`|否|拨码|
|`++communicationMode`|`body`|`string`|否|通信方式，0：韦根;|
|`++parentIndexCode`|`body`|`string`|否|父级资源编号|
|`++sort`|`body`|`number`|否|显示顺序|
|`++capability`|`body`|`string`|否|设备能力，详见数据字典A.44|
|`++createTime`|`body`|`string`|否|创建时间，ISO8601标准|
|`++updateTime`|`body`|`string`|否|更新时间，ISO8601标准|
|`++description`|`body`|`string`|否|描述|
|`++comId`|`body`|`string`|否|标签|
|`++doorNo`|`body`|`string`|否|门禁点序号|
|`++acsReaderCardCapacity`|`body`|`string`|否|卡容量|
|`++acsReaderFingerCapacity`|`body`|`string`|否|指纹容量|
|`++acsReaderFaceCapacity`|`body`|`string`|否|人脸容量|
|`++regionPathName`|`body`|`string`|否|所属区域目录名，符号"@"进行分隔|
|`++regionName`|`body`|`string`|否|区域名称|

补充说明:
- 根节点\子区域1\子区域2
- 1：RS232;
- 2：RS485
