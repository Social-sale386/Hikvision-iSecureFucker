# 查询卡口抓拍事件详情

> 接口说明
> 根据条件分页查询抓拍事件详情。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/mpc/v1/events/vehicleRecord`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/mpc/v1/events/vehicleRecord
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|目标页码,范围 ( 0 , ~ )|
|`pageSize`|`body`|`integer`|是|每页记录数,范围 ( 0 , 1000 ]|
|`beginTime`|`body`|`string`|否|过车开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）注：【过车开始、结束时间】和【入库开始时间、结束时间】只能配套使用，不能同时传值|
|`endTime`|`body`|`string`|否|过车结束时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）注：【过车开始、结束时间】和【入库开始时间、结束时间】只能配套使用，不能同时传值|
|`createBeginTime`|`body`|`string`|否|入库开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）注：【过车开始、结束时间】和【入库开始时间、结束时间】只能配套使用，不能同时传值|
|`createEndTime`|`body`|`string`|否|入库结束时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）注：【过车开始、结束时间】和【入库开始时间、结束时间】只能配套使用，不能同时传值|
|`plateNo`|`body`|`string`|否|车牌号，支持模糊查询|
|`vehicleType`|`body`|`string`|否|车辆类型,详见附录A.75 卡口车辆类型|
|`plateType`|`body`|`string`|否|车牌类型,详见附录A.76 卡口车牌类型|
|`monitoringId`|`body`|`string`|否|卡口点id，不传代表查询全部， 支持多个indexcode , 逗号分割，可通过查询园区卡口点列表接口获取返回参数indexCode|
|`minSpeed`|`body`|`integer`|否|最低速度(单位km/h)，大于等于0|
|`maxSpeed`|`body`|`integer`|否|最高速度(单位km/h)，不超过int的最大值|
|`eventId`|`body`|`string`|否|事件的唯一编号|

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.2 园区卡口错误码|
|`msg`|`body`|`string`|否|返回提示信息|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总记录数|
|`+pageSize`|`body`|`number`|否|分页大小|
|`+pageNo`|`body`|`number`|否|页码|
|`+list`|`body`|`object[]`|否|记录数据集合|
|`++crossTime`|`body`|`string`|否|过车时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`++eventId`|`body`|`string`|否|事件的唯一标识|
|`++plateNo`|`body`|`string`|否|车牌号|
|`++vehicleType`|`body`|`string`|否|车辆类型,详见附录A.75 卡口车辆类型|
|`++plateType`|`body`|`string`|否|车牌类型,详见附录A.76 卡口车牌类型|
|`++monitoringId`|`body`|`string`|否|事件源(卡口点的编号)|
|`++monitoringName`|`body`|`string`|否|事件源(卡口点的名称)|
|`++platePicUri`|`body`|`string`|否|车牌图片uri|
|`++carPicUri`|`body`|`string`|否|车辆图片uri|
|`++aswSyscode`|`body`|`string`|否|图片服务的唯一标识|
|`++uuid`|`body`|`string`|否|事件唯一标识|
|`++speed`|`body`|`number`|否|车速(单位km/h)，大于0|
