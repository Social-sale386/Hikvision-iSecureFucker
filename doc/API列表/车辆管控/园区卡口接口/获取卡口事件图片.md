# 获取卡口事件图片

> 接口说明
> 园区卡口过车记录、违章记录带有车辆或车牌图片信息，此接口通过重定向的方式提供这些图片信息展示的数据。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/mpc/v1/events/image`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/mpc/v1/events/image
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`aswSyscode`|`body`|`string`|是|图片服务唯一标识码，，提供picUri处会提供此字段|
|`picUri`|`body`|`string`|是|图片的相对地址|

## 请求参数示例
```json
{
  "aswSyscode": "erghbwerh5t234t45y22t34",
  "picUri": "/pic?=d7ei703i10cd*73a-d5108a--22cd0c9d6592aiid="
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.2 园区卡口错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|返回数据|
