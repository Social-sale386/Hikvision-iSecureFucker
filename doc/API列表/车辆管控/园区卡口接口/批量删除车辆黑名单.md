# 批量删除车辆黑名单

> 接口说明
> 根据车辆黑名单唯一编号批量删除车辆黑名单，多个车辆黑名单编号以英文逗号分隔。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/mpc/v1/alarm/black/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/mpc/v1/alarm/black/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCodes`|`body`|`string`|否|黑名单编号,调用查询车辆黑名单信息接口获取返回参数indexCode，多个编号以英文逗号分隔，和车牌号二选一|
|`plateNos`|`body`|`string`|否|车牌号，长度不能超过32位，多个车牌号以英文逗号分隔，和黑名单编号二选一|

## 请求参数示例
```json
{
  "indexCodes": "9aef05a6725f41d1977c8becc59d3d63,9aef05a6725f41d1977c8becc59d3d89"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.2 园区卡口错误码|
|`msg`|`body`|`string`|否|返回提示信息|
|`data`|`body`|`string`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```
