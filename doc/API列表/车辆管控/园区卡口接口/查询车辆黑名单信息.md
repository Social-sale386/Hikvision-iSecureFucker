# 查询车辆黑名单信息

> 接口说明
> 根据指定条件分页查询车辆黑名单信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/mpc/v1/alarm/black/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/mpc/v1/alarm/black/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|目标页码,范围 ( 0 , ~ )|
|`pageSize`|`body`|`integer`|是|每页记录数,范围 ( 0 , 1000 ]|
|`indexCode`|`body`|`string`|否|黑名单唯一编号|
|`plateNo`|`body`|`string`|否|车牌号，支持模糊查询|
|`beginTime`|`body`|`string`|否|布防开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`endTime`|`body`|`string`|否|布防结束时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`ownerName`|`body`|`string`|否|车主姓名|
|`phone`|`body`|`string`|否|手机号码|
|`reason`|`body`|`integer`|否|布防原因，|
|`description`|`body`|`string`|否|描述|

补充说明:
- 1（被盗车），2（被抢车），3（嫌疑车），4（交通违法车），5（紧急查控车）

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 15,
  "indexCode": "d100bacf5cd9430a8ca99710b137c57a",
  "plateNo": "浙59SVN",
  "beginTime": "2019-07-26T15:00:00.000+08:00",
  "endTime": "2019-07-28T15:00:00.000+08:00",
  "ownerName": "刘先生",
  "phone": "17788754333",
  "reason": 1,
  "description": "描述信息"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.2 园区卡口错误码|
|`msg`|`body`|`string`|否|返回提示信息|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总记录数|
|`+pageSize`|`body`|`number`|否|分页大小|
|`+pageNo`|`body`|`number`|否|页码|
|`+list`|`body`|`object[]`|否|记录数据集合|
|`++indexCode`|`body`|`string`|否|黑名单唯一编号|
|`++plateNo`|`body`|`string`|否|车牌号|
|`++beginTime`|`body`|`string`|否|布防开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`++endTime`|`body`|`string`|否|布防结束时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`++ownerName`|`body`|`string`|否|车主姓名，在车辆黑名单关联了人员的时候会返回值|
|`++phone`|`body`|`string`|否|手机号码，在车辆黑名单关联了手机号码的时候会返回值|
|`++reason`|`body`|`number`|否|布防原因，默认-1（全部原因），1（被盗车），2（被抢车），3（嫌疑车），4（交通违法车），5（紧急查控车）|
|`++description`|`body`|`string`|否|描述，当黑名单有描述的时候会返回值|
