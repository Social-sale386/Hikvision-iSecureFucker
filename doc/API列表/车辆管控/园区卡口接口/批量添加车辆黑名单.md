# 批量添加车辆黑名单

> 接口说明
> 批量添加车辆黑名单，该车辆布防为黑名单后，当有过车事件上报时，会产生对应的黑名单事件，一次添加最大不超过400个车辆黑名单。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/mpc/v1/alarm/black/add`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/mpc/v1/alarm/black/add
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`Array[0]`|`body`|`object`|否||
|`+plateNo`|`Body`|`string`|是|车牌号|
|`+beginTime`|`Body`|`string`|否|布防开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`+endTime`|`Body`|`string`|是|布防结束时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`+ownerName`|`Body`|`string`|否|车主姓名|
|`+phone`|`Body`|`string`|否|手机号码|
|`+reason`|`Body`|`number`|是|布防原因,|
|`+description`|`Body`|`string`|否|描述|

补充说明:
- 1（被盗车），2（被抢车），3（嫌疑车），4（交通违法车），5（紧急查控车）

## 请求参数示例
```json
[
  {
    "plateNo": "浙59SVN",
    "beginTime": "2019-07-26T15:00:00.000+08:00",
    "endTime": "2019-07-28T15:00:00.000+08:00",
    "ownerName": "刘先生",
    "phone": "17788754333",
    "reason": 1,
    "description": "描述信息"
  }
]
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.2 园区卡口错误码|
|`msg`|`body`|`string`|否|显示|
|`data`|`body`|`object[]`|否|返回数据|
|`+indexCode`|`body`|`string`|否|黑名单唯一编号|
|`+plateNo`|`body`|`string`|否|车牌号|

补充说明:
- 返回提示信息

## 返回参数示例

```json
  "data": [
    {

    }
  ]

```
