# 查询车辆白名单信息

> 接口说明
> 根据指定条件分页查询车辆白名单信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/mpc/v1/alarm/white/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/mpc/v1/alarm/white/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCode`|`body`|`string`|否|白名单唯一编号|
|`pageNo`|`body`|`integer`|是|目标页码,范围 ( 0 , ~ )|
|`pageSize`|`body`|`integer`|是|每页记录数,范围 ( 0 , 1000 ]|
|`plateNo`|`body`|`string`|否|车牌号，支持模糊查询|
|`regionIndexCode`|`body`|`string`|否|区域编号|
|`beginTime`|`body`|`string`|否|布防开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`endTime`|`body`|`string`|否|布防结束时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`personId`|`body`|`string`|否|人员Id|
|`phone`|`body`|`string`|否|车主号码|
|`description`|`body`|`string`|否|描述|

## 请求参数示例
```json
{
  "indexCode": "d100bacf5cd9430a8ca99710b137c57a",
  "pageNo": 1,
  "pageSize": 15,
  "plateNo": "浙59SVN",
  "regionIndexCode": "9e9f23e0-b90a-11e9-9477-6fe002d97059",
  "beginTime": "2019-07-26T15:00:00.000+08:00",
  "endTime": "2019-07-28T15:00:00.000+08:00",
  "personId": "a39af90a-6e69-417a-b822-bCCeec9837e1",
  "phone": "17755555555",
  "description": "描述信息"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.2 园区卡口错误码|
|`msg`|`body`|`string`|否|返回提示信息|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总记录数|
|`+pageSize`|`body`|`number`|否|分页大小|
|`+pageNo`|`body`|`number`|否|页码|
|`+list`|`body`|`object[]`|否|记录数据集合|
|`++indexCode`|`body`|`string`|否|白名单唯一编号|
|`++plateNo`|`body`|`string`|否|车牌号|
|`++regionIndexCode`|`body`|`string`|否|区域编号|
|`++beginTime`|`body`|`string`|否|布防开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`++endTime`|`body`|`string`|否|布防结束时间，可能查询出结果为null值，代表永久（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`++personId`|`body`|`string`|否|车主id，当这个车辆在车辆信息管理中关联上人员就会有值返回，uuid的格式|
|`++phone`|`body`|`string`|否|车主号码，当这个车辆关联了手机号码的时候就会返回值|
|`++description`|`body`|`string`|否|描述，当白名单有描述的时候会返回值|
