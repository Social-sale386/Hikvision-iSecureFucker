# 批量添加车辆白名单

> 接口说明
> 批量添加车辆白名单，该车辆布防为白名单后，不会产生违章事件，优先级最高，一次添加最大不超过400个车辆白名单。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/mpc/v1/alarm/white/add`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/mpc/v1/alarm/white/add
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`Array[0]`|`body`|`object`|否||
|`+plateNo`|`Body`|`string`|是|车牌号|
|`+regionIndexCode`|`Body`|`string`|是|区域indexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`+beginTime`|`Body`|`string`|否|布防开始时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`+endTime`|`Body`|`string`|否|布防结束时间（格式：yyyy-MM-ddTHH:mm:ss.sss+当前时区）|
|`+personId`|`Body`|`string`|否|车主id，来源于车辆信息管理，uuid的格式|
|`+phone`|`Body`|`string`|否|车主号码|
|`+description`|`Body`|`string`|否|描述|

## 请求参数示例
```json
[
  {
    "plateNo": "浙59SVN",
    "regionIndexCode": "9e9f23e0-b90a-11e9-9477-6fe002d97059",
    "beginTime": "2019-07-26T15:00:00.000+08:00",
    "endTime": "2019-07-28T15:00:00.000+08:00",
    "personId": "a39af90a-6e69-417a-b822-bCCeec9837e1",
    "phone": "17755555555",
    "description": "描述信息"
  }
]
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.2 园区卡口错误码|
|`msg`|`body`|`string`|否|返回提示信息|
|`data`|`body`|`object[]`|否|返回数据|
|`+indexCode`|`body`|`string`|否|白名单唯一编号|
|`+plateNo`|`body`|`string`|否|车牌号|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": [
    {
      "indexCode": "d100bacf5cd9430a8ca99710b137c57a",
      "plateNo": "浙59SVN"
    }
  ]
}
```
