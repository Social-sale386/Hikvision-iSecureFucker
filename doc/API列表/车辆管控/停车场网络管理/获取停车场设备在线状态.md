# 获取停车场设备在线状态

> 接口说明
> 根据条件获取停车场设备在线状态在线状态接口

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/nms/v1/online/pms/device/get`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/nms/v1/online/pms/device/get
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`regionId`|`body`|`string`|否|区域编码，查询区域列表v2接口获取返回报文中的indexCode字段|
|`includeSubNode`|`body`|`string`|否|是否包含下级（1包含，0不包含）|
|`resourceType`|`body`|`string`|是|设备类型，参考附录A.88 停车场网管资源类型|
|`indexCodes`|`body`|`string[]`|否|设备标识集合，从获取资源列表v2接口获取返回参数indexCode，多个用逗号隔开|
|`status`|`body`|`string`|否|状态，1-在线，0-离线，-1-未检测|
|`pageNo`|`body`|`integer`|否|页码，范围 ( 0 , ~ )，不填默认为1|
|`pageSize`|`body`|`integer`|否|页大小，范围 ( 0 , 1000 ]，不填默认为10|

## 请求参数示例
```json
{
  "regionId": "CC29247a-3b60-4cf9-84d5-4d95fef9311f",
  "includeSubNode": "1",
  "resourceType": "cross",
  "indexCodes": [
    "tfgwc3254234ctg365435634v"
  ],
  "status": "0",
  "pageNo": 1,
  "pageSize": 100
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.4 网管错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|数据信息|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|每页记录数|
|`+totalPage`|`body`|`number`|否|总页数|
|`+total`|`body`|`number`|否|总记录数|
|`+list`|`body`|`object[]`|否|资源信息集|
|`++deviceType`|`body`|`string`|否|设备型号|
|`++deviceIndexCode`|`body`|`string`|否|设备唯一编码|
|`++regionIndexCode`|`body`|`string`|否|区域编码|
|`++collectTime`|`body`|`string`|否|采集时间|
|`++regionName`|`body`|`string`|否|区域名字|
|`++indexCode`|`body`|`string`|否|资源唯一编码|
|`++cn`|`body`|`string`|否|设备名称|
|`++treatyType`|`body`|`string`|否|协议类型， 海康私有协议： hiksdk_net， GB/T28181： gb_reg, eHome协议： ehome_reg， 大华私有协议： dhsdk_net， ONVIF协议： onvif_net， 萤石协议： ezviz_net， 级联： cascade|
|`++manufacturer`|`body`|`string`|否|厂商|
|`++ip`|`body`|`string`|否|ip地址|
|`++port`|`body`|`number`|否|端口|
|`++online`|`body`|`number`|否|在线状态，0离线，1在线|
