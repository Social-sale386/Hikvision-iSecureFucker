# 查询车辆信息

> 接口说明
> 分页查询固定车辆信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/car/getVehicleInfoPage`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/car/getVehicleInfoPage
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`string`|否|页码|
|`pageSize`|`body`|`string`|否|每页行数|
|`plateNo`|`body`|`string`|否|车牌号|
|`cardNo`|`body`|`string`|否|卡号|
|`ownerName`|`body`|`string`|否|车主姓名|
|`plateType`|`body`|`string`|否|车牌类型，逗号隔开 1,2,3,4,5|
|`vehicleType`|`body`|`string`|否|车辆类型，逗号隔开 1,2,3,4,5|
|`regionIndexCode`|`body`|`string`|是|区域编号|
|`carGroup`|`body`|`string`|否|车辆群组，多个用英文逗号隔开|
|`vehicleIdList`|`body`|`string[]`|否|车辆id集合，可根据车辆id查询车辆信息|

补充说明:
- 车牌类型
- 0:标准民用车,1:02式民用车,2:警车,3:警车,4:民用车双行尾牌车,5:使馆车,6:农用车,7:摩托车,8:新能源车
- 0：其他车；
- 1：小型车；
- 2：大型车；
- 3：摩托车

```json
  "pageNo": 1,
  "pageSize": 15,
  "plateNo": "浙A111111",
  "cardNo": "q242342",
  "ownerName": "张三",
  "organizationId": "242342423",
  "plateType": "1",
  "vehicleType": "0",
  "regionIndexCode": "14234234",
  "carGroup": "234234234",
  "vehicleIdList": [
    "234234",
    "sdsdfsdf"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总记录数|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|页面大小|
|`+list`|`body`|`object[]`|否|车辆集合|
|`++vehicleId`|`body`|`string`|否|车辆编号|
|`++plateNo`|`body`|`string`|否|车牌号|
|`++cardNo`|`body`|`string`|否|卡号|
|`++personId`|`body`|`string`|否|车主编号|
|`++plateType`|`body`|`string`|否|车牌类型|
|`++plateColor`|`body`|`string`|否|车牌颜色|
|`++vehicleType`|`body`|`string`|否|车辆类型|
|`++vehicleColor`|`body`|`string`|否|车辆颜色|
|`++regionIndexCode`|`body`|`string`|否|区域编号|
|`++vehicleGroup`|`body`|`string`|否|车辆群组编号|
