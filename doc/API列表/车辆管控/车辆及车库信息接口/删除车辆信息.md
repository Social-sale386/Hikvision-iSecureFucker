# 删除车辆信息

> 接口说明
> 按照车辆编号删除车辆信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/car/deleteVehicle`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/car/deleteVehicle
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`ids`|`body`|`string[]`|是|车辆编号集合|
|`regionIndexCode`|`body`|`string`|是|区域编号|

## 请求参数示例
```json
{
  "ids": [
    "asdasd",
    "234234234"
  ],
  "regionIndexCode": "root00000000"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|请求结果码|
|`msg`|`body`|`string`|是|提示信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success"
}
```
