# 获取停车库节点详细信息

> 接口说明
> 根据节点编号indexCode、类型查询节点详细信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/park/detail/get`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/park/detail/get
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCodes`|`body`|`string[]`|是|资源唯一标示|
|`resourceType`|`body`|`string`|是|资源类型，parking、entrance、roadway、parkFloor、parkSpace|

## 请求参数示例
```json
{
  "indexCodes": [
    "v62365v16b13vr151617b871"
  ],
  "resourceType": "parking"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0:接口业务处理成功，其它参考附录E.other.1|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object[]`|否|返回信息|
|`+resourceType`|`body`|`string`|否|资源类型 停车库:parking 出入口：entrance 车道：roadway 楼层：parkFloor 车位：parkSpace|
|`+indexCode`|`body`|`string`|否|资源唯一标示|
|`+name`|`body`|`string`|否|资源名称|
|`+parentIndexCode`|`body`|`string`|否|父节点唯一标示|
|`+regionIndexCode`|`body`|`string`|否|区域唯一标示|
|`+updateTime`|`body`|`string`|否|创建时间，遵守ISO8601标准，如2018-07-26T21:30:08.322+08:00|
|`+createTime`|`body`|`string`|否|更新时间，遵守ISO8601标准，如2018-07-26T21:30:08.322+08:00|
|`+regionName`|`body`|`string`|否|区域名称|
|`+regionPath`|`body`|`string`|否|所属区域目录，由唯一标示组成，最大10级，格式： @根节点@子区域1@子区域2@|
|`+regionPathName`|`body`|`string`|否|区域目录名称|
