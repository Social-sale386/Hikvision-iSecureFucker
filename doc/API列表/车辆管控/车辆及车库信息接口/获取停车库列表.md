# 获取停车库列表

> 接口说明
> 根据停车场唯一标识集合获取停车列表信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/park/parkList`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/park/parkList
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkIndexCodes`|`body`|`string`|否|停车场唯一标识集合|

补充说明:
- 多个值使用英文逗号分隔，不超过1000个

## 请求参数示例
```json
{
  "parkIndexCodes": "e8629893ef8e4734b7771cc03dad4283,e10ead8c9cc74eb3b05e3c249264836e,21971dd5fe1d4d958ffcdfc182bcaede"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object[]`|否|停车场信息结构体|
|`+parkIndexCode`|`body`|`string`|否|停车场唯一标识（最大长度64）|
|`+parentParkIndexCode`|`body`|`string`|否|父停车场唯一标识（最大长度64）|
|`+parkName`|`body`|`string`|否|停车场名称（最大长度64）|
|`+createTime`|`body`|`string`|否|创建时间|
|`+updateTime`|`body`|`string`|否|更新时间|

补充说明:
- 要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00
- 表示北京时间2017年7月26日21时30分08秒
- 要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00
- 表示北京时间2017年7月26日21时30分08秒
