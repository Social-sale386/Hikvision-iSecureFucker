# 添加或更新车辆

> 接口说明
> 单个添加车辆信息接口

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/car/vehicleSaveOrUpdate`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/car/vehicleSaveOrUpdate
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`vehicleId`|`body`|`string`|否|车辆id，车辆更新时必传|
|`plateNo`|`body`|`string`|是|车牌号码|
|`personId`|`body`|`string`|否|车主系统编号，有卡号时必传|
|`cardNo`|`body`|`string`|否|卡号|
|`plateType`|`body`|`integer`|否|车牌类型|
|`plateColor`|`body`|`integer`|否|车牌颜色，|
|`vehicleType`|`body`|`integer`|否|车辆类型|
|`vehicleColor`|`body`|`integer`|否|车辆颜色|
|`mark`|`body`|`string`|否|描述|
|`regionIndexCode`|`body`|`string`|是|区域编号|
|`vehicleGroup`|`body`|`string`|否|车辆群组编号|

补充说明:
- 0:标准民用车,1:02式民用车,2:警车,3:警车,4:民用车双行尾牌车,5:使馆车,6:农用车,7:摩托车,8:新能源车
- 0:蓝色,1:黄色,2:白色,3:黑色,4:绿色,5:民航黑色, 255:其他颜色
- 0：其他车；
- 1：小型车；
- 2：大型车；
- 3：摩托车
- 0：其他颜色；
- 1：白色；
- 2：银色；
- 3：灰色；
- 4：黑色；
- 5：红色；
- 6：深蓝色；
- 7：蓝色；
- 8：黄色；
- 9：绿色；
- 10：棕色；
- 11：粉色；
- 12：紫色

```json
}
```
