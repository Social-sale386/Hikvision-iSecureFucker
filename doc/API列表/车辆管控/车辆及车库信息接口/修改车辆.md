# 修改车辆

> 接口说明
> 根据车辆编号修改车辆信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/vehicle/single/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/vehicle/single/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`vehicleId`|`body`|`string`|是|车辆ID|
|`plateNo`|`body`|`string`|否|车牌号码|
|`personId`|`body`|`string`|否|人员ID|
|`plateType`|`body`|`string`|否|车牌类型，详见附录A.14 车牌类型|
|`plateColor`|`body`|`string`|否|车牌颜色，详见附录A.15 车牌颜色|
|`vehicleType`|`body`|`string`|否|车辆类型，详见附录A.16 车辆类型|
|`vehicleColor`|`body`|`string`|否|车辆颜色，详见附录A.17 车辆颜色|
|`description`|`body`|`string`|否|车辆描述|

## 请求参数示例
```json
{
  "vehicleId": "1324d8c78y80h350c1y5c18925",
  "plateNo": "浙A111111",
  "personId": "26v25v62562v65727v27",
  "plateType": "1",
  "plateColor": "1",
  "vehicleType": "0",
  "vehicleColor": "0",
  "description": "test"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`message`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`string`|否|执行成功时为空|

## 返回参数示例
```json
{
  "code": "0",
  "message": "success",
  "data": ""
}
```
