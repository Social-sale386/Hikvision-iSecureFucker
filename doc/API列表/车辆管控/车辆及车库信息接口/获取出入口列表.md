# 获取出入口列表

> 接口说明
> 根据停车场唯一标识集合取指定的车入口信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/entrance/entranceList`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/entrance/entranceList
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkIndexCodes`|`body`|`string`|是|停车场唯一标识集|

补充说明:
- 多个值使用英文逗号分隔,不超过1000个

## 请求参数示例
```json
{
  "parkIndexCodes": "7e09fdCC66264eaab4146ef2d6daCCbb,5a20fb59cf4f4191b3127b54a146141f"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object[]`|否|出入口信息结构体|
|`+entranceIndexCode`|`body`|`string`|否|出入口唯一标识（最大长度64）|
|`+entranceName`|`body`|`string`|否|出入口名称（最大长度64）|
|`+parkIndexCode`|`body`|`string`|否|停车场唯一标识（最大长度64）|
|`+roadNum`|`body`|`number`|否|车道数（最大长度32）|
|`+createTime`|`body`|`string`|否|创建时间|
|`+updateTime`|`body`|`string`|否|更新时间|

补充说明:
- 要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00
- 表示北京时间2017年7月26日21时30分08秒
- 要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00
- 表示北京时间2017年7月26日21时30分08秒

```json
  "code": "0",

```
