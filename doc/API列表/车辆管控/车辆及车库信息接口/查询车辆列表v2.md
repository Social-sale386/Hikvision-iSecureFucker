# 查询车辆列表v2

> 接口说明
> 查询车辆列表接口可以根据车牌号码、车主姓名、车辆类型、车牌类型、是否关联人员、车辆状态这些查询条件来进行高级查询；若不指定查询条件，即全量获取所有的车辆信息。返回结果分页展示。
> 注：若指定多个查询条件，表示将这些查询条件进行“与”的组合后进行精确查询
> 当一个车辆属于多个区域时，查询时会返回多条记录。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/resource/v2/vehicle/advance/vehicleList`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v2/vehicle/advance/vehicleList
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`plateNo`|`body`|`string`|否|车牌号码（最大长度64）|
|`pageNo`|`body`|`integer`|是|当前页码|
|`pageSize`|`body`|`integer`|是|每页记录总数|
|`personIds`|`body`|`string`|否|人员ID集，多个值使用英文逗号分隔，单次操作上限为1000条。可从获取人员列表v2接口获取返回值personId|
|`vehicleType`|`body`|`integer`|否|车辆类型，详见附录A.16 车辆类型|
|`plateType`|`body`|`integer`|否|车牌类型，详见附录A.14 车牌类型|
|`isBandPerson`|`body`|`integer`|否|车辆是否和人员绑定，带有人员信息；0:不关联；1:关联|
|`orderBy`|`body`|`string`|否|注意：排序字段必须是查询条件，否则返回参数错误|
|`orderType`|`body`|`string`|否|降序：desc|

补充说明:
- 支持模糊查询
- 当前页码需大于0（pageNo>0）
- 每页记录展示的数目应大于0，小于等于1000（0<pageSize<=1000）
- 升序：asc
- sort、sortBy必须成对出现

## 请求参数示例
```json
{
  "plateNo": "浙",
  "pageNo": 1,
  "pageSize": 1000,
  "personIds": "d7228950bf29490484bc845dff756b95",
  "vehicleType": 1,
  "plateType": 0,
  "isBandPerson": 1,
  "orderBy": "plateType",
  "orderType": "asc"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|车辆信息结构体|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|实际每页记录总数|
|`+list`|`body`|`object[]`|否|车辆列表|
|`++vehicleId`|`body`|`string`|否|车辆ID|
|`++plateNo`|`body`|`string`|否|车牌号码|
|`++personId`|`body`|`string`|否|人员ID|
|`++plateType`|`body`|`number`|否|车牌类型，详见附录A.14 车牌类型|
|`++plateColor`|`body`|`number`|否|车牌颜色，详见附录A.15 车牌颜色|
|`++vehicleType`|`body`|`number`|否|车辆类型，详见附录A.16 车辆类型|
|`++vehicleColor`|`body`|`number`|否|车辆颜色，详见附录A.17 车辆颜色|
|`++description`|`body`|`string`|否|车辆描述|
|`++regionIndexCode`|`body`|`string`|否|所属区域|
|`++regionName`|`body`|`string`|否|所属区域名|
|`++regionPath`|`body`|`string`|否|所属区域目录，@分隔|
|`++regionPathName`|`body`|`string`|否|所属区域目录名|
|`++createTime`|`body`|`string`|否|创建时间|
|`++updateTime`|`body`|`string`|否|修改时间|
|`++isBandPerson`|`body`|`number`|否|车辆是否和人员绑定，带有人员信息；1:关联；0:不关联|
