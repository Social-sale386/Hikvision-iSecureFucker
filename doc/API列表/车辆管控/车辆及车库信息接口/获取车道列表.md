# 获取车道列表

> 接口说明
> 根据出入口唯一标识集合获取车道信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/roadway/roadwayList`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/roadway/roadwayList
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`entranceIndexCodes`|`body`|`string`|是|出入口唯一标识集|

补充说明:
- 多个值使用英文逗号分隔,不超过1000个

## 请求参数示例
```json
{
  "entranceIndexCodes": "02efe65c5e9b43508a268133d8cd3808"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object[]`|否|车道信息结构体|
|`+entranceIndexCode`|`body`|`string`|否|出入口唯一标识（最大长度64）|
|`+roadwayIndexCode`|`body`|`string`|否|车道唯一标识（最大长度64）|
|`+roadwayName`|`body`|`string`|否|车道名称（最大长度64）|
|`+roadwayType`|`body`|`number`|否|车道类型，详见附录A.18 车道类型|
|`+createTime`|`body`|`string`|否|创建时间|
|`+updateTime`|`body`|`string`|否|更新时间|

补充说明:
- 要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00
- 表示北京时间2017年7月26日21时30分08秒
- 要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00
- 表示北京时间2017年7月26日21时30分08秒

    }
