# 批量删除车辆

> 接口说明
> 根据车辆编码删除车辆。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/vehicle/batch/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/vehicle/batch/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`vehicleIds`|`body`|`string[]`|是|待删除的车辆Id列表，单次提交记录上限为1000条|

## 请求参数示例
```json
{
  "vehicleIds": [
    "1728ayr80h08h08v84105"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`message`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object[]`|否|若执行结果为部分删除成功，则返回删除失败的车辆id和错误信息，若全部成功，则该字段为空。|
|`+vehicleId`|`body`|`string`|否|车辆标志|
|`+code`|`body`|`string`|否|错误码|
|`+msg`|`body`|`string`|否|错误信息|

## 返回参数示例
```json
{
  "code": "0",
  "message": "success",
  "data": [
    {
      "vehicleId": "8070yf0awyfe8fyh8w",
      "code": "-1",
      "msg": "error"
    }
  ]
}
```
