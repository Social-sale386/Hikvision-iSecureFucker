# 增量获取车辆数据

> 接口说明
> 根据查询条件查询车辆列表信息，主要根据时间段获取车辆信息，包含已删除数据。其中开始日期与结束日期的时间差必须在1-48小时内。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/resource/v1/vehicle/timeRange`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/vehicle/timeRange
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`startTime`|`body`|`string`|是|开始日期，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00表示北京时间2017年7月26日21时30分08秒|
|`endTime`|`body`|`string`|否|截止日期，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00表示北京时间2017年7月26日21时30分08秒，不传时默认为当前时间|
|`pageNo`|`body`|`integer`|是|页码,pageNo≥1|
|`pageSize`|`body`|`integer`|是|页面大小,0<pageSize≤1000|

## 请求参数示例
```json
{
  "startTime": "2019-08-06T14:01:17.839+08:00",
  "endTime": "2019-08-06T16:01:17.839+08:00",
  "pageNo": 1,
  "pageSize": 1000
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否||
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|车辆信息结构体|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+list`|`body`|`object[]`|否|车辆列表|
|`++vehicleId`|`body`|`string`|否|车辆Id|
|`++plateNo`|`body`|`string`|否|车牌号码|
|`++isBandPerson`|`body`|`number`|否|是否关联人员|
|`++personId`|`body`|`string`|否|人员Id|
|`++plateType`|`body`|`number`|否|车牌类型，详见附录A.14 车牌类型|
|`++plateColor`|`body`|`number`|否|车牌颜色，详见附录A.15 车牌颜色|
|`++vehicleType`|`body`|`number`|否|车辆类型，详见附录A.16 车辆类型|
|`++vehicleColor`|`body`|`number`|否|车辆颜色，详见附录A.17 车辆颜色|
|`++description`|`body`|`string`|否|车辆描述|
|`++regionIndexCode`|`body`|`string`|否|所属区域|
|`++createTime`|`body`|`string`|否|更新日期，遵守ISO8601标准，如2018-07-26T21:30:08+08:00表示北京时间2017年7月26日21时30分08秒|
|`++updateTime`|`body`|`string`|否|创建日期，遵守ISO8601标准，如2018-07-26T21:30:08+08:00表示北京时间2017年7月26日21时30分08秒|
|`++status`|`body`|`number`|否|状态码小于0则代表资源已被删除|

补充说明:
- 返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码
