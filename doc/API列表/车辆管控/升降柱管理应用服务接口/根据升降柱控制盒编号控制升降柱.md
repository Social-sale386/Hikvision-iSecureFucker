# 根据升降柱控制盒编号控制升降柱

> 接口说明
> 根据升降柱控制盒编号控制升降柱

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/lcmas/v1/deviceControl`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/lcmas/v1/deviceControl
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`command`|`body`|`integer`|是|控制命令，1-上升，2-下降，3-停止|
|`bollardIndexCode`|`body`|`string`|是|升降柱控制盒唯一标识码，可以调用【查询资源列表v2】接口查询，资源类型传“securitycheck”，或者详见附录数据字典说明(数据字典中需要补充控制盒的类型说明)|

## 请求参数示例
```json
{
  "command": 1,
  "bollardIndexCode": "zdfsdfsd2323"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|返回请求结果码|
|`message`|`body`|`string`|是|提示信息|

## 返回参数示例
```json
{
  "code": "0",
  "message": "success"
}
```
