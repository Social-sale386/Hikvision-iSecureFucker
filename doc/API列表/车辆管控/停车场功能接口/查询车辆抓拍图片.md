# 查询车辆抓拍图片

> 接口说明
> 停车库过车记录、预账单中带有车辆或人员图片信息，此接口提供这些图片信息展示的数据

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/pms/v1/image`|
|请求方法|`POST`|

## 快速调用
```text
POST /api/pms/v1/image
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`aswSyscode`|`body`|`string`|是|图片服务唯一标识码，|
|`picUri`|`body`|`string`|是|图片Uri|

补充说明:
- 从其他返回图片Uri接口处获取

## 请求参数示例
```json
{
  "aswSyscode": "hnj5h245h5234h45345y",
  "picUri": "/pic?=d7ei703i10cd*73a-d5108a--22cd0c9d6592aiid="
}
```
