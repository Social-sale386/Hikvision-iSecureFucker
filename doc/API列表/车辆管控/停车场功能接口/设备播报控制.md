# 设备播报控制

> 接口说明
> 简述：控制车道上的语音设备进行语音播报。
> 支持：通过车道或者设备编号指定需要控制的设备范围，如果传入车道，将对车道下所有的语音设备发送语音控制命令。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/device/voice/control`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/device/voice/control
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`deviceSyscode`|`body`|`string`|否|设备编号,查询出入口设备关联关系接口获取返回参数deviceSyscode,（车道编号和设备编号必填其一）|
|`roadwaySyscode`|`body`|`string`|否|车道编号,查询出入口设备关联关系接口获取返回参数roadwaySyscode,（车道编号和设备编号必填其一）|
|`voiceContent`|`body`|`string`|否|播报内容|

## 请求参数示例
```json
{
  "deviceSyscode": "agfdgdfgdfgdfg3424232342343",
  "roadwaySyscode": "aafsdf252341312hr54y54h2143",
  "voiceContent": "欢迎光临"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": null
}
```
