# 查询停车库剩余诱导车位数

> 接口说明
> 查询停车库剩余车位信息（诱导车位）

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/park/induce/remainSpaceNum`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/park/induce/remainSpaceNum
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`floorSyscode`|`body`|`string`|否|楼层唯一标识，可从查询停车库节点信息接口获取返回参数indexCode|

## 请求参数示例
```json
{
  "parkSyscode": "v45gh45gergf34g34g354g35ger",
  "floorSyscode": "hddgfgsd34de5gdg34ws5345y"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|结果总数|
|`+list`|`body`|`object[]`|否|记录数据集合|
|`++parkSyscode`|`body`|`string`|否|停车库唯一标识|
|`++parkName`|`body`|`string`|否|停车库名称|
|`++totalGuidePlace`|`body`|`number`|否|停车库诱导车位总数|
|`++leftGuidePlace`|`body`|`number`|否|停车库诱导车位剩余数|
|`++floorSpaces`|`body`|`object[]`|否|楼层车位剩余数信息|
|`+++floorSyscode`|`body`|`string`|否|楼层唯一标识码|
|`+++floorName`|`body`|`string`|否|楼层名称|
|`+++totalGuidePlace`|`body`|`number`|否|楼层诱导车位数|
|`+++leftGuidePlace`|`body`|`number`|否|楼层诱导车位剩余数|

```json
{
  "code": "0",
  "msg": "Success",
  "data": {
    "total": 10,
    "list": [
      {
        "parkSyscode": "fwergqw4rgwqfg\twefefw",

        "totalGuidePlace": 1000,

        "floorSpaces": [
          {

            "floorName": "b1",
            "totalGuidePlace": 1000,
            "leftGuidePlace": 1000
          }
        ]
      }
    ]
  }
}
```
