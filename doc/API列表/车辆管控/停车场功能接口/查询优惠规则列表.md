# 查询优惠规则列表

> 接口说明
> 简述：可通过此接口查询某个停车库或者平台所有的优惠规则。
> 支持：支持查询某个停车库的优惠规则信息，支持分页。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/reductRule/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/reductRule/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|页码，0<pageNo|
|`pageSize`|`body`|`integer`|是|每页结果数，0<pageSize≤1000|
|`parkSyscode`|`body`|`string`|否|停车场编号，获取停车库列表接口获取返回参数parkIndexCode|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 15,
  "parkSyscode": "h356j246h245gh2458ds88f0sdfh245"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总记录数|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|页大小|
|`+list`|`body`|`object[]`|否|数据集合|
|`++ruleSyscode`|`body`|`string`|否|规则唯一标识|
|`++name`|`body`|`string`|否|规则名称|
|`++parkSyscode`|`body`|`string`|否|停车场syscode|
|`++type`|`body`|`number`|否|优惠类型，|
|`++content`|`body`|`string`|否|减免内容|
|`++startTime`|`body`|`string`|否|开始时间，ISO8601时间格式|
|`++endTime`|`body`|`string`|否|结束时间，ISO8601时间格式|
|`++note`|`body`|`string`|否|规则说明|

补充说明:
- 0：优惠金额；
- 1：折扣；
- 2：全免；
- 4：优惠时长
