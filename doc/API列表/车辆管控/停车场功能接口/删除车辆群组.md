# 删除车辆群组

> 接口说明
> 根据车辆群组系统编码删除车辆群组，支持批量删除

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/carGroup/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/carGroup/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCodes`|`body`|`string`|是|车辆群组系统编码，多个用英文逗号隔开|

## 请求参数示例
```json
{
  "indexCodes": "132134234,2342345"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|请求结果码|
|`msg`|`body`|`string`|是|提示信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success"
}
```
