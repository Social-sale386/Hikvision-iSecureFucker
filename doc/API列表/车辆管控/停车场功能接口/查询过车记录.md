# 查询过车记录

> 接口说明
> 简述：当车辆进、出停车场时，会产生过车记录，过车记录信息包括车辆的车牌号、车辆所在的车道标识、出入口标识、停车场标识、出场标识、过车时间、车辆图片信息等，调用方可以使用该接口同步过车信息，了解车辆进出场情况；
> 支持：调用方可以通过该接口分页获取过车记录信息，每页记录总数不超过1000，其它查询条件之间是与的关系，可以根据这些字段进行过滤筛选查询到的数据。
> 注：车牌条件为模糊查询条件

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v1/crossRecords/page`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/crossRecords/page
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode，|
|`entranceSyscode`|`body`|`string`|否|出入口唯一标识，可从获取出入口列表接口获取返回参数entranceIndexCode，|
|`plateNo`|`body`|`string`|否|车牌号|
|`cardNo`|`body`|`string`|否|卡号|
|`startTime`|`body`|`string`|否|查询开始时间|
|`endTime`|`body`|`string`|否|查询结束时间|
|`vehicleOut`|`body`|`integer`|否|进出场标识，0-进场|
|`vehicleType`|`body`|`integer`|否|车辆类型，0：其他车|
|`releaseResult`|`body`|`integer`|否|放行结果，0-未放行|
|`releaseWay`|`body`|`integer`|否|放行方式，10-未开闸|
|`releaseReason`|`body`|`integer`|否|放行原因，100-固定车自动放行|
|`carCategory`|`body`|`string`|否|车辆分类，9-黑名单|
|`pageNo`|`body`|`integer`|是|目标页码|
|`pageSize`|`body`|`integer`|是|每页记录数|
