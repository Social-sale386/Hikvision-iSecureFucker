# 车位预约

> 接口说明
> 车辆进入停车场前预约指定停车场车位（目前提供给访客组件使用）

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/pms/v1/parkingSpace/reservations/addition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parkingSpace/reservations/addition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|是|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`plateNo`|`body`|`string`|是|车牌号码|
|`owner`|`body`|`string`|否|联系人姓名|
|`startTime`|`body`|`string`|是|预约开始时间|
|`endTime`|`body`|`string`|是|预约结束时间|
|`phoneNo`|`body`|`string`|是|手机号，纯数字|
|`plateColor`|`body`|`integer`|否|车牌颜色，不传默认蓝色|
|`keepTime`|`body`|`integer`|否|进场预留时长，单位：分钟；若不传则默认预留到预约结束时间|

补充说明:
- ISO8601格式：
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- ISO8601格式：
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 请求参数示例
```json
{
  "parkSyscode": "gb45h4h53h4g245t45y54t45",
  "plateNo": "浙A12345",
  "owner": "张三",
  "resvType": 0,
  "startTime": "2018-07-26T15:00:00+08:00",
  "endTime": "2018-07-27T15:00:00+08:00",
  "phoneNo": "13454563454",
  "plateColor": 1,
  "keepTime": 100
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-预约结果记录对象|
|`+reserveOrderNo`|`body`|`string`|否|预约单号|
|`+reserveTime`|`body`|`number`|否|预约有效期|
|`+startTime`|`body`|`string`|否|预约开始时间（格式：yyyy-MM-ddTHH:mm:ss+当前时区）|
|`+endTime`|`body`|`string`|否|预约结束时间（格式：yyyy-MM-ddTHH:mm:ss+当前时区）|

补充说明:
- 分钟

## 返回参数示例

```json
    "endTime": "2018-07-26T15:00:00+08:00"
  }
}
```
