# 车辆布防

> 接口说明
> 简述：车辆被布防后，进出场时出入口岗亭会对车辆进行相应的布防措施，完善出入口管控场景；根据布防时间段进行管控，时间段为空时则全时段布控。
> 支持：支持通过车牌号或卡号进行车辆布防。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/pms/v1/alarmCar/addition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/alarmCar/addition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`plateNo`|`body`|`string`|否|车牌号码（车牌、卡号二选一）|
|`cardNo`|`body`|`string`|否|卡号（车牌、卡号二选一）|
|`driver`|`body`|`string`|否|驾驶员名称|
|`driverPhone`|`body`|`string`|否|驾驶员电话|
|`remark`|`body`|`string`|否|备注信息|
|`endTime`|`body`|`string`|否|布防结束时间,ISO8601格式：|
|`regionIndexCode`|`body`|`string`|否|区域IndexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`parkingIndexCode`|`body`|`string`|否|停车库编号，不传默认该区域下所有停车库|
|`plateColor`|`body`|`integer`|否|车牌颜色，不传默认蓝色|

补充说明:
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 请求参数示例
```json
{
  "plateNo": "浙A12345",
  "driver": "周三",
  "driverPhone": "18969188862",
  "remark": "test",
  "endTime": "2018-07-27T15:00:00+08:00",
  "regionIndexCode": "root00000",
  "parkingIndexCode": "1231231234",
  "plateColor": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-布防车辆对象|
|`+alarmSyscode`|`body`|`string`|否|布防车辆唯一标识|
|`+plateNo`|`body`|`string`|否|车牌号码|
|`+cardNo`|`body`|`string`|否|卡号|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "alarmSyscode": "g3g5hy25t234t45y22t34",
    "plateNo": "浙A12345",
    "cardNo": null
  }
}
```
