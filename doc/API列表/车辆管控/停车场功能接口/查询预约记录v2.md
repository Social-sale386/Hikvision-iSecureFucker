# 查询预约记录v2

> 接口说明
> 简述：提供第三方通过接口查询平台预约记录；
> 支持：支持通过车牌号、预约状态、预约方式、预约类型、停车库及时间段查询预约信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v2/reserveRecord/page`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v2/reserveRecord/page
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`plateNo`|`body`|`string`|否|车牌号|
|`resvState`|`body`|`integer`|否|预约状态，0:已预约|
|`resvWay`|`body`|`integer`|否|预约方式，4:后台管理员|
|`allowTimes`|`body`|`integer`|否|允许进出次数，0：单次|
|`isCharge`|`body`|`integer`|否|是否收费，0：收费|
|`startTime`|`body`|`string`|否|查询开始时间,ISO8601格式：|
|`endTime`|`body`|`string`|否|查询结束时间,ISO8601格式：|
|`pageNo`|`body`|`integer`|是|目标页码,pageNo≥1|
|`pageSize`|`body`|`integer`|是|每页记录数,0<pageSize≤1000|

补充说明:
- 1:用户取消
- 2:到期
- 3:已进场
- 4:已出场
- 5:访客平台取消
- 5:第三方预约
- 6:访客
- 1：多次
- 1：免费
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 请求参数示例
```json
{
  "parkSyscode": "dfasgqegbq34g3q4g34g",
  "plateNo": "浙A12345",
  "resvState": 1,
  "resvWay": 4,
  "allowTimes": 1,
  "isCharge": 1,
  "startTime": "2018-07-26T15:00:00+08:00",
  "endTime": "2018-07-27T15:00:00+08:00",
  "pageNo": 1,
  "pageSize": 15
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否||
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-预约记录分页结构|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+list`|`body`|`object[]`|否|预约记录对象列表|
|`++recordSyscode`|`body`|`string`|否|预约记录唯一标识|
|`++reserveOrderNo`|`body`|`string`|否|预约单号|
|`++parkSyscode`|`body`|`string`|否|停车库唯一标识|
|`++parkName`|`body`|`string`|否|停车库名称|
|`++plateNo`|`body`|`string`|否|车牌号|
|`++phoneNo`|`body`|`string`|否|手机号|
|`++ower`|`body`|`string`|否|联系人|
|`++resvState`|`body`|`number`|否|预约状态，0:已预约|
|`++resvWay`|`body`|`number`|否|预约方式，4:后台管理员|
|`++allowTimes`|`body`|`number`|否|允许进出次数，0：单次|
|`++isCharge`|`body`|`number`|否|是否收费，0：收费|
|`++startTime`|`body`|`string`|否|预约开始时间|
|`++endTime`|`body`|`string`|否|预约结束时间|
|`++plateColorName`|`body`|`string`|否|车牌颜色|
|`++plateColor`|`body`|`number`|否|车牌颜色|

补充说明:
- 返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码
- 1:用户取消
- 2:到期
- 3:已进场
- 4:已出场
- 5:第三方预约
- 6:访客
- 1：多次
- 1：免费
