# 查询停车库剩余车位数

> 接口说明
> 简述：当车辆出入停车场时，停车场的车位数量随之会发生变化，此接口可获取某个停车库的车位使用数量信息；
> 支持：支持通过停车库唯一标识获取当前停车库车位剩余数目情况。
> 支持停车库唯一标识为空时，获取全部停车库的车位剩余情况。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/pms/v1/park/remainSpaceNum`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/park/remainSpaceNum
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode，|

补充说明:
- 为空时获取全部停车库的车位剩余情况

## 请求参数示例
```json
{
  "parkSyscode": "v45gh45gergf34g34g354g35ger"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object[]`|否|返回数据-剩余车位数信息结构集合|
|`+parkSyscode`|`body`|`string`|否|停车库唯一标识码|
|`+parkName`|`body`|`string`|否|停车库名称|
|`+parentParkSyscode`|`body`|`string`|否|父停车库唯一标识|
|`+totalPlace`|`body`|`number`|否|停车库车位总数|
|`+totalPermPlace`|`body`|`number`|否|停车库固定车位总数|
|`+totalReservePlace`|`body`|`number`|否|停车库预约车位总数|
|`+leftPlace`|`body`|`number`|否|停车库车位剩余数|
|`+leftPermPlace`|`body`|`number`|否|停车库固定车位剩余数|
|`+leftReservePlace`|`body`|`number`|否|停车库预约车位剩余数|
