# 对视频桩下发车位白名单

> 接口说明
> 对视频桩下发车位白名单

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/parkingSpace/whiteCarControl`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parkingSpace/whiteCarControl
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`listType`|`body`|`string`|是|视频桩名单类型，blockList：非授权名单，allowList：授权名单|
|`plateColor`|`body`|`string`|是|车牌颜色（black：黑色，blue：蓝色，civilAviationBlack：民航黑色，civilAviationGreen：民航绿色，golden：金色，green：绿色，mixedColor：花底，newEnergyGreen：新能源绿色，newEnergyYellowGreen：新能源黄绿色，red：橙色，white：白色，yellow：黄色，red：红色，other：其他颜色，unknown：未知）|
|`licensePlate`|`body`|`string`|是|车牌号，范围[1,32]，车牌号+车牌颜色可以作为唯一标志进行名单权限时间修改|
|`spaceNo`|`body`|`string`|是|车位号|
|`floorSyscode`|`body`|`string`|是|楼层唯一标识码，可从查询车位信息接口获取返回参数floorSyscode|
|`createTime`|`body`|`string`|是|有效期开始日期|
|`effectiveTime`|`body`|`string`|是|有效终止时间|

## 请求参数示例
```json
{
  "listType": "1231231",
  "plateColor": "blue",
  "licensePlate": "浙A11111",
  "spaceNo": "B10",
  "floorSyscode": "23542sdfsdfs",
  "createTime": "2022-01-01T00:00:00+08:00",
  "effectiveTime": "2022-02-01T00:00:00+08:00"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|返回码，0 – 成功，其他- 失败|
|`msg`|`body`|`string`|是|返回描述|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
