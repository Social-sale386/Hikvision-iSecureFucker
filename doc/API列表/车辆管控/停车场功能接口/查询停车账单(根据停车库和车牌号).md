# 查询停车账单(根据停车库和车牌号)

> 接口说明
> 缴费预账单是停车缴费在特定收费规则计算下的结算信息，包括车牌号码、停车时长、应收费用、总费用等一系列表明停车缴费关系的参数，为缴费提供数据来源和参考

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/pms/v1/pay/quickPreBill`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/pay/quickPreBill
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode，|
|`plateNo`|`body`|`string`|是|车牌号码（最大长度16）|

补充说明:
- 不填则针对所有停车库进行查询
- 长度不少于3个字符

## 请求参数示例
```json
{
  "parkSyscode": "fq34f3gq345gq34g34g34",
  "plateNo": "浙A12345"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-账单信息|
|`+parkSyscode`|`body`|`string`|否|停车库唯一标|
|`+chargeRuleSyscode`|`body`|`string`|否|收费规则唯一标，正常收费情况下有此信息|
|`+parkName`|`body`|`string`|否|停车库名称|
|`+inRecordSyscode`|`body`|`string`|否|停车信息唯一标识|
|`+enterTime`|`body`|`string`|否|入场时间 ISO8601格式： yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间： 2018-07-26T15:00:00+08:00 详见ISO8601协议解释|
|`+vehiclePicUri`|`body`|`string`|否|入场车辆图片uri，设备上报图片事件后，才有此信息|
|`+plateNoPicUri`|`body`|`string`|否|入场车牌图片uri，设备上报图片事件后，才有此信息|
|`+aswSyscode`|`body`|`string`|否|图片服务唯一标，设备上报图片事件后，才有此信息|
|`+plateNo`|`body`|`string`|否|车牌号|
|`+cardNo`|`body`|`string`|否|卡号|
|`+parkTime`|`body`|`number`|否|停车时长，分钟|
|`+billSyscode`|`body`|`string`|否|账单唯一标识|
|`+calcType`|`body`|`number`|否|账单计算类型 0-首次收费 1-超时收费|
|`+supposeCost`|`body`|`string`|否|应收金额（最大长度8） 单位：元|
|`+deduceCost`|`body`|`string`|否|减免金额（最大长度8） 单位：元|
|`+paidCost`|`body`|`string`|否|已支付金额（最大长度8） 单位：元|
|`+totalCost`|`body`|`string`|否|账单总金额（最大长度8） 单位：元|
|`+isUsedCoupon`|`body`|`number`|否|是否使用优惠券 0：不使用 1：使用|
|`+couponCode`|`body`|`string`|否|优惠券码（最大长度64）|
|`+couponUsedMsg`|`body`|`number`|否|优惠券失败原因 1:抵扣券已失效 2:抵扣券不存在 3:请使用当前停车场抵扣券 4:超时缴费不能使用抵扣券减免 5:已使用抵扣券减免 6:已使用过优惠券，不能多次使用|
|`+delayTime`|`body`|`number`|否|缴费后允许延时出场时间 单位：分钟|
|`+currentDeduceCost`|`body`|`string`|否|本次账单减免金额，单位：元|
