# 查询停车账单(根据停车信息)

> 接口说明
> 简述：缴费预账单是停车缴费在特定收费规则计算下的结算信息，包括车牌号码、停车时长、应收费用、总费用等一系列表明停车缴费关系的参数，为缴费提供数据来源和参考；
> 支持：支持通过特定过车记录唯一标识定位应缴费车辆，生成账单；
> 支持通过车道信息获取当前车道的未放行应缴费车辆，生成账单。
> 支持通过停车库标识、完整车牌号码定位应缴费车辆，生成账单。
> 支持优惠券抵扣，若请求带有有效优惠券参数，生成优惠后的账单。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/pay/preBill`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/pay/preBill
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`inRecordSyscode`|`body`|`string`|否|场内车停车信息唯一标识，可从查询场内车停车信息接口获取返回参数inRecordSyscode，inRecordSyscode与roadwaySyscode，plateNo三选一使用|
|`roadwaySyscode`|`body`|`string`|否|车道唯一标识，可从获取车道列表接口获取返回参数roadwayIndexCode，inRecordSyscode与roadwaySyscode，plateNo三选一使用|
|`plateNo`|`body`|`string`|否|车牌号码（完整车牌），inRecordSyscode与roadwaySyscode，plateNo三选一使用|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode，以plateNo查询时，此字段必填|
|`couponCode`|`body`|`string`|否|优惠券码(与条件deductRuleType，deductContent二选一使用)|
|`deductRuleType`|`body`|`integer`|否|优惠规则类型(与条件couponCode二选一使用)， 0：优惠金额； 1：折扣； 2：全免； 4：优惠时长|
|`deductContent`|`body`|`string`|否|减免内容(与deductRuleType同时使用)(deductRuleType为0，单元:分，deductRuleType为1，单位:%，deductRuleType为4，单位:分钟)|

## 请求参数示例
```json
{
  "inRecordSyscode": "fq34f3gq345gq34g34g34",
  "roadwaySyscode": "ukg43ygkj23h234324l",
  "plateNo": "浙A13456",
  "parkSyscode": "a8f78ywecy99r99",
  "couponCode": "123143",
  "deductRuleType": 1,
  "deductContent": "10"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据-账单信息|
|`+parkSyscode`|`body`|`string`|否|停车库唯一标识|
|`+chargeRuleSyscode`|`body`|`string`|否|收费规则唯一标识，正常收费情况下有此信息|
|`+parkName`|`body`|`string`|否|停车库名称|
|`+inRecordSyscode`|`body`|`string`|否|停车信息唯一标识|
|`+enterTime`|`body`|`string`|否|入场时间|
|`+vehiclePicUri`|`body`|`string`|否|入场车辆图片uri，设备上报图片事件后，才有此信息|
|`+plateNoPicUri`|`body`|`string`|否|入场车牌图片uri，设备上报图片事件后，才有此信息|
|`+aswSyscode`|`body`|`string`|否|图片服务唯一标识，设备上报图片事件后，才有此信息|
|`+plateNo`|`body`|`string`|否|车牌号|
|`+cardNo`|`body`|`string`|否|卡号|
|`+parkTime`|`body`|`number`|否|停车时长，|
|`+billSyscode`|`body`|`string`|否|账单唯一标识|
|`+calcType`|`body`|`number`|否|账单计算类型|
|`+supposeCost`|`body`|`string`|否|应收金额，|
|`+deduceCost`|`body`|`string`|否|减免金额，|
|`+paidCost`|`body`|`string`|否|已支付金额，|
|`+totalCost`|`body`|`string`|否|账单总金额，|
|`+couponCode`|`body`|`string`|否|优惠券码|
|`+isUsedCoupon`|`body`|`number`|否|是否使用优惠券|
|`+delayTime`|`body`|`number`|否|缴费后允许延时出场时间|
|`+couponUsedMsg`|`body`|`number`|否|优惠券失败原因|
|`+currentDeduceCost`|`body`|`string`|否|本次账单减免金额，单位：元|

- 分钟
- 0-首次收费 1-超时收费
- 单位：元
- 单位：元
- 单位：元
- 单位：元
- 0：不使用
- 1：使用
- 单位：分钟
- 1:抵扣券已失效
- 2:抵扣券不存在

- 5:已使用抵扣券减免
