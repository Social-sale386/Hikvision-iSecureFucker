# 车辆批量充值接口

> 接口说明
> 支持通过车牌号进行特定停车场的包期充值。若车辆不存在，则先添加车辆再充值。车辆包期时段全量更新，不支持增量添加包期时段

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/car/batchCharge`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/car/batchCharge
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`vehicleIds`|`body`|`string`|否|车辆id，多个逗号分割，与车牌信息二选一使用|
|`vehiclePlateNos`|`body`|`object[]`|否|车牌信息集合，与车辆id二选一使用|
|`+plateNo`|`Body`|`string`|是|车牌号|
|`+plateColor`|`Body`|`number`|否|车牌颜色，不传默认蓝色|
|`parkSyscode`|`body`|`string`|是|停车库编号，多个停车库用英文逗号分割|
|`startTime`|`body`|`string`|是|充值开始时间（格式：yyyy-MM-dd,默认当天开始,兼容格式：yyyy-MM-dd)|
|`endTime`|`body`|`string`|是|充值结束时间（格式：yyyy-MM-dd,默认当天结束,兼容格式：yyyy-MM-dd）|
|`lanes`|`body`|`string`|否|车道indexCode，多个车道用英文逗号分割，不传默认停车库下所有车道有权限|

## 请求参数示例
```json
{
  "vehicleIds": "gb45h4h53h4,2534534534",
  "vehiclePlateNos": [
    {
      "plateNo": "琼E8AB036",
      "plateColor": 1
    }
  ],
  "parkSyscode": "gb45h4h53h4,2534534534",
  "startTime": "2018-07-26",
  "endTime": "2018-07-27",
  "lanes": "gb45h4h53h4,2534534534"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|返回请求结果码|
|`msg`|`body`|`string`|是|返回提示信息|
|`data`|`body`|`string`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": null
}
```
