# 账单支付确认v2

> 接口说明
> 简述：账单支付确认是缴费预账单完成支付流程后和平台同步结算信息的操作，记录一次停车缴费的结算结果；
> 支持： 支持通过账单号确认账单支付。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v2/pay/receipt`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v2/pay/receipt
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`billSyscode`|`body`|`string`|是|账单唯一标识（最大长度64），可从查询停车账单接口获取返回参数billSyscode|

## 请求参数示例
```json
{
  "billSyscode": "gbgrtg2345g245h245g2345g5g"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息（最大长度256）|
|`data`|`body`|`string`|否|返回数据|

补充说明:
- 为空

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
