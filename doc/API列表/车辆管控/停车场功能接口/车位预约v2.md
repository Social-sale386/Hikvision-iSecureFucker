# 车位预约v2

> 接口说明
> 简述：车辆进入停车场前预约指定停车场车位（目前提供给访客组件使用）。
> 支持按时间段进行车辆预约，根据需要配置是否收费、进出次数及联系人信息；在预约时间段内有效，可进行一次或多次进出场，以及出场时收费还是免费放行。
> 车位预约有诸多限制，允许临时车预约，固定车、一户多车车辆、黑名单车辆、场内车不允许预约，同一车牌号只能预约一次。
> 支持：支持通过停车库唯一标识、车牌号等信息进行车位预约。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v2/parkingSpace/reservations/addition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v2/parkingSpace/reservations/addition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|是|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`plateNo`|`body`|`string`|是|车牌号码|
|`phoneNo`|`body`|`string`|否|手机号|
|`owner`|`body`|`string`|否|联系人姓名|
|`allowTimes`|`body`|`integer`|否|允许进出次数，不传默认单次，|
|`isCharge`|`body`|`string`|否|是否收费，不传默认收费，0：收费|
|`plateColor`|`body`|`integer`|否|车牌颜色，不传默认蓝色|
|`keepTime`|`body`|`integer`|否|进场预留时长，单位：分钟；若不传则默认预留到预约结束时间|
|`lanes`|`body`|`string`|否|车道编号，多个逗号分割，不传则默认所有车道有权限|
|`resvTime`|`body`|`object[]`|是|预约时间段，支持多段时间|
|`+startTime`|`Body`|`string`|是|预约开始时间（格式：yyyy-MM-ddTHH:mm:ss+当前时区）|
|`+endTime`|`Body`|`string`|是|预约结束时间（格式：yyyy-MM-ddTHH:mm:ss+当前时区）|

补充说明:
- 纯数字，位数暂不限制
- 0：单次
- 1：多次
- 1：免费

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-预约结果记录对象|
|`+resvList`|`body`|`object[]`|否|车位预约（V2）返回结果集合|
|`++reserveOrderNo`|`body`|`string`|否|预约单号|
|`++startTime`|`body`|`string`|否|预约开始时间（格式：yyyy-MM-ddTHH:mm:ss+当前时区）|
|`++endTime`|`body`|`string`|否|预约结束时间（格式：yyyy-MM-ddTHH:mm:ss+当前时区）|
|`++reserveTime`|`body`|`number`|否|预约有效期（单位:分钟）|
