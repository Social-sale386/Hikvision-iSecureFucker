# 取消车辆布防

> 接口说明
> 简述：车辆取消布防后，进出场和普通车辆一样，不再被布防。
> 支持：支持通过布防车辆编号集合取消车辆布防。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/pms/v1/alarmCar/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/alarmCar/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`alarmSyscodes`|`body`|`string`|是|布防车辆唯一标识集合(编号间用‘,’分隔)，从车辆布防接口获取返回参数alarmSyscode|

## 请求参数示例
```json
{
  "alarmSyscodes": "4h53h4g245t45y54,fg453fhgwgr4g"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
