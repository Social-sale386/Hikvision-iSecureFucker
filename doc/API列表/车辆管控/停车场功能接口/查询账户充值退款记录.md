# 查询账户充值退款记录

> 接口说明
> 简述：提供对接方通过接口查询平台账户充值、退款记录。
> 支持：支持通过账户名称、交易类型、交易方式、收费员及时间段进行账户充值退款记录查询。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/pms/v1/account_charge/record/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/account_charge/record/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`account`|`body`|`string`|否|账户名称|
|`tradeType`|`body`|`integer`|否|交易类型，0：充值|
|`tradeWay`|`body`|`double`|否|交易方式，|
|`startTime`|`body`|`string`|否|查询开始时间，ISO8601格式：|
|`endTime`|`body`|`string`|否|查询结束时间，ISO8601格式：|
|`operator`|`body`|`string`|否|收费员名称|
|`pageNo`|`body`|`integer`|是|目标页码，范围 ( 0 , ~ ) ，若是范围内数字超过实际最大页码值，返回最后一页记录|
|`pageSize`|`body`|`integer`|是|每页记录数，范围 ( 0 , 1000 ]|

补充说明:
- 1：退款
- 1：现金
- 4：支付宝
- 5：微信
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 请求参数示例
```json
{
  "account": "詹三",
  "tradeType": 0,
  "tradeWay": 1,
  "startTime": "2018-07-26T15:00:00+08:00",
  "endTime": "2018-07-26T15:00:00+08:00",
  "operator": "test",
  "pageNo": 1,
  "pageSize": 15
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否||
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+list`|`body`|`object[]`|否|账户充值退款记录对象列表|
|`++account`|`body`|`string`|否|账户名称|
|`++tradeType`|`body`|`number`|否|交易类型，0：充值|
|`++tradeWay`|`body`|`number`|否|交易方式，|
|`++fee`|`body`|`string`|否|交易金额，单位：元|
|`++tradeTime`|`body`|`string`|否|交易时间，ISO8601格式：|
|`++operator`|`body`|`string`|否|收费员名称|

补充说明:
- 返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码
- 1：退款
- 1：现金
- 4：支付宝
- 5：微信
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "pageNo": 1,
    "pageSize": 15,
    "list": [
      {
        "account": "詹三",
        "tradeType": 1,
        "tradeWay": 1,
        "fee": "200.00",
        "tradeTime": " 2018-07-27T15:00:00+08:00",
        "operator": "test"
      }
    ]
  }
}
```
