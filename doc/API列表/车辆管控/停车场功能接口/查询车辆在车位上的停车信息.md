# 查询车辆在车位上的停车信息

> 接口说明
> 简述：用于寻车时终点位置的查询，可以按车牌号、车位号、停车时间段和无牌车不同业务场景分别查询出车辆停车信息。
> 按车牌号：查询此车牌停在车位上的信息
> 按车位号：查询车辆停在此车位上的信息
> 按停车时间段：查询此时间段车辆停车的信息
> 按无牌车：查询场内无牌车的停车信息
> 支持：支持通过车牌号、车位号、停车时间段、无牌车和停车场编号查询车辆的停车信息；支持分页查询。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/pms/v1/parkingRecord/query`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parkingRecord/query
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`queryType`|`body`|`integer`|是|查询标识（用于标识接口以哪种方式查询，1：车牌查询 2：车位查询 3：停车时间段查询 4：无牌车停车查询）|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`plateNo`|`body`|`string`|否|车牌号码|
|`spaceNo`|`body`|`string`|否|车位号|
|`startTime`|`body`|`string`|否|查询开始时间,ISO8601格式：|
|`endTime`|`body`|`string`|否|查询结束时间,ISO8601格式：|
|`pageNo`|`body`|`integer`|是|目标页码|
|`pageSize`|`body`|`integer`|是|每页记录数|

补充说明:
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：2018-07-26T15:00:00+08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：2018-07-27T15:00:00+08:00

## 请求参数示例
```json
{
  "queryType": 1,
  "parkSyscode": "hddgfgsd34de5gdg34ws5345y",
  "plateNo": "浙A12345",
  "spaceNo": "12",
  "startTime": "2018-07-26T15:00:00+08:00",
  "endTime": "2018-07-27T15:00:00+08:00",
  "pageNo": 1,
  "pageSize": 15
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-停车信息集合|
|`+total`|`body`|`number`|否|数据总数|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|每页数据数|
|`+totalPage`|`body`|`number`|否|总页数|
|`+list`|`body`|`object[]`|否|返回数据-停车信息集合|
|`++recordSyscode`|`body`|`string`|否|停车记录编号|
|`++spaceSyscode`|`body`|`string`|否|车位编号|
|`++spaceNo`|`body`|`string`|否|车位号|
|`++spacePicUri`|`body`|`string`|否|车位图片uri|
|`++parkingTime`|`body`|`string`|否|停车时间,ISO8601格式：|
|`++parkSyscode`|`body`|`string`|否|车辆所在停车库唯一标识|
|`++parkName`|`body`|`string`|否|车辆所在停车库名称|
|`++floorSyscode`|`body`|`string`|否|车辆所在楼层唯一标识|
|`++floorName`|`body`|`string`|否|车辆所在楼层名称|
|`++plateNoPicUri`|`body`|`string`|否|车牌照片uri|
|`++aswSyscode`|`body`|`string`|否|图片服务唯一标识|
|`++plateNo`|`body`|`string`|否|车牌号码|
|`++spacePicUrl`|`body`|`string`|否|车位图片url|
|`++plateNoPicUrl`|`body`|`string`|否|车牌照片url|

补充说明:
- 0: 成功
- 其他参考附录E.3.1 停车场应用错误码
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
