# 生成优惠券

> 接口说明
> 简述：优惠券是优惠规则使用的体现，优惠规则建立后进行优惠券的发放，用户进出场缴费时可用优惠券进行减免优惠。
> 支持：支持针对某个优惠规则进行优惠券的发放；如果请求中传入车牌，将对车牌和优惠券进行绑定；支持传入优惠规则详情（指定优惠规则类型和减免内容）与传入的车牌进行绑定优惠。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/coupon/addition`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/coupon/addition
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`coupons`|`body`|`object[]`|是|优惠券参数列表|
|`+ruleSyscode`|`Body`|`string`|否|优惠规则syscode，可从查询优惠规则列表接口获取返回参数ruleSyscode|
|`+merchantSyscode`|`Body`|`string`|否|商户syscode，可从查询商户列表接口获取返回参数merchantSyscode|
|`+couponCode`|`Body`|`string`|是|优惠券码|
|`+plateNo`|`Body`|`string`|否|车牌号|
|`+deductRuleType`|`Body`|`number`|否|优惠规则类型(与条件ruleSyscode二选一使用)， 0：优惠金额； 1：折扣； 2：全免； 4：优惠时长|
|`+deductContent`|`Body`|`number`|否|减免内容(与deductRuleType同时使用)(deductRuleType为0，单元:分，deductRuleType为1，单位:%，deductRuleType为4，单位:分钟)|
|`+parkSyscode`|`Body`|`string`|否|停车库编号，与deductRuleType同时使用，停车库唯一标识，可从查询停车库节点信息接口获取返回参数indexCode|
|`+startTime`|`Body`|`string`|否|优惠券有效期开始时间，与deductRuleType同时使用，ISO8601格式：|
|`+endTime`|`Body`|`string`|否|优惠券有效期结束时间，与deductRuleType同时使用，ISO8601格式：|
|`+plateColor`|`Body`|`number`|否|车牌颜色，不传默认蓝色|

补充说明:
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 请求参数示例
```json
{
  "coupons": [
    {
      "ruleSyscode": " 9220ef23cb6a4cb4a35ae94e35aea721 ",
      "merchantSyscode": "29124f98527146dCCa43f631ef507210",
      "couponCode": " 123165383 ",
      "plateNo": "浙A11111 ",
      "deductRuleType": 1,
      "deductContent": 10,
      "parkSyscode": "da307c5534ec4c4293ae141067a9c0a1",
      "startTime": "2018-07-26T15:00:00+08:00",
      "endTime": "2018-07-27T15:00:00+08:00",
      "plateColor": 1
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": null
}
```
