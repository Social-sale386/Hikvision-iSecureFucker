# 根据车牌或者卡号查询车辆群组

> 接口说明
> 支持根据车牌或者卡号查询所在的车辆群组信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/carGroup/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/carGroup/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`plateNo`|`body`|`string`|否|车牌号，与卡号不能同时为空|
|`cardNo`|`body`|`string`|否|卡号，与车牌号不能同时为空|
|`regionIndexCode`|`body`|`string`|是|区域编码|

## 请求参数示例
```json
{
  "plateNo": "浙A12345",
  "cardNo": "",
  "regionIndexCode": "34234234"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|请求结果码|
|`msg`|`body`|`string`|是|提示信息|
|`data`|`body`|`object`|否|群组信息|
|`+indexCode`|`body`|`string`|是|车辆群组系统编码|
|`+name`|`body`|`string`|是|群组名称|
|`+remark`|`body`|`string`|否|群组描述信息|
|`+regionIndexCode`|`body`|`string`|是|区域编码|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "indexCode": "324532452",
    "name": "群组名称",
    "remark": "群组描述信息",
    "regionIndexCode": "1423425345"
  }
}
```
