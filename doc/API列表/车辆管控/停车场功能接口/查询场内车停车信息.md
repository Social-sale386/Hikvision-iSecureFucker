# 查询场内车停车信息

> 接口说明
> 简述：场内车停车信息即为某一停车库或部分停车库内未出场车辆的信息，包括车牌号、车辆入场时间、车辆图片等，是用于停车场缴费、场内找车等业务的前置业务场景。
> 支持：支持通过停车库的唯一标识、车牌号码（模糊）、停车时长及停车库信息查询场内车停车信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/pms/v1/tempCarInRecords/page`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/tempCarInRecords/page
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode， 不填则针对所有停车库进行查询|
|`plateNo`|`body`|`string`|否|车牌号码，至少包含3个字符|
|`cardNo`|`body`|`string`|否|卡号|
|`parkTime`|`body`|`string`|否|停车时长，单位：小时，与“查询开始时间、查询结束时间”互斥使用|
|`startTime`|`body`|`string`|否|与“停车时长”查询互斥使用|
|`endTime`|`body`|`string`|否|与“停车时长”查询互斥使用|
|`pageNo`|`body`|`integer`|是|目标页码，范围 ( 0 , ~ )，若是范围内数字超过实际最大页码值|
|`pageSize`|`body`|`integer`|是|每页记录数，范围 ( 0 , 1000 ]|
|`operation`|`body`|`string`|否|是否开启车牌相识度匹配查询,0:关闭(不填默认),1:开启|

补充说明:
- ISO8601格式：
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- ISO8601格式：
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 请求参数示例
```json
{
  "parkSyscode": "v45gh45gergf34g34g354g35ger",
  "plateNo": "浙A12345",
  "cardNo": "55446622",
  "parkTime": "10",
  "startTime": "2018-07-26T15:00:00+08:00",
  "endTime": "2018-07-26T17:00:00+08:00",
  "pageNo": 1,
  "pageSize": 15,
  "operation": "1"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+list`|`body`|`object[]`|否|对象列表-停车信息集合|
|`++inRecordSyscode`|`body`|`string`|否|停车信息唯一标识|
|`++vehiclePicUri`|`body`|`string`|否|车辆图片uri，设备上报图片事件后，才有此信息|
|`++cardNo`|`body`|`string`|否|卡号，和车牌信息不一定同时存在|
|`++inTime`|`body`|`string`|否|入场时间，ISO8601格式：|
|`++parkTime`|`body`|`string`|否|停车时长，(x天x小时x分钟)|
|`++parkSyscode`|`body`|`string`|否|车辆所在停车库唯一标识|
|`++parkName`|`body`|`string`|否|车辆所在停车库名称|
|`++plateNoPicUri`|`body`|`string`|否|车牌照片uri，设备上报图片事件后，才有此信息|
|`++aswSyscode`|`body`|`string`|否|图片服务唯一标识，设备上报图片事件后，才有此信息|
|`++plateNo`|`body`|`string`|否|车牌号码，和卡号信息不一定同时存在|

补充说明:
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- 此字段仅用作展示，如要获取具体时间可根据返回的入场时间计算，展示依据接口传递的
