# 车位绑定车辆

> 接口说明
> 简述：车库车位有固定车位、特殊车位等不同车位使用场景，绑定车辆后可以控制车辆停放的权限；绑定的车辆可以停放，非绑定车辆不可以停放并给与提示。
> 支持：支持车库车位绑定车辆车牌信息，支持批量绑定。绑定区分允许停放车辆和禁止停放车辆，需设定，默认允许停放类型。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v1/parking_space/car_bind/add`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parking_space/car_bind/add
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`spaceNos`|`body`|`string`|是|绑定的车位集，多个用‘,’分隔，可从查询车位信息接口获取返回参数spaceNo|
|`floorSyscode`|`body`|`string`|是|楼层唯一标识码，可从查询车位信息接口获取返回参数floorSyscode|
|`plateNos`|`body`|`string`|是|绑定的车牌集合，多个用‘,’分隔|
|`bindType`|`body`|`integer`|否|绑定类型，0：允许停放（默认）|

补充说明:
- 1：禁止停放

## 请求参数示例
```json
{
  "spaceNos": "12,23,43,45,125",
  "floorSyscode": "hddgfgsd34de5gdg34ws5345y",
  "plateNos": "浙A12345,京A00001,沪C32KG9",
  "bindType": 0
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
