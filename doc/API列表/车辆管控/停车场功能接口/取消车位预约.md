# 取消车位预约

> 接口说明
> 车辆进入停车场前取消预约

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/pms/v1/parkingSpace/reservations/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parkingSpace/reservations/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`reserveOrderNo`|`body`|`string`|是|预约单号，从车位预约v2接口获取返回参数reserveOrderNo|
|`way`|`body`|`integer`|否|取消方式，1:用户取消（默认）|

补充说明:
- 5:访客平台取消

## 请求参数示例
```json
{
  "reserveOrderNo": "6452t245y24ghgf3egq345g34",
  "way": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否||
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`string`|否|返回数据|

补充说明:
- 返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
