# 查询商户列表

> 接口说明
> 简述：可通过此接口查询平台的商户信息。
> 支持：支持查询平台的商户信息，支持分页

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/merchant/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/merchant/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|页码，0<pageNo|
|`pageSize`|`body`|`integer`|是|每页结果数，0<pageSize≤1000|
|`name`|`body`|`string`|否|商户名称|
|`businessLicense`|`body`|`string`|否|营业许可证|
|`manager`|`body`|`string`|否|经营者姓名|
|`phone`|`body`|`string`|否|联系方式|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 15,
  "name": "海康自营",
  "businessLicense": "1234567812345678",
  "manager": "海康",
  "phone": "0571-100100100"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总记录数|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|页大小|
|`+list`|`body`|`object[]`|否|数据集合|
|`++businessLicense`|`body`|`string`|否|营业许可证|
|`++manager`|`body`|`string`|否|法人/经营者姓名|
|`++merchantSyscode`|`body`|`string`|否|商户系统编号|
|`++name`|`body`|`string`|否|户名称|
|`++phone`|`body`|`string`|否|电话号码|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": {
    "total": 1,
    "pageNo": 1,
    "pageSize": 1,
    "list": [
      {
        "businessLicense": "1234567812345678",
        "manager": "海康",
        "merchantSyscode": "tyuiuvgioasdhaugdhj",
        "name": "海康自营",
        "phone": "0571-100100100"
      }
    ]
  }
}
```
