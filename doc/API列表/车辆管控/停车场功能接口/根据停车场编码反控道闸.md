# 根据停车场编码反控道闸

> 接口说明
> 简述：车道下的道闸拥有关闸、常开的批量管控能力，为调用方提供批量控闸能力；
> 支持：支持通过控闸命令批量控制停车库下的所有道闸。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/pms/v1/deviceControlBatch`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/deviceControlBatch
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|是|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`command`|`body`|`integer`|是|控闸命令（0：关闸 3：常开）|

## 请求参数示例
```json
{
  "parkSyscode": "hddgfgsd34de5gdg34ws5345y",
  "command": 0
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`string`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
