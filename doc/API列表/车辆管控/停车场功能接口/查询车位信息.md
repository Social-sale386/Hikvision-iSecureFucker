# 查询车位信息

> 接口说明
> 根据车位号以及停车库编码查询车位信息。当参数停车库编码为空，则根据车位号查询所有停车库车位信息；当参数停车库编码不为空，则根据绑定的车牌号查询当前停车库的车位信息。支持通过车位号和停车场编号查询车位信息，通过车位状态查询空闲车位、已停车车位的信息，支持查询绑定某个车辆的车位信息，可分页查询。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v1/parkingSpace/spaceNo`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parkingSpace/spaceNo
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`spaceNo`|`body`|`string`|否|车位号，多个用‘,’分隔|
|`state`|`body`|`integer`|否|车位状态，0：空闲|
|`bindCars`|`body`|`string`|否|绑定的车辆车牌号，多个用‘,’分隔|
|`pageNo`|`body`|`integer`|是|目标页码，范围 ( 0 , ~ )，若是范围内数字超过实际最大页码值，返回最后一页记录|
|`pageSize`|`body`|`integer`|是|每页记录数，0 < pageSize≤1000|

补充说明:
- 1：停车

## 请求参数示例
```json
{
  "parkSyscode": "hddgfgsd34de5gdg34ws5345y",
  "spaceNo": "21",
  "state": 0,
  "bindCars": "“浙M32HF9",
  "pageNo": 1,
  "pageSize": 15
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-车位信息分页结构|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+list`|`body`|`object[]`|否|对象列表-车位信息集合|
|`++spaceSyscode`|`body`|`string`|否|车位编号|
|`++spaceNo`|`body`|`string`|否|车位号|
|`++parkSyscode`|`body`|`string`|否|车位所在停车库唯一标识|
|`++parkName`|`body`|`string`|否|车位所在停车库名称|
|`++floorSyscode`|`body`|`string`|否|车位所在楼层唯一标识|
|`++floorName`|`body`|`string`|否|车位所在楼层名称|
|`++longitude`|`body`|`string`|否|车位经度|
|`++latitude`|`body`|`string`|否|车位维度|
|`++cameraSyscode`|`body`|`string`|否|车位相机唯一标识|
|`++state`|`body`|`number`|否|车位状态（是否停车），0：空闲|
|`++parkingTime`|`body`|`string`|否|车辆停入时间，ISO8601格式：|
|`++spacePicUri`|`body`|`string`|否|车位图片uri|
|`++plateNoPicUri`|`body`|`string`|否|车牌照片uri|
|`++aswSyscode`|`body`|`string`|否|图片服务唯一标识|
|`++plateNos`|`body`|`string`|否|关联的允许停放车辆|
|`++alarmPlateNos`|`body`|`string`|否|关联的不允许停放车辆|
|`++spaceType`|`body`|`string`|否|车位类型，普通车位：Normal Parking Space，|

补充说明:
- 1：停车
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- 固定车位：Registered Parking Space，
- 特殊车位：Dedicated Parking Space，
- 充电车位：Charging Parking Space，
- 行动不便人员车位：Accessible Parking Space
