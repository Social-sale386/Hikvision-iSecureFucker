# 通过视频桩查询车位锁状态

> 接口说明
> 通过视频桩查询车位锁状态

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v1/parkingSpace/getParkingLock`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parkingSpace/getParkingLock
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`spaceNo`|`body`|`string`|是|车位号|
|`floorSyscode`|`body`|`string`|是|楼层唯一标识码，可从查询车位信息接口获取返回参数floorSyscode|

## 请求参数示例
```json
{
  "spaceNo": "B120",
  "floorSyscode": "1231231"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|返回码，0 – 成功，其他- 失败|
|`msg`|`body`|`string`|是|返回描述|
|`data`|`body`|`boolean`|否|车位锁状态，true：开启，false：关闭|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": true
}
```
