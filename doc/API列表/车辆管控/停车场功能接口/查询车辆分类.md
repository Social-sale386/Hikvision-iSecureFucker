# 查询车辆分类

> 接口说明
> 简述：常用车辆分类信息有固定车、临时车、预约车、特殊车、黑名单，以及各种车辆群组。
> 支持：支持获取系统所有车辆分类信息，用作相关业务逻辑处理和展示。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/pms/v1/car/category/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/car/category/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`regionIndexCode`|`body`|`string`|否|区域IndexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|

## 请求参数示例
```json
{
  "regionIndexCode": "root000000"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object[]`|否|返回数据|
|`+categoryCode`|`body`|`string`|否|车辆分类标识|
|`+categoryName`|`body`|`string`|否|车辆分类名称|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": [
    {
      "categoryCode": "10",
      "categoryName": "固定车"
    }
  ]
}
```
