# 车辆充值V2

> 接口说明
> 简述：车辆添加后，有临时车、固定车之分，充值包期后是固定车，未包期或包期过期的是临时车，车辆出场需要进行收费。
> 支持：支持通过车牌号进行特定停车场、车道的包期充值。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V2.1.0及以上版本|
|接口地址|`/api/pms/v2/car/charge`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v2/car/charge
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|是|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`plateNo`|`body`|`string`|是|车牌号码|
|`fee`|`body`|`string`|否|包期费用，单位:元，2位小数|
|`startTime`|`body`|`string`|是|包期开始时间|
|`endTime`|`body`|`string`|是|包期结束时间|
|`lanes`|`body`|`string`|否|车道编号，多个用英文逗号隔开；不传默认充值所有车道|
|`plateColor`|`body`|`integer`|否|车牌颜色，不传默认蓝色|

补充说明:
- 时间格式：yyyy-MM-dd，如：2018-07-26
- 时间格式：yyyy-MM-dd，如：2018-07-27

## 请求参数示例
```json
{
  "parkSyscode": "gb45h4h53h4g245t45y54t45",
  "plateNo": "浙A12345",
  "fee": "30",
  "startTime": "2018-07-26",
  "endTime": "2018-07-27",
  "lanes": "13123,12324ssss",
  "plateColor": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|是|请求结果码|
|`msg`|`body`|`string`|是|提示信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success"
}
```
