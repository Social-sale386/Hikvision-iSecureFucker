# 查询车辆缴费记录

> 接口说明
> 简述：提供对接方通过接口查询平台车辆缴费记录。
> 支持：支持通过车牌号、卡号、停车库、缴费方式、车辆类型、收费员及时间段查询车辆缴费记录信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v1/charge_bill/record/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/charge_bill/record/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|是|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`entranceSyscode`|`body`|`string`|否|出入口唯一标识，可从获取出入口列表接口获取返回参数entranceIndexCode|
|`plateNo`|`body`|`string`|否|车牌号|
|`cardNo`|`body`|`string`|否|卡号|
|`chargeSource`|`body`|`integer`|否|收费来源，|
|`chargeType`|`body`|`integer`|否|收费方式，|
|`vehicleType`|`body`|`double`|否|车辆类型，-1：不区分类型|
|`startTime`|`body`|`string`|否|查询开始时间，ISO8601格式：|
|`endTime`|`body`|`string`|否|查询结束时间，ISO8601格式：|
|`operator`|`body`|`string`|否|收费员名称|
|`pageNo`|`body`|`integer`|是|目标页码，范围 ( 0 , ~ ) ，若是范围内数字超过实际最大页码值，返回最后一页记录|
|`pageSize`|`body`|`integer`|是|每页记录数，范围 ( 0 , 1000 ]|
