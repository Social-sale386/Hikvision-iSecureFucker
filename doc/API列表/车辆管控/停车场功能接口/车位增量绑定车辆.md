# 车位增量绑定车辆

> 接口说明
> 车位增量绑定车辆信息，传参的车辆做增量绑定，不会覆盖已有的绑定车辆

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/parkingSpace/carBind/batch`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parkingSpace/carBind/batch
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`spaceNos`|`body`|`string`|是|车位号集合（以英文逗号‘,’分割；车位号不唯一时需传floorSyscode），从查询车位信息接口 获取返回参数spaceNo|
|`floorSyscode`|`body`|`string`|否|楼层唯一标识|
|`plateNos`|`body`|`string`|是|车牌号集合（以英文逗号‘,’分割）|
|`bindType`|`body`|`integer`|否|绑定类型（默认0） 0:允许停放，1:禁止停放|

## 请求参数示例
```json
{
  "spaceNos": "12,432,63",
  "floorSyscode": "hddgfgsd34de5gdg34ws5345y",
  "plateNos": "浙M12JG3,浙AFU432,皖N63886",
  "bindType": 0
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": null
}
```
