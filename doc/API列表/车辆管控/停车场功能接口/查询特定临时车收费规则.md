# 查询特定临时车收费规则

> 接口说明
> 收费规则用于停车场内车辆出场时缴费、场内中央缴费等场景，是获取缴费账单费用信息的依据

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/pms/v1/chargeRule`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/chargeRule
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`chargeRuleSyscode`|`body`|`string`|是|收费规则唯一标识（最大长度64）|

补充说明:
- 来源于账单返回的收费规则唯一标识

## 请求参数示例
```json
{
  "chargeRuleSyscode": "g4wh45y245y45y54ty"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据-单个收费规则信息|
|`+chargeRuleSyscode`|`body`|`string`|否|收费规则唯一标识码|
|`+chargeRuleType`|`body`|`number`|否|收费规则类型 0:按单位时段收费 1:按总计时长收费 2:按次收费 3:日夜组合收费 4:按停车时间收费|
|`+chargeRuleDetail`|`body`|`string`|否|收费规则详情，Json字符串|
|`+dayLimitFee`|`body`|`string`|否|按天封顶费用（最大长度8） 单位：元|
|`+dayLimitFlag`|`body`|`number`|否|按天封顶开关 一天内多次进出总费用封顶 0-不启用， 1-启用|
|`+parkSyscode`|`body`|`string`|否|停车库唯一标识码|
|`+parkName`|`body`|`string`|否|停车库名称|
|`+vehicleType`|`body`|`number`|否|缴费车辆类型 使用该收费规则缴费的车辆类型 -1：不区分车辆类型 0：其他车； 1：小型车； 2：大型车； 3：摩托车|
|`+chargeRuleName`|`body`|`string`|否|收费规则名称|
