# 查询车辆充值退款记录

> 接口说明
> 简述：提供对接方通过接口查询平台车辆包期充值、退款记录。
> 支持：支持通过车牌号、卡号、交易类型、交易方式、停车库、收费员及时间段进行车辆包期充值退款记录查询。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v1/car_charge/record/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/car_charge/record/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`plateNo`|`body`|`string`|否|车牌号|
|`cardNo`|`body`|`string`|否|卡号|
|`tradeType`|`body`|`integer`|否|交易类型，1：包期充值|
|`tradeWay`|`body`|`integer`|否|交易方式，1：现金，|
|`startTime`|`body`|`string`|否|查询开始时间，ISO8601格式：|
|`endTime`|`body`|`string`|否|查询结束时间，ISO8601格式：|
|`operator`|`body`|`string`|否|操作员名称|
|`pageNo`|`body`|`integer`|是|目标页码，范围 ( 0 , ~ ) ，若是范围内数字超过实际最大页码值，返回最后一页记录|
|`pageSize`|`body`|`integer`|是|每页记录数，范围 ( 0 , 1000 ]|

补充说明:
- 2：退款
- 2：储值账户，
- 3：第三方充值，
- 4：支付宝，
- 5：微信
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00
- yyyy-MM-ddTHH:mm:ss+当前时区，例如北京时间：
- 2018-07-26T15:00:00+08:00

## 请求参数示例
```json
{
  "parkSyscode": "dfasgqegbq34g3q4g34g",
  "plateNo": "浙A12345",
  "cardNo": "22335544",

  "tradeWay": 1,

  "endTime": "2018-07-26T15:00:00+08:00",

  "pageSize": 15

```

|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+list`|`body`|`object[]`|否|车辆充值退款记录对象列表|
