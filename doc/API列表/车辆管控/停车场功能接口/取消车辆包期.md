# 取消车辆包期

> 接口说明
> 简述：车辆取消包期后变为临时车，可以取消某个停车库的包期，也可以取消平台所有停车库下的包期。
> 支持：支持通过车牌号、停车库编号取消包期；停车库编号可为空，为空时取消平台所有包期。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/pms/v1/car/charge/deletion`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/car/charge/deletion
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parkSyscode`|`body`|`string`|否|停车库唯一标识，可从获取停车库列表接口获取返回参数parkIndexCode|
|`plateColor`|`body`|`integer`|否|车牌颜色，车辆唯一标识是车牌+车牌颜色时必传|
|`plateNo`|`body`|`string`|是|车牌号码|

## 请求参数示例
```json
{
  "plateNo": "浙AJ2681",
  "parkSyscode": "a324aea17ef34d978a65e596a6598be3",
  "plateColor": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`string`|否|返回数据|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
