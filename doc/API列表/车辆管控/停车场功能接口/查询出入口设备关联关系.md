# 查询出入口设备关联关系

> 接口说明
> 简述：用于查询设备和停车场、车道的关联关系。
> 支持：按停车场、出入口、车道获取包含下级节点的所有设备。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pms/v1/park/deviceRelation/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/park/deviceRelation/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`syscode`|`body`|`string`|是|结点唯一标志，查询停车库节点信息接口获取返回参数indexCode|
|`nodeType`|`body`|`integer`|是|节点类型（1：停车场，2：出入口，4：车道，16：子库停车场）|
|`subNode`|`body`|`boolean`|否|是否包含下级节点，不填默认false|
|`deviceType`|`body`|`string`|否|设备资源类型，|

补充说明:
- terminal：岗亭缴费终端；
- controlDevice：出入口控制设备；
- displayDevice：出入口显示设备；
- cardReader：读卡设备

## 请求参数示例
```json
{
  "syscode": "hddgfgsd34de5gdg34ws5345y",
  "nodeType": 1,
  "subNode": false,
  "deviceType": "terminal"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object[]`|否|返回数据|
|`+deviceSyscode`|`body`|`string`|否|设备编号|
|`+parkSyscode`|`body`|`string`|否|停车场编号|
|`+roadwaySyscode`|`body`|`string`|否|车道编号|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Success",
  "data": [
    {
      "deviceSyscode": "123232323113",
      "parkSyscode": "435345435345",
      "roadwaySyscode": "56756756756767"
    }
  ]
}
```
