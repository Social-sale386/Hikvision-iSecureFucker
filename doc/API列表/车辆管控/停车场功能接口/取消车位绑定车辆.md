# 取消车位绑定车辆

> 接口说明
> 简述：车位绑定车辆后，因业务需要取消之前绑定的信息，以恢复车位未绑定的状态。
> 支持：支持通过车牌号、车位号进行取消关联；支持批量取消。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pms/v1/parking_space/car_bind/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pms/v1/parking_space/car_bind/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`spaceNos`|`body`|`string`|是|取消绑定的车位集合，|
|`floorSyscode`|`body`|`string`|否|楼层唯一标识码，可从查询车位信息接口获取返回参数floorSyscode|
|`plateNos`|`body`|`string`|是|取消绑定的车牌集合，多个用‘,’分隔|

补充说明:
- 多个用‘,’分隔，可从查询车位信息接口获取返回参数spaceNo

## 请求参数示例
```json
{
  "spaceNos": "12,23,43,45,125",
  "floorSyscode": "hddgfgsd34de5gdg34ws5345y",
  "plateNos": "浙A12345,京A00001,沪C32KG9"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.3.1 停车场应用错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": "null"
}
```
