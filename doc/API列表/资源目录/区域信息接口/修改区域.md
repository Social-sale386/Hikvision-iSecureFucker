# 修改区域

> 接口说明
> 根据区域唯一标志修改区域信息

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/region/single/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/region/single/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`regionIndexCode`|`body`|`string`|是|区域唯一标志，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`parentIndexCode`|`body`|`string`|否|父区域唯一标识码，修改时以regionIndexCode和此字段标志出region，同时用于同级region重名校验，不能修改region的parentIndexCode值|
|`regionName`|`body`|`string`|否|区域名称|
|`regionCode`|`body`|`string`|否|外码编码，regionType为12时有效，regionType为10时忽略该值|
|`regionType`|`body`|`integer`|否|区域类型，10-普通区域，12-楼栋单元|
|`description`|`body`|`string`|否|区域描述信息|

## 请求参数示例
```json
{
  "regionIndexCode": "af8vya00hrch234hc25y40c4u",
  "parentIndexCode": "7618af38-b278-411c-a74b-41e1e4f8dd1c",
  "regionName": "区域1",
  "regionCode": "3q8c8y820y",
  "regionType": 10,
  "description": "区域描述"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|返回结果，成功则为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": ""
}
```
