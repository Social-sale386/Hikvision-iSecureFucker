# 获取根区域信息

> 接口说明
> 获取根区域信息接口

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/regions/root`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/regions/root
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`treeCode`|`body`|`string`|否|保留字段，暂不启用，不传不影响返回结果|

## 请求参数示例
```json
{
  "treeCode": "0"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|区域信息结构体|
|`+indexCode`|`body`|`string`|否|区域唯一标识码|
|`+name`|`body`|`string`|否|区域名称|
|`+parentIndexCode`|`body`|`string`|否|父区域唯一标识码|
|`+treeCode`|`body`|`string`|否|树编号|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "indexCode": "083b2031c1CC4f368f015fe2562e0012",
    "name": "全国",
    "parentIndexCode": "-1",
    "treeCode": "0"
  }
}
```
