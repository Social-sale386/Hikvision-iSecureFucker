# 增量获取资源数据

> 接口说明
> 根据资源类型、时间段增量获取资源，包含已删除数据。其中开始日期与结束日期的时间差必须在1-48小时内。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/resource/v1/resource/timeRange`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/resource/timeRange
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`startTime`|`body`|`string`|是|开始日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`endTime`|`body`|`string`|否|截止日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`pageNo`|`body`|`integer`|是|页码,pageNo≥1|
|`pageSize`|`body`|`integer`|是|页面大小,0<pageSize≤1000|
|`resourceType`|`body`|`string`|是|资源类型，详见[附录A.2 资源类型]@[软件产品-综合安防管理平台-附录-附录A%20数据字典#附录A.2 资源类型|

## 请求参数示例
```json
{
  "startTime": "2019-08-06T14:01:17.839+0800",
  "endTime": "2019-08-06T16:01:17.839+0800",
  "pageNo": 1,
  "pageSize": 1000,
  "resourceType": "camera"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|记录总数|
|`+list`|`body`|`object[]`|否|资源数据列表，根据resourceType返回对应的资源对象，详见[附录C.6 资源目录服务](/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录C%20对象信息-附录C.6 资源目录服务)（1、编码设备：EncodeDeviceDTO； 2、监控点：CameraDTO； 3、IO通道： IoChannelDTO； 4、门禁设备：AcsDeviceDTO； 5、门禁点：DoorDTO； 6、报警主机：IasDeviceDTO； 7、子系统通道：SubSysDTO； 8、防区通道：DefenceDTO； 9、可视对讲：VisDeviceDTO； 10、梯控-控制器：EcsDeviceDTO； 11、梯控-读卡器：ReaderDTO； 12、梯控-楼层：FloorDTO；）|
|`++belongIndexCode`|`body`|`string`|否|所属服务编号|
|`++capability`|`body`|`string`|否|能力集，详见附录A.44 设备能力集|
|`++deviceKey`|`body`|`string`|否|设备驱动|
|`++deviceType`|`body`|`string`|否|设备系列|
|`++devSerialNum`|`body`|`string`|否|设备序列号|
|`++deviceCode`|`body`|`string`|否|主动设备编号|
|`++indexCode`|`body`|`string`|否|资源唯一编码|
|`++ip`|`body`|`string`|否|设备ip|
|`++manufacturer`|`body`|`string`|否|厂商|
|`++name`|`body`|`string`|否|资源名称|
|`++netZoneId`|`body`|`string`|否|网域|
|`++port`|`body`|`string`|否|端口|
|`++regionIndexCode`|`body`|`string`|否|所属区域|
|`++resourceType`|`body`|`string`|否|资源类型|
|`++treatyType`|`body`|`string`|否|接入协议|
|`++createTime`|`body`|`string`|否|创建时间|
|`++updateTime`|`body`|`string`|否|修改时间|
|`++status`|`body`|`number`|否|小于0则代表资源已被删除|
