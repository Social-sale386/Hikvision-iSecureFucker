# 查询资源列表v2

> 接口说明
> 根据条件查询目录下有权限的资源列表。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/irds/v2/resource/resourcesByParams`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/irds/v2/resource/resourcesByParams
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`name`|`body`|`string`|否|名称，模糊搜索，最大长度32，若包含中文，最大长度指不超过按照指定编码的字节长度，即getBytes(“utf-8”).length|
|`regionIndexCodes`|`body`|`string[]`|否|区域编号,可以为空;|
|`isSubRegion`|`body`|`boolean`|否|true时，搜索regionIndexCodes及其子孙区域的资源；|
|`resourceType`|`body`|`string`|是|资源类型，详见附录A 数据字典|
|`pageNo`|`body`|`integer`|是|当前页码|
|`pageSize`|`body`|`integer`|是|分页大小|
|`authCodes`|`body`|`string[]`|否|权限码集合，详见附录A.3 资源类型/资源权限码|
|`capabilitySet`|`body`|`array`|否|设备能力集(含设备上的智能能力)，详见附录A.44 设备能力集|
|`orderBy`|`body`|`string`|否|排序字段,注意：排序字段必须是查询条件，否则返回参数错误|
|`orderType`|`body`|`string`|否|降序升序,降序：desc 升序：asc|
|`expressions`|`body`|`object[]`|否|查询表达式|
|`+key`|`Body`|`string`|是|资源属性名，key传updateTime，operator传between可以查询特定时间段更新的数据，考虑到校时和夏令时，建议值查询过去一天的数据变更|
|`+operator`|`Body`|`number`|是|操作运算符， 0 ：=， 1 ：>=， 2 ：<=， 3 ：in， 4 ：not in， 5 ：between， 6 ：like， 7 ：pre like， 8 ：suffix like|
|`+values`|`Body`|`string[]`|是|资源属性值，=、>=、<=、like、values数组长度只能是1； in、not in，values数组长度大于1，最大不超时20； in_array用于查询key值有多个value的情况，例如行车监控添加的设备类型为encodeDevice、encodeDeviceMss两个类型，使用encodeDevice或者encodeDeviceMss都可以查询到； between只能用于整形、日期 ； like只能用于字符串。|

补充说明:
- String[]，支持根据区域批量查询；
- 区域编号个数
- <=1000个；
- 单个长度<=64Byte；，可从查询区域列表v2接口获取返回参数indexCode
- false时，只搜索 regionIndexCodes的资源

## 请求参数示例
```json
{
  "name": "门",
  "regionIndexCodes": [
    "8fyc8qw8280y0y43"
  ],
  "isSubRegion": true,
  "resourceType": "door",
  "pageNo": 1,
  "pageSize": 1,
  "authCodes": [
    "view"
  ],
  "orderBy": "name",
  "orderType": "desc",
  "expressions": [
    {
      "key": "indexCode",
      "operator": 0,
      "values": [
        "ayd8y80y1y09078979882ye01y2e8"
      ]
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|分页大小|
|`+list`|`body`|`object[]`|否|资源数据列表，根据resourceType返回对应的资源对象，本例返回门禁点对象DoorDTO；|
|`++indexCode`|`body`|`string`|否|门禁点唯一标识|
|`++name`|`body`|`string`|否|门禁点名称|
|`++resourceType`|`body`|`string`|否|资源类型，详见附录A.3 资源类型/资源权限码|
|`++doorNo`|`body`|`number`|否|门禁点编号|
|`++description`|`body`|`string`|否|描述|
|`++parentIndexCodes`|`body`|`string`|否|父级资源编号|
|`++regionIndexCode`|`body`|`string`|否|所属区域唯一标识|
|`++regionPath`|`body`|`string`|否|所属区域目录，由唯一标示组成，最大10级，格式： @根节点@子区域1@子区域2@|
|`++channelType`|`body`|`string`|否|通道类型|
|`++channelNo`|`body`|`string`|否|通道号|
|`++installLocation`|`body`|`string`|否|安装位置|
|`++capabilitySet`|`body`|`string`|否|设备能力集，详见附录A.44 设备能力集|
|`++controlOneId`|`body`|`string`|否|一级控制器id|
|`++controlTwoId`|`body`|`string`|否|二级控制器id|
|`++readerInId`|`body`|`string`|否|读卡器1|
|`++readerOutId`|`body`|`string`|否|读卡器2|
|`++comId`|`body`|`string`|否|组件标志|
|`++createTime`|`body`|`string`|否|创建时间|
|`++updateTime`|`body`|`string`|否|更新时间|

补充说明:
- 其他资源对象详见[附录C.6 资源目录服务](/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录C%20对象信息-附录C.6 资源目录服务)（1、编码设备：EncodeDeviceDTO；
- 2、监控点：CameraDTO；
- 3、IO通道：
- IoChannelDTO；
- 4、门禁设备：AcsDeviceDTO；
- 5、门禁点：DoorDTO；
- 6、报警主机：IasDeviceDTO；
- 7、子系统通道：SubSysDTO；
- 8、防区通道：DefenceDTO；
- 9、可视对讲：VisDeviceDTO；
- 10、梯控-控制器：EcsDeviceDTO；
- 11、梯控-读卡器：ReaderDTO；
- 12、梯控-楼层：FloorDTO；）
