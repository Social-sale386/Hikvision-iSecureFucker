# 获取资源的历史在线记录

> 接口说明
> 获取资源的历史在线记录接口

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/nms/v1/online/history_status`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/nms/v1/online/history_status
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCode`|`body`|`string`|是|资源编码，从获取资源列表v2接口获取返回报文中的indexCode字段|
|`startDate`|`body`|`string`|否|开始日期，yyyy-MM-dd格式|
|`endDate`|`body`|`string`|否|结束日期，yyyy-MM-dd格式|
|`pageNo`|`body`|`integer`|否|页码,从1开始|
|`pageSize`|`body`|`integer`|否|页大小,正整数，最大值1000，默认值500|
|`resourceType`|`body`|`string`|是|资源类型,参考附录A.64 网管资源类型|

## 请求参数示例
```json
{
  "indexCode": "ce91c758-5af4-4539-845a-1b603746c55",
  "startDate": "2018-01-02",
  "endDate": "2019-01-03",
  "pageNo": 1,
  "pageSize": 20,
  "resourceType": "encodeDevice"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总条数|
|`+list`|`body`|`object[]`|否|在线记录列表|
|`++collectTime`|`body`|`string`|否|状态采集时间，时间格式为ISO格式|
|`++online`|`body`|`number`|否|监控点在线状态，0-离线，1-在线|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "Operation succeeded",
  "data": {
    "total": 1,
    "list": [
      {
        "collectTime": "2018-12-28T10:21:40.000+08:00",
        "online": 1
      }
    ]
  }
}
```
