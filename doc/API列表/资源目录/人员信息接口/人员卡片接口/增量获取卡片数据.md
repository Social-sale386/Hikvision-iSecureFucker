# 增量获取卡片数据

> 接口说明
> 根据查询条件查询卡片列表信息，主要根据时间段分页获取卡片信息，包含已删除数据。其中开始日期与结束日期的时间差必须在1-48小时内。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/resource/v1/card/timeRange`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/card/timeRange
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`startTime`|`body`|`string`|是|开始日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`endTime`|`body`|`string`|否|截止日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`pageNo`|`body`|`integer`|是|页码,pageNo≥1|
|`pageSize`|`body`|`integer`|是|页面大小,0<pageSize≤1000|

## 请求参数示例
```json
{
  "startTime": "2019-08-06T14:01:17.839+0800",
  "endTime": "2019-08-06T16:01:17.839+0800",
  "pageNo": 1,
  "pageSize": 1000
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码,0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|卡片信息结构体|
|`+total`|`body`|`number`|否|记录总数|
|`+list`|`body`|`object[]`|否|返回卡片数据列表|
|`++cardId`|`body`|`string`|否|卡片ID|
|`++cardNo`|`body`|`string`|否|卡号|
|`++personIndexCode`|`body`|`string`|否|持卡人员indexCode|
|`++useStatus`|`body`|`number`|否|使用状态标记，1：正常 2：挂失|
|`++startDate`|`body`|`string`|否|生效日期，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00 表示北京时间2017年7月26日21时30分08秒|
|`++endDate`|`body`|`string`|否|失效日期，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00 表示北京时间2017年7月26日21时30分08秒|
|`++lossDate`|`body`|`string`|否|挂失时间，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00 表示北京时间2017年7月26日21时30分08秒|
|`++unlossDate`|`body`|`string`|否|解除挂失时间，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00 表示北京时间2017年7月26日21时30分08秒|
|`++description`|`body`|`string`|否|描述|
|`++createTime`|`body`|`string`|否|创建时间，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00 表示北京时间2017年7月26日21时30分08秒|
|`++updateTime`|`body`|`string`|否|创建时间，要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00 表示北京时间2017年7月26日21时30分08秒|
|`++cardType`|`body`|`number`|否|卡类型|
|`++orgIndexCode`|`body`|`string`|否|所属组织|
|`++isBindFinger`|`body`|`number`|否|是否绑定指纹|
|`++isBindFace`|`body`|`number`|否|是否绑定人脸|
|`++status`|`body`|`number`|否|小于0则代表资源已被删除|
