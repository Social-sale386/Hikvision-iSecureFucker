# 批量挂失

> 接口说明
> 用于卡片批量挂失，解挂卡数量不能超过200。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/cis/v1/card/batch/loss`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/cis/v1/card/batch/loss
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`cardList`|`body`|`object[]`|是|人卡关联结合数组(批量挂失或解挂数量不能超过200个)|
|`+cardNumber`|`Body`|`string`|是|卡号|
|`+personId`|`Body`|`string`|否|人员唯一标示， 根据获取人员列表v2接口获取返回参数personId|

## 请求参数示例
```json
{
  "cardList": [
    {
      "cardNumber": "1111111111",
      "personId": "ab8fbf2b7d1c489a80e1676850a318a8"
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码,0 – 成功，其他- 失败，参考附录E.other.3 卡片错误码|
|`msg`|`body`|`string`|否|返回描述（记录接口执行情况说明信息）|
|`data`|`body`|`string`|否|成功则为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": null
}
```
