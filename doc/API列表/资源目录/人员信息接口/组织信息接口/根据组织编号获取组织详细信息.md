# 根据组织编号获取组织详细信息

> 接口说明
> 根据组织编号orgIndexCode列表获取指定的组织信息，如存在多组织则返回总条数及多组织信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/resource/v1/org/orgIndexCodes/orgInfo`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/org/orgIndexCodes/orgInfo
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`orgIndexCodes`|`body`|`string[]`|是|组织编号数组,从获取组织列表接口获取返回参数orgIndexCode|

## 请求参数示例
```json
{
  "orgIndexCodes": [
    "root000000"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0– 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|组织信息结构体|
|`+total`|`body`|`number`|否|记录总数|
|`+list`|`body`|`object[]`|否|返回数据列表|
|`++orgIndexCode`|`body`|`string`|否|组织唯一标识码|
|`++organizationCode`|`body`|`string`|否|组织编码|
|`++orgName`|`body`|`string`|否|组织名称|
|`++orgPath`|`body`|`string`|否|组织目录|
|`++parentOrgIndexCode`|`body`|`string`|否|父组织唯一标识|
|`++available`|`body`|`boolean`|否|用于标识组织节点是否有权限操作|
|`++leaf`|`body`|`boolean`|否|标识组织节点是否叶子节点|
|`++sort`|`body`|`number`|否|显示顺序|
|`++createTime`|`body`|`string`|否|创建时间,要求遵守ISO8601标准，如2018-07-26T21:30:08.000+08:00 表示北京时间2017年7月26日21时30分08秒|
|`++updateTime`|`body`|`string`|否|修改时间,要求遵守ISO8601标准，如2018-07-26T21:30:08.000+08:00 表示北京时间2017年7月26日21时30分08秒|
