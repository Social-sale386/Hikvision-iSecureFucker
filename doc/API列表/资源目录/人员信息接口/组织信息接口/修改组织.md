# 修改组织

> 接口说明
> 根据组织编号修改组织信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/org/single/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/org/single/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`orgIndexCode`|`body`|`string`|是|组织标识，必须保证已存在,从获取组织列表接口获取返回参数orgIndexCode|
|`orgName`|`body`|`string`|否|组织名称，1~32个字符；不能包含 ’ / \ : * ? " < >|

## 请求参数示例
```json
{
  "orgIndexCode": "1rwad89d-0ce6-4826-9146-6b71f037d81e",
  "orgName": "组织test1"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0– 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`string`|否|返回结果，成功则data为空|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": ""
}
```
