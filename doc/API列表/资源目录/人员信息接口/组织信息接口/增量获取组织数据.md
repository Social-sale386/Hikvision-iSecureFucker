# 增量获取组织数据

> 接口说明
> 根据查询条件查询组织列表信息，主要根据时间段分页获取组织数据，包含已删除数据。其中开始日期与结束日期的时间差必须在1-48小时内。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/resource/v1/org/timeRange`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/org/timeRange
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`startTime`|`body`|`string`|是|开始日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`endTime`|`body`|`string`|否|截止日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`pageNo`|`body`|`integer`|是|页码,pageNo≥1|
|`pageSize`|`body`|`integer`|是|页面大小,0<pageSize≤1000|

## 请求参数示例
```json
{
  "startTime": "2019-08-06T14:01:17.839+0800",
  "endTime": "2019-08-06T16:01:17.839+0800",
  "pageNo": 1,
  "pageSize": 1000
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0– 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|组织信息结构体|
|`+total`|`body`|`number`|否|结果总数|
|`+list`|`body`|`object[]`|否|组织信息列表|
|`++orgIndexCode`|`body`|`string`|否|组织唯一标识码|
|`++organizationCode`|`body`|`string`|否|组织编码|
|`++orgName`|`body`|`string`|否|组织名称|
|`++orgPath`|`body`|`string`|否|组织目录|
|`++parentOrgIndexCode`|`body`|`string`|否|父组织唯一标识|
|`++sort`|`body`|`number`|否|显示顺序|
|`++createTime`|`body`|`string`|否|创建时间|
|`++updateTime`|`body`|`string`|否|修改时间|
|`++status`|`body`|`number`|否|状态小于0则代表资源已被删除|
|`++available`|`body`|`boolean`|否|用于标识组织节点是否有权限操作|
|`++leaf`|`body`|`boolean`|否|标识组织节点是否叶子节点|
