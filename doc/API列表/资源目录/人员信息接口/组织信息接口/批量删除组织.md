# 批量删除组织

> 接口说明
> 仅支持删除无子结点且组织下不存在人员的组织。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/org/batch/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/org/batch/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCodes`|`body`|`string[]`|是|待删除的组织indexCode列表，单次提交上限为1000条|

## 请求参数示例
```json
{
  "indexCodes": [
    "b8d059a7-608a-4800-9b84-4eb8332978d2"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object[]`|否|若执行结果为部分删除成功，则返回删除失败的组织标志和错误信息，若全部成功，则该字段为空|
|`+orgIndexCode`|`body`|`string`|否|组织标志|
|`+code`|`body`|`string`|否|错误码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`+msg`|`body`|`string`|否|错误信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": [
    {
      "orgIndexCode": "8f0c821038cy090c9y3",
      "code": "-1",
      "msg": "error"
    }
  ]
}
```
