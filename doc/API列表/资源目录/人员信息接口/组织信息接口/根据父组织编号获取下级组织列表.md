# 根据父组织编号获取下级组织列表

> 接口说明
> 根据父组织编号获取下级组织列表，主要用于逐层获取父组织的下级组织信息，返回结果分页展示。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/org/parentOrgIndexCode/subOrgList`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/org/parentOrgIndexCode/subOrgList
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`parentOrgIndexCode`|`body`|`string`|是|父组织编号,从获取组织列表接口获取返回参数orgIndexCode|
|`pageNo`|`body`|`integer`|是|当前页码|
|`pageSize`|`body`|`integer`|是|每页记录总数|

补充说明:
- 当前页码需大于0（pageNo>0）
- 每页记录展示的数目应大于0，小于等于1000（0<pageSize<=1000）

## 请求参数示例
```json
{
  "parentOrgIndexCode": "4CC7c89d-0ce6-4826-9146-6b71f037d81e",
  "pageNo": 1,
  "pageSize": 1000
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|组织信息结构体|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|实际每页记录总数|
|`+list`|`body`|`object[]`|否|组织对象列表|
|`++orgIndexCode`|`body`|`string`|否|组织唯一标识码|
|`++orgNo`|`body`|`string`|否|组织编号|
|`++orgName`|`body`|`string`|否|组织名称，如默认部门|
|`++orgPath`|`body`|`string`|否|组织目录，例如：默认部门/综合安防/开发小组|
|`++parentOrgIndexCode`|`body`|`string`|否|父组织唯一标识码|
|`++parentOrgName`|`body`|`string`|否|父组织名称|
|`++updateTime`|`body`|`string`|否|更新时间 要求遵守ISO8601标准，如2018-07-26T21:30:08.000+08:00 表示北京时间2017年7月26日21时30分08秒|

```json
        "parentOrgName": "默认部门",

      }
    ]
  }
}
```
