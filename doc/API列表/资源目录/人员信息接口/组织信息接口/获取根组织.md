# 获取根组织

> 接口说明
> 获取根组织接口用来获取组织的根节点。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/org/rootOrg`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/org/rootOrg
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`object`|否|根组织信息结构体|
|`+orgIndexCode`|`body`|`string`|否|组织唯一标识码|
|`+orgNo`|`body`|`string`|否|组织编号|
|`+orgName`|`body`|`string`|否|当前组织的名称，如默认部门|
|`+orgPath`|`body`|`string`|否|组织目录|
|`+parentOrgIndexCode`|`body`|`string`|否|父组织唯一标识码|
|`+parentOrgName`|`body`|`string`|否|父组织名称|
|`+updateTime`|`body`|`string`|否|更新时间 要求遵守ISO8601标准，如2018-07-26T21:30:08+08:00 表示北京时间2017年7月26日21时30分08秒|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "ok",
  "data": {
    "orgIndexCode": "4CC7c89d-0ce6-4826-9146-6b71f037d81e",
    "orgNo": "0000",
    "orgName": "默认部门",
    "orgPath": "默认部门",
    "parentOrgIndexCode": "0",
    "parentOrgName": "null",
    "updateTime": "null"
  }
}
```
