# 修改人脸

> 接口说明
> 根据人脸Id修改人脸。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/face/single/update`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/face/single/update
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`faceId`|`body`|`string`|是|人脸Id，添加人脸接口返回报文中的faceId字段，或获取人员列表v2接口返回报文中的personPhotoIndexCode字段|
|`faceData`|`body`|`string`|是|人脸图片base64编码后的字符|

## 请求参数示例
```json
{
  "faceId": "b8d059a7-608a-4800-9b84-4eb8332978ec",
  "faceData": "/9j/4AAQSkZJRgABAQEAAAAAAAD/4QBCRXhpZgAATU..."
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+faceId`|`body`|`string`|否|人脸Id|
|`+faceUrl`|`body`|`string`|否|人脸图片Url|
|`+personId`|`body`|`string`|否|人员ID|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "faceId": "fd1004e3-4b99-4a9f-bce5-3d00344c507e",
    "faceUrl": "/pic?=d5=ia95z8452s741-c83428--de366289d69f9i7b7*=ids1*=idp2*=pd*m1i1t=5e5458991-88id06*e057fd2",
    "personId": "be7be3b49e4740c3bef45fc7cdd9fb40"
  }
}
```
