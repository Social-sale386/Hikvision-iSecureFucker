# 批量删除人员

> 接口说明
> 根据编号删除人员，人员删除是软删除，可以在安保基础数据查询已删除人员。支持批量删除人员。删除人员将会同时删除人员关联的指纹和人脸信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/person/batch/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/person/batch/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`personIds`|`body`|`string[]`|是|人员Id，单次操作上限为1000条。可从获取人员列表v2接口获取返回值personId|

## 请求参数示例
```json
{
  "personIds": [
    "d7228950bf29490484bc845dff756b95"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object[]`|否|若执行结果为部分删除成功，则返回删除失败的人员标志和错误原因，若全部成功，则该字段为空。|
|`+personId`|`body`|`string`|否|人员标志|
|`+code`|`body`|`string`|否|错误码|
|`+msg`|`body`|`string`|否|错误信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": [
    {
      "personId": "12803709iafhych8yr01c9",
      "code": "-1",
      "msg": "error"
    }
  ]
}
```
