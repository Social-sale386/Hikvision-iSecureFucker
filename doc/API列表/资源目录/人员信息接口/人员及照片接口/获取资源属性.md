# 获取资源属性

> 接口说明
> 查询当前平台指定资源已定义的属性信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/resource/properties`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/resource/properties
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`resourceType`|`body`|`string`|是|资源类型，人员-person，区域-region，车辆-vehicle，组织-organization|

## 请求参数示例
```json
{
  "resourceType": "person"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回说明|
|`data`|`body`|`object[]`|否|资源已定义的属性|
|`+id`|`body`|`string`|否|属性id|
|`+name`|`body`|`string`|否|属性名称|
|`+required`|`body`|`number`|否|属性是否必输，0-非必输，1-必输|
|`+fieldType`|`body`|`string`|否|属性类型|
|`+desc`|`body`|`string`|否|属性描述|
|`+minLength`|`body`|`number`|否|属性最小长度|
|`+maxLength`|`body`|`number`|否|属性最大长度|
|`+valueType`|`body`|`string`|否|值类型|
|`+optionKey`|`body`|`number`|否|是否是备选主键|
|`+key`|`body`|`number`|否|是否是主键|
|`+regular`|`body`|`string`|否|属性正则表达式|
|`+unique`|`body`|`number`|否|是否唯一|
|`+ldapName`|`body`|`string`|否|ldap中属性名称|
|`+objectClass`|`body`|`string`|否|ldap中模型|
