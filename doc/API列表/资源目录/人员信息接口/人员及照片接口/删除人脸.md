# 删除人脸

> 接口说明
> 根据人脸Id删除人脸。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/resource/v1/face/single/delete`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/face/single/delete
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`faceId`|`body`|`string`|是|人脸Id，添加人脸接口返回报文中的faceId字段，或获取人员列表v2接口返回报文中的personPhotoIndexCode字段|

## 请求参数示例
```json
{
  "faceId": "fd1004e3-4b99-4a9f-bce5-3d00344c507"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`message`|`body`|`string`|否|返回信息|
|`data`|`body`|`string`|否|删除成功则为空|

## 返回参数示例
```json
{
  "code": "0",
  "message": "success",
  "data": ""
}
```
