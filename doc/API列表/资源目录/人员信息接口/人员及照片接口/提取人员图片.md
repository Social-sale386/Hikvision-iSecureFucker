# 提取人员图片

> 接口说明
> 根据获取人员信息的接口中拿到的图片URI和图片服务器唯一标识，通过该接口得到完整的URL，可以直接从图片服务下载图；
> 此接口返回重定向请求redirect：picUrl

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/resource/v1/person/picture`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/person/picture
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`picUri`|`body`|`string`|是|图片相对URI，获取人员列表v2接口返回报文中的picUri字段|
|`serverIndexCode`|`body`|`string`|是|图片服务器唯一标识，获取人员列表v2接口返回报文中的serverIndexCode字段|

## 请求参数示例
```json
{
  "picUri": "/pic?8dd685i46-e*9b649962225a--0a15b67eb3a78i7b5*=ids1*=idp2*=0d8t0pe*m5i11=4-3634fc91z20ds=4i32=",
  "serverIndexCode": "71d96061-9f20-4c46-b7c6-f75b7e5ca5e7"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|接口执行情况说明信息|
|`data`|`body`|`string`|否|返回数据|
