# 分页查询消防监测数据v2

> 接口说明
> 本接口提供分页查询设备监测历史数据功能，返回设备监测数据信息.

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/fpms/v2/minitor/data/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/fpms/v2/minitor/data/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`regionIndexCode`|`body`|`string`|是|所属区域indexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`includeDown`|`body`|`integer`|是|是否包含下级 (1-包含,0-不包含)|
|`resourceTypeCodes`|`body`|`string[]`|否|设备类型或传感器类型，参考附录A.55 消防设备类型，参考附录A.57 消防传感器大类型|
|`pageSize`|`body`|`integer`|否|每页记录数，每页最大数量1000,默认20|
|`pageNo`|`body`|`integer`|否|目标页码，默认从1开始，表示第一页|

## 请求参数示例
```json
{
  "regionIndexCode": "04c60702-f555-4887-9ae8-dc1186a7537a",
  "includeDown": 1,
  "resourceTypeCodes": [
    "20001"
  ],
  "pageSize": 20,
  "pageNo": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.2 消防错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageSize`|`body`|`number`|否|每页条数|
|`+total`|`body`|`number`|否|结果总数|
|`+pageNo`|`body`|`number`|否|页码|
|`+totalPage`|`body`|`number`|否|总页数|
|`+rows`|`body`|`object[]`|否|返回结果集|
|`++indexCode`|`body`|`string`|否|通用资源编码|
|`++resourceIndexCode`|`body`|`string`|否|设备或者传感器真实编号|
|`++name`|`body`|`string`|否|设备或者传感器名称|
|`++resourceType`|`body`|`number`|否|资源类型，1代表传感器，2代表设备|
|`++type`|`body`|`number`|否|传感器或者设备具体类型，参考附录A.55 消防设备类型，参考附录A.57 消防传感器大类型|
|`++updateTime`|`body`|`string`|否|更新时间，ISO8601格式|
|`++values`|`body`|`object[]`|否|监测数据集|
|`+++monitorTypeCode`|`body`|`string`|否|监测类型，参考附录A.70 消防监测类型中类型大类|
|`+++monitorTypeName`|`body`|`string`|否|监测名称|
|`+++monitorValue`|`body`|`string`|否|监测值|
|`+++monitorChannelId`|`body`|`string`|否|监测值所属通道Id|
|`+++monitorSpecificName`|`body`|`string`|否|监测小类名称|
|`+++monitorTypeKey`|`body`|`string`|否|监测小类key，，参考附录A.70 消防监测类型中类型小类key|
