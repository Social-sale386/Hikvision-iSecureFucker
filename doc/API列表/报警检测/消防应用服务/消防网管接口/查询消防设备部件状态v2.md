# 查询消防设备部件状态v2

> 接口说明
> 分页查询消防设备部件状态

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/fpnms/v2/device/status/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/fpnms/v2/device/status/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`includeSubNode`|`body`|`integer`|是|是否包含下级 (1-包含, 0-不包含)|
|`regionIndexCode`|`body`|`string`|是|所属区域indexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`deviceName`|`body`|`string`|否|设备名称|
|`deviceType`|`body`|`integer`|否|消防设备类型，详见附录A.55 消防设备类型|
|`online`|`body`|`integer`|否|在线状态 0-离线 1-在线 空-所有状态|
|`unitStatus`|`body`|`integer`|否|部件状态，0：未注册 1：正常 2: 修复中 3：异常状态（修复中状态目前没用到，未注册时设备没有在离线状态） 空-所有状态|
|`deviceIndexCode`|`body`|`string`|否|设备编号，根据查询消防设备列表接口获取返回报文中的devSerialNum字段|
|`pageNo`|`body`|`integer`|是|目标页码，默认从1开始，表示第一页|
|`pageSize`|`body`|`integer`|是|每页记录数，每页最大数量1000,默认20|

## 请求参数示例
```json
{
  "includeSubNode": 1,
  "regionIndexCode": "04c60702-f555-4887-9ae8-dc1186a7537a",
  "deviceName": "园区用传设备",
  "deviceType": 20001,
  "online": 1,
  "unitStatus": 1,
  "deviceIndexCode": "13212347-9365-dc1186a753",
  "pageNo": 1,
  "pageSize": 20
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.2 消防错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageSize`|`body`|`number`|否|每页条数|
|`+total`|`body`|`number`|否|结果总数|
|`+page`|`body`|`number`|否|页码|
|`+totalPage`|`body`|`number`|否|总页数|
|`+rows`|`body`|`object[]`|否|返回结果集|
|`++cn`|`body`|`string`|否|设备名称|
|`++regionIndexCode`|`body`|`string`|否|所属区域编号|
|`++indexCode`|`body`|`string`|否|设备编码|
|`++regionName`|`body`|`string`|否|区域目录名称，"/"分隔|
|`++deviceIndexCode`|`body`|`string`|否|设备编号|
|`++deviceType`|`body`|`number`|否|消防设备类型，详见附录A.55 消防设备类型|
|`++unitStatus`|`body`|`number`|否|部件状态，0：未注册 1：正常 2: 修复中 3：异常状态（修复中状态目前没用到，未注册时设备没有在离线状态） 空-所有状态|
|`++online`|`body`|`number`|否|在线状态，0：离线，1：在线|
|`++collectTime`|`body`|`string`|否|更新时间，IOS8601格式|
|`++faultType`|`body`|`string`|否|故障类型，","号分隔，详见[附录A.89 消防故障类型(/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录A%20数据字典#ca931162)|
|`++faultName`|`body`|`string`|否|故障类型名称，","号分隔|
|`++faultNum`|`body`|`string`|否|故障次数|
