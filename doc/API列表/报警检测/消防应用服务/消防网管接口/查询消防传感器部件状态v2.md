# 查询消防传感器部件状态v2

> 接口说明
> 分页查询消防传感器部件状态。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/fpnms/v2/sensor/status/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/fpnms/v2/sensor/status/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`includeSubNode`|`body`|`integer`|是|是否包含下级 (1-包含、0-不包含)|
|`regionIndexCode`|`body`|`string`|是|所属区域indexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`cn`|`body`|`string`|否|传感器名称|
|`channelType`|`body`|`integer`|否|传感器类型，详见附录A.57 消防传感器大类型|
|`unitStatus`|`body`|`integer`|否|部件状态，0：未注册 1：正常 2: 修复中 3：异常状态（修复中状态目前没用到，未注册时设备没有在离线状态） 空-所有状态|
|`channelIndexCode`|`body`|`string`|否|传感器编号|
|`pageNo`|`body`|`integer`|是|目标页码，默认从1开始，表示第一页|
|`pageSize`|`body`|`integer`|是|每页记录数，每页最大数量1000,默认20|

## 请求参数示例
```json
{
  "includeSubNode": 1,
  "regionIndexCode": "04c60702-f555-4887-9ae8-dc1186a7537a",
  "cn": "园区传感器",
  "channelType": 1020008,
  "unitStatus": 1,
  "channelIndexCode": "124afdva41a753",
  "pageNo": 1,
  "pageSize": 20
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.2 消防错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageSize`|`body`|`number`|否|每页条数|
|`+total`|`body`|`number`|否|结果总数|
|`+page`|`body`|`number`|否|页码|
|`+totalPage`|`body`|`number`|否|总页数|
|`+rows`|`body`|`object[]`|否|返回结果集|
|`++cn`|`body`|`string`|否|传感器名称|
|`++indexCode`|`body`|`string`|否|传感器编码|
|`++regionIndexCode`|`body`|`string`|否|所属区域|
|`++regionName`|`body`|`string`|否|区域目录名称，“\”分隔|
|`++channelIndexCode`|`body`|`string`|否|传感器编号|
|`++channelType`|`body`|`number`|否|传感器大类型，详见附录A.57 消防传感器大类型|
|`++channelSubType`|`body`|`string`|否|传感器小类型，详见附录A.56 消防传感器小类型|
|`++unitStatus`|`body`|`number`|否|部件状态，0：未注册 1：正常 2: 修复中 3：异常状态（修复中状态目前没用到，未注册时设备没有在离线状态） 空-所有状态|
|`++collectTime`|`body`|`string`|否|更新时间，IOS8601格式|
|`++parentName`|`body`|`string`|否|父资源名称|
|`++faultType`|`body`|`string`|否|故障类型，","号分隔，详见附录A.89 消防故障类型|
|`++faultName`|`body`|`string`|否|故障类型名称，”,“号分隔|
|`++faultNum`|`body`|`string`|否|故障次数|
