# 根据事件源编号查询监控点信息

> 接口说明
> 根据事件源的防区通道编号，查询所属设备的监控点信息，包括监控点id和名称。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/escpms/v1/channel/camera`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/escpms/v1/channel/camera
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCode`|`body`|`string`|是|事件源通道编号，，紧急报警事件日志查询接口获取返回参数srcIndex|

## 请求参数示例
```json
{
  "indexCode": "242b84ceae2f483cace50df7ed7acaab"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考 附录E.4.4 紧急报警错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据|
|`+deviceIndexCode`|`body`|`string`|否|所属设备编号|
|`+deviceName`|`body`|`string`|否|所属设备名称|
|`+cameras`|`body`|`object[]`|否|监控点信息|
|`++indexCode`|`body`|`string`|否|通道编号|
|`++channelName`|`body`|`string`|否|通道名称|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "deviceIndexCode": "a78f02hc30uy49uc293arc2083c",
    "deviceName": "设备1",
    "cameras": [
      {
        "indexCode": "cr80w0c90290c49029034c32rcatc",
        "channelName": "通道"
      }
    ]
  }
}
```
