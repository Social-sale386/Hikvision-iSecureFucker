# 紧急报警事件日志查询

> 接口说明
> 用户可以使用该接口查询事件日志，该接口提供条件筛选功能， 包括开始时间、结束时间、事件源名称、事件源类型等。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/escpms/v1/eventLogs/searches`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/escpms/v1/eventLogs/searches
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码，0 < pageNo|
|`pageSize`|`body`|`integer`|是|每页数据数，0 < pageSize≤1000|
|`eventType`|`body`|`integer`|否|10进制事件码，紧急报警：327687|
|`srcName`|`body`|`string`|否|事件源名称|
|`startTime`|`body`|`string`|否|开始时间，采用ISO8601时间格式|
|`endTime`|`body`|`string`|否|结束时间，采用ISO8601时间格式|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 800,
  "eventType": 327687,
  "srcName": "紧急报警柱",
  "startTime": "2018-04-02T08:00:00.000+08:00",
  "endTime": "2018-04-02T08:00:00.000+08:00"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考 附录E.4.4 紧急报警错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+hasNextPage`|`body`|`boolean`|否|是否有下一页|
|`+list`|`body`|`object[]`|否|事件对象列表|
|`++eventId`|`body`|`string`|否|事件唯一标志|
|`++startTime`|`body`|`string`|否|开始时间，采用ISO8601时间格式|
|`++endTime`|`body`|`string`|否|结束时间，采用ISO8601时间格式|
|`++regionName`|`body`|`string`|否|事件源所属区域目录，多级以"/"隔开|
|`++srcName`|`body`|`string`|否|事件源名称|
|`++srcType`|`body`|`string`|否|事件源资源类型，详见附录A.84 报警设备类型|
|`++eventType`|`body`|`number`|否|事件类型，327687：紧急报警|
|`++srcIndex`|`body`|`string`|否|事件源indexCode|
|`++srcParentIndex`|`body`|`string`|否|事件源所属设备id|
|`++status`|`body`|`number`|否|事件状态，0-瞬时 1-开始 2-停止 3-事件脉冲 4-事件联动结果更新|
