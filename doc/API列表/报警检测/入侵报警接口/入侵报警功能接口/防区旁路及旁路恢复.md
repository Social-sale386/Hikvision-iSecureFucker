# 防区旁路及旁路恢复

> 接口说明
> 1、设置防区旁路、旁路恢复状态；
> 2、单次操作最大支持10个通道；
> 3、由于设备原因同一个防区多次操作可能会失败。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/scpms/v1/defence/status/issue`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v1/defence/status/issue
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`defenceList`|`body`|`object[]`|是|防区操作信息列表，操作命令按顺序下发。|
|`+defenceIndexCode`|`Body`|`string`|是|防区indexCode，可从查询入侵报警主机通道列表v2获取返回值indexCode|
|`+status`|`Body`|`number`|是|防区操作，0-旁路恢复，1-旁路|

## 请求参数示例
```json
{
  "defenceList": [
    {
      "defenceIndexCode": "5fa7cdca292249e3a541a6e794f5707f",
      "status": 0
    }
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回结果|
|`+total`|`body`|`number`|否|数据记录总数|
|`+list`|`body`|`object[]`|否|返回数据列表|
|`++indexCode`|`body`|`string`|否|子系统编码|
|`++success`|`body`|`boolean`|否|操作结果，true – 成功， false - 失败|
|`++errorCode`|`body`|`string`|否|错误码，参考附录E.4.1 入侵报警错误码|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "list": [
      {
        "indexCode": "5fa7cdca292249e3a541a6e794f5707f",
        "success": true,
        "errorCode": "0"
      }
    ]
  }
}
```
