# 查询子系统状态

> 接口说明
> 在查询到子系统信息的情况下，可以依据子系统indexCode作为查询参数，调用此接口来获取对应的子系统状态信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/scpms/v1/subSys/status`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v1/subSys/status
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`subSysIndexCodes`|`body`|`string[]`|是|子系统编码列表|

## 请求参数示例
```json
{
  "subSysIndexCodes": [
    "514277b5522446cc92962f7aaa886c21"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|错误消息|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+list`|`body`|`object[]`|否|对象列表|
|`++subSystemIndexCode`|`body`|`string`|否|子系统编码|
|`++status`|`body`|`number`|否|子系统状态，-1：未知，0：撤防，1：布防|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "list": [
      {
        "subSystemIndexCode": "514277b5522446cc92962f7aaa886c21",
        "status": 1
      }
    ]
  }
}
```
