# 查询子系统关联防区

> 接口说明
> 用户可以根据子系统编码查询子系统关联的防区列表。该接口会对用户进行权限校验。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/scpms/v1/subsystem/defences/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v1/subsystem/defences/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`subSysIndexCodes`|`body`|`string[]`|是|子系统编码列表|

## 请求参数示例
```json
{
  "subSysIndexCodes": [
    "5fa7cdca292249e3a541a6e794f5707f"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回结果|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+list`|`body`|`object[]`|否|返回对象列表|
|`++success`|`body`|`boolean`|否|操作结果|
|`++errorCode`|`body`|`string`|否|错误码,参考附录E.4.1 入侵报警错误码|
|`++subSystemIndexCode`|`body`|`string`|否|子系统编码|
|`++defenceIndexCodes`|`body`|`string[]`|否|防区编码列表|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "list": [
      {
        "success": true,
        "errorCode": "0",
        "subSystemIndexCode": "5fa7cdca292249e3a541a6e794f5707f",
        "defenceIndexCodes": [
          "1v51346v654yutuybeybe"
        ]
      }
    ]
  }
}
```
