# 入侵报警事件日志查询v2

> 接口说明
> 用户可以根据事件源、起止时间、事件类型以及事件名称组装成筛选条件，查询入侵报警的历史事件，事件类型分成两大类：
> 子系统：布防，撤防，消警。
> 防区：旁路，旁路恢复，报警。

## 基本信息
|项目|值|
|---|---|
|接口版本|v2|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/scpms/v2/eventLogs/searches`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v2/eventLogs/searches
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|当前页码，大于0|
|`pageSize`|`body`|`integer`|是|每页数据记录数，最大值不超过1000|
|`srcType`|`body`|`string`|否|事件源类型(子系统:subSys,防区:defence)|
|`startTime`|`body`|`string`|否|开始时间，ISO8601标准格式|
|`endTime`|`body`|`string`|否|结束时间，ISO8601标准格式|
|`srcName`|`body`|`string`|否|事件源名称，模糊匹配|
|`eventType`|`body`|`integer`|否|事件类型，参考[附录D.4.1.1 入侵报警事件类型](/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录D%20事件列表-附录D4 报警检测事件-附录D4.1 入侵报警事件#d6ed20fa)|

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 15,
  "srcType": "defence",
  "startTime": "2018-05-01T00:00:00.000+08:00",
  "endTime": "2018-05-01T20:00:00.000+08:00",
  "srcName": "admin",
  "eventType": 327681
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+pageNo`|`body`|`number`|否|当前页码|
|`+pageSize`|`body`|`number`|否|每页记录总数|
|`+list`|`body`|`object[]`|否|对象列表|
|`++eventId`|`body`|`string`|否|主键|
|`++startTime`|`body`|`string`|否|开始时间|
|`++endTime`|`body`|`string`|否|结束时间|
|`++eventType`|`body`|`number`|否|事件类型，参考[附录D.4.1.1 入侵报警事件类型](/docs/docId?productId=5c67f1e2f05948198c909700&version=/29c78ef52ca842c7933bd2b8e051e9d0&tagPath=附录-附录D%20事件列表-附录D4 报警检测事件-附录D4.1 入侵报警事件#d6ed20fa)|
|`++srcIndex`|`body`|`string`|否|事件源indexCode|
|`++srcName`|`body`|`string`|否|事件源名称|
|`++srcType`|`body`|`string`|否|事件源资源类型，参考详见附录A.3 资源类型/资源权限码|
|`++regionId`|`body`|`string`|否|事件发生区域|
|`++status`|`body`|`number`|否|事件状态，0-瞬时 1-开始 2-停止 3-事件脉冲 4-事件联动结果更新|
