# 子系统消警

> 接口说明
> 当防区发生报警后，用户通过该接口可实现对子系统进行消警。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/scpms/v1/subsystem/alarm/eliminate`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v1/subsystem/alarm/eliminate
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`subSysIndexCodes`|`body`|`string[]`|是|子系统编码列表|

## 请求参数示例
```json
{
  "subSysIndexCodes": [
    "5fa7cdca292249e3a541a6e794f5707f"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回结果|
|`+total`|`body`|`number`|否|数据记录总数|
|`+list`|`body`|`object[]`|否|返回数据列表|
|`++indexCode`|`body`|`string`|否|子系统编码|
|`++success`|`body`|`boolean`|否|操作结果，true – 成功， false - 失败|
|`++errorCode`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，，参考附录E.4.1 入侵报警错误码|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "list": [
      {
        "indexCode": "5fa7cdca292249e3a541a6e794f5707f",
        "success": true,
        "errorCode": "0"
      }
    ]
  }
}
```
