# 防区添加到区域

> 接口说明
> 1、防区添加到区域；
> 2、单次操作最大支持256个通道;
> 3、同样支持移动已加入区域的防区；
> 4、仅支持修改入侵报警管理的设备（comId=scpms），暂不支持修改简单设备管理的第三方厂商设备。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/scpms/v1/defence/region/bindings`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v1/defence/region/bindings
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`regionIndexCode`|`body`|`string`|是|区域IndexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`defenceIndexCodes`|`body`|`string[]`|是|防区编号列表|

## 请求参数示例
```json
{
  "regionIndexCode": "root000000",
  "defenceIndexCodes": [
    "242b84ceae2f483cace50df7ed7acaab"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回结果|
|`+total`|`body`|`number`|否|数据记录总数|
|`+list`|`body`|`object[]`|否|返回数据列表|
|`++regionIndexCode`|`body`|`string`|否|区域编号|
|`++defenceIndexCode`|`body`|`string`|否|防区编号|
|`++success`|`body`|`boolean`|否|操作结果，true – 成功， false - 失败|
|`++errorCode`|`body`|`string`|否|错误码，参考附录E.4.1 入侵报警错误码|
|`++msg`|`body`|`string`|否|返回描述|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "error",
  "data": {
    "total": 100,
    "list": [
      {
        "regionIndexCode": "1123",
        "defenceIndexCode": "a4049c9535764f74b36cfdaccdf5e7b0",
        "success": true,
        "errorCode": "0",
        "msg": "success"
      }
    ]
  }
}
```
