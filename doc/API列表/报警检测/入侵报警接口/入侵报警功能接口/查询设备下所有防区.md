# 查询设备下所有防区

> 接口说明
> 1、查询设备下所有防区；
> 2、parentChannelIndex为空，说明防区未关联子系统，该防区不支持添加到区域下，必须关联上子系统后才能添加到区域；
> 3、regionIndexCode为空，说明该防区未关联到区域，防区功能受限，不会记录事件及实时变更状态。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/scpms/v1/iasDevice/defence`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v1/iasDevice/defence
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`deviceIndexCode`|`body`|`string`|是|设备编码，可从查询入侵报警主机列表v2 获取返回值IndexCode|

## 请求参数示例
```json
{
  "deviceIndexCode": "242b84ceae2f483cace50df7ed7acaab"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|总记录数|
|`+list`|`body`|`object[]`|否|防区信息列表|
|`++defenceIndexCode`|`body`|`string`|否|防区indexCode|
|`++defenceName`|`body`|`string`|否|防区名称|
|`++defenceNo`|`body`|`string`|否|防区通道号|
|`++subSystemIndexCode`|`body`|`string`|否|所属子系统indexCode|
|`++regionIndexCode`|`body`|`string`|否|所属区域indexCode|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 100,
    "list": [
      {
        "defenceIndexCode": "242b84ceae2f483cace50df7ed7acaab",
        "defenceName": "防区1",
        "defenceNo": "1",
        "subSystemIndexCode": "242b84ceae2f483cace50da7ed7acaab",
        "regionIndexCode": "242b84ceae2f483cace5aaa7ed7acaab"
      }
    ]
  }
}
```
