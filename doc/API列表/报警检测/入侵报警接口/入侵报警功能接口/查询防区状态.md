# 查询防区状态

> 接口说明
> 在查询到防区信息的情况下，可以依据防区indexCode作为查询参数，调用此接口来获取对应的防区状态信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.2及以上版本|
|接口地址|`/api/scpms/v1/defence/status`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/scpms/v1/defence/status
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`defenceIndexCodes`|`body`|`string[]`|是|防区编码列表|

## 请求参数示例
```json
{
  "defenceIndexCodes": [
    "1a3220227d27424296aCCe36d4bfeefa"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.1 入侵报警错误码|
|`msg`|`body`|`string`|否|错误描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|查询数据记录总数|
|`+list`|`body`|`object[]`|否|对象列表|
|`++defenceIndexCode`|`body`|`string`|否|防区编码|
|`++status`|`body`|`number`|否|防区状态，未知：-1，离线：0，正常：1，故障：2，报警：3，旁路：4|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "total": 1,
    "list": [
      {
        "defenceIndexCode": "1a3220227d27424296aCCe36d4bfeefa",
        "status": 1
      }
    ]
  }
}
```
