# 分页查询动环传感器

> 接口说明
> 支持分页查询指定区域下的传感器，可以选择是否包含下级区域

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pems/v1/monitor/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pems/v1/monitor/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`pageNo`|`body`|`integer`|是|页码， ( 0 , ~ )|
|`pageSize`|`body`|`integer`|是|每页大小， ( 0 , 1000 ]|
|`regionIndexCode`|`body`|`string`|否|区域indexCode，根据查询区域列表v2接口获取返回报文中的indexCode字段|
|`isIncludeSonRegion`|`body`|`boolean`|否|是否包含下级区域，默认为true|
|`name`|`body`|`string`|否|传感器名称|
|`type`|`body`|`string`|否|传感器类型，ups：UPS|
|`maxSensorCount`|`body`|`integer`|否|传感器下最多返回的环境量数量，超过传感器下的最多环境量数量时，会返回全部环境量|

补充说明:
- coulombmeter：电量表传感器
- toxicGasDetection：有毒气体传感器
- windSpeed：风速传感器
- tempHum：温湿度传感器
- airconditioner：空调系统
- transTempMeter：变压器温显表
- waterLevel：水位传感器
- envAcquistion：环境采集仪
- dustDetector：扬尘噪声传感器
- generalTransducer：通用传感器

## 请求参数示例
```json
{
  "pageNo": 1,
  "pageSize": 10,
  "regionIndexCode": "764e384b2df64f3eae92c3bbaa0127c2",
  "isIncludeSonRegion": true,
  "name": "传感器1",
  "type": "ups",
  "maxSensorCount": "10"
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.3 动环错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+list`|`body`|`object[]`|否|查询结果集|
|`++indexCode`|`body`|`string`|否|传感器编码|
|`++name`|`body`|`string`|否|名称|
|`++transducerType`|`body`|`string`|否|传感器类型|
|`++regionIndexCode`|`body`|`string`|否|所属区域编码|
|`++parentIndexCode`|`body`|`string`|否|父设备编号|
|`++channelType`|`body`|`string`|否|通道类型|
|`++channelNo`|`body`|`number`|否|通道号|
|`++addressNo`|`body`|`number`|否|通信地址号|
|`++altitude`|`body`|`number`|否|海拔|
|`++latitude`|`body`|`number`|否|纬度|
|`++longitude`|`body`|`number`|否|经度|
|`++parentIndexCode`|`body`|`string`|否|父设备编号|
|`++regionIndexCode`|`body`|`string`|否|所属区域编码|
|`++sensorList`|`body`|`object[]`|否|该传感器下包含的环境量列表|
|`+++alarmHigh`|`body`|`number`|否|偏高告警值|
|`+++alarmLow`|`body`|`number`|否|偏低告警值|
|`+++indexCode`|`body`|`string`|否|环境量编码|
|`+++isTopUp`|`body`|`boolean`|否|是否置顶|
|`+++latestIsAlarm`|`body`|`boolean`|否|最新数据是否告警|
|`+++latestTime`|`body`|`string`|否|最新数据上报时间|
|`+++latestValue`|`body`|`number`|否|最新数据|
|`+++measureHigh`|`body`|`number`|否|量程上限|
|`+++measureLow`|`body`|`number`|否|量程下限|
|`+++name`|`body`|`string`|否|环境量名称|
|`+++parentName`|`body`|`string`|否|环境量所属传感器名称|
|`+++sensitive`|`body`|`number`|否|灵敏度|
|`+++sensorShowValue`|`body`|`string`|否|环境量显示数据|
|`+++type`|`body`|`string`|否|环境量类型,temp：温度|
|`+++typeName`|`body`|`string`|否|环境量类型展示名称|
|`+++unit`|`body`|`string`|否|单位|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|页大小|
|`+total`|`body`|`number`|否|查询总数|
|`+totalPage`|`body`|`number`|否|总页数|

补充说明:
- humidity：湿度
- wind：风速
- gas：气体浓度
- voltage：电压
- current：电流
- frequency：频率
- power：功率
- flux：流量
- batteryBackup：电池备份时间
- lf：负载率
- pf：功率因数
- precipitation：遥控点
- dust：降水量
- noise：扬尘
- wl：噪声
- generalSensor：水位
- alarmPoint：通用环境量
- windDirection：风向
- pressure：压力
