# 增量获取动环设备数据

> 接口说明
> 根据资源类型、时间段增量获取资源，包含已删除数据。资源密码以*代替，其中开始日期与结束日期的时间差必须在1-48小时内。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.4及以上版本|
|接口地址|`/api/resource/v1/peDevice/timeRange`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/resource/v1/peDevice/timeRange
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`startTime`|`body`|`string`|是|开始日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`endTime`|`body`|`string`|否|截止日期,格式：yyyy-MM-dd’T’HH:mm:ss.SSS|
|`pageNo`|`body`|`integer`|是|页码,pageNo≥1|
|`pageSize`|`body`|`integer`|是|页面大小,0<pageSize≤1000|

## 请求参数示例
```json
{
  "startTime": "2019-08-06T14:01:17.839+0800",
  "endTime": "2019-08-06T16:01:17.839+0800",
  "pageNo": 1,
  "pageSize": 1000
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.1 资源目录错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+total`|`body`|`number`|否|记录总数|
|`+list`|`body`|`object[]`|否|返回数据|
|`++resourceType`|`body`|`string`|否|资源类型，peDevice：动环设备|
|`++indexCode`|`body`|`string`|否|资源唯一标志|
|`++name`|`body`|`string`|否|资源名称|
|`++belongIndexCode`|`body`|`string`|否|所属接入服务编号|
|`++regionIndexCode`|`body`|`string`|否|所属区域编号|
|`++ip`|`body`|`string`|否|设备IP|
|`++port`|`body`|`string`|否|设备端口|
|`++deviceCode`|`body`|`string`|否|主动设备编号|
|`++deviceKey`|`body`|`string`|否|设备驱动|
|`++deviceKeyVersion`|`body`|`string`|否|设备驱动编号|
|`++deviceModel`|`body`|`string`|否|设备型号|
|`++deviceType`|`body`|`string`|否|设备系列|
|`++devSerialNum`|`body`|`string`|否|设备的序列号|
|`++capability`|`body`|`string`|否|设备能力，详见附录A.44 设备能力集|
|`++netZoneId`|`body`|`string`|否|所属网域ID|
|`++treatyType`|`body`|`string`|否|设备接入协议，详见附录A.6 编码设备接入协议|
|`++extend`|`body`|`object`|否|扩展信息,KV展开的json字段，用于业务扩展|
|`++createTime`|`body`|`string`|否|创建时间，ISO8601标准|
|`++updateTime`|`body`|`string`|否|更新时间，ISO8601标准|
|`++status`|`body`|`number`|否|状态小于0则资源已删除|
|`++disOrder`|`body`|`number`|否|数据在界面上显示的顺序|
