# 动环环境量布防

> 接口说明
> 布防环境量，如果环境量原先为撤防状态，会修改为布防，否则无影响。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pems/v1/monitor/sensor/layout`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pems/v1/monitor/sensor/layout
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`list`|`body`|`string[]`|是|需要布防的环境量编号列表|

## 请求参数示例
```json
{
  "list": [
    "18956215bf6643d4a0fa4287de0bedaa"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.3 动环错误码|
|`msg`|`body`|`string`|否|返回描述-记录接口执行情况说明信息|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success"
}
```
