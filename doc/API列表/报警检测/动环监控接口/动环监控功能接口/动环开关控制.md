# 动环开关控制

> 接口说明
> 简述：实现指定开关的控制, 实现开关的开启和关闭
> 支持：支持批量控制开关

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.5及以上版本|
|接口地址|`/api/pems/v1/monitor/pemsIoOut/control`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pems/v1/monitor/pemsIoOut/control
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`indexCodes`|`body`|`string`|是|开关量编码，由开关量编码和所属设备编码组成通过下划线分隔，多个以逗号分隔，需要布防的报警输入编号，调用查询动环开关列表接口获取返回参数indexCode和parentIndexCode|
|`type`|`body`|`integer`|是|控制类型:0关 1开|

## 请求参数示例
```json
{
  "indexCodes": "b4eec988d1384a2e89839ae2b6cf262b_927ad6552b19482ba428aa883cd9dad5,7d113eb5ca2c48a795e2452e1eb211d5_927ad6552b19482ba428aa883cd9dad5",
  "type": 1
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.3 动环错误码|
|`data`|`body`|`object`|否|返回数据|
|`+failCount`|`body`|`number`|否|控制失败数|
|`+failList`|`body`|`array`|否|控制失败开关编码|
|`+failNameList`|`body`|`array`|否|控制失败开关名称|
|`+successCount`|`body`|`number`|否|控制成功数据|
|`+unPermissionCount`|`body`|`number`|否|无权限开关数|
|`+unPermissionList`|`body`|`array`|否|无权限开关编码列表|
|`+unPermissionNameList`|`body`|`array`|否|无权限开关名称列表|
|`msg`|`body`|`string`|否|返回说明|

## 返回参数示例
```json
{
  "code": "0",
  "data": {
    "failCount": 2,
    "failList": [
      "b4eec988d1384a2e89839ae2b6cf262b",
      "7d113eb5ca2c48a795e2452e1eb211d5"
    ],
    "failNameList": [],
    "successCount": 0,
    "unPermissionCount": 0,
    "unPermissionList": [],
    "unPermissionNameList": []
  },
  "msg": "success"
}
```
