# 获取传感器布防状态

> 接口说明
> 获取传感器布撤防状态

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/pems/v1/{param}/transducer/state`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/pems/v1/{param}/transducer/state
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`list`|`body`|`string[]`|是|需要查询布撤防状态的资源列表，调用分页查询动环环境量接口获取返回参数indexCode|

## 请求参数示例
```json
{
  "list": [
    "8c80fb6ef41047d6b559e71991bc33f9"
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.4.3 动环错误码|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+a7b6c07d7a914d858eced24decc93610`|`body`|`number`|否|key是资源编号，值是布撤防状态,布撤防状态: 0-未布防，1-布防 2-部分布防（传感器下的环境量若存在布防和撤防两种状态，则该传感器为部分布防）|
|`+660bc1d7ced54abCC349ecf6587e631e`|`body`|`number`|否|key是资源编号，值是布撤防状态,布撤防状态: 0-未布防，1-布防 2-部分布防（传感器下的环境量若存在布防和撤防两种状态，则该传感器为部分布防）|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success",
  "data": {
    "a7b6c07d7a914d858eced24decc93610": 1,
    "660bc1d7ced54abCC349ecf6587e631e": 1
  }
}
```
