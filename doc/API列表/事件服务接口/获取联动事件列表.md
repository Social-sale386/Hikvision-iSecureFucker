# 获取联动事件列表

> 接口说明
> 通过输入查询条件获取事件列表，通过分页的形式获取事件日志大概描述信息。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.3及以上版本|
|接口地址|`/api/els/v1/events/search`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/els/v1/events/search
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`eventRuleId`|`body`|`string`|否|事件规则id|
|`ability`|`body`|`string`|否|事件分类，详见附录A.62 事件分类|
|`regionIndexCode`|`body`|`string`|否|区域编号|
|`locationIndexCodes`|`body`|`string[]`|否|所属位置，详见附录A.81 安装位置|
|`resName`|`body`|`string`|否|事件源名称|
|`resIndexCodes`|`body`|`string[]`|否|事件源编号|
|`resTypes`|`body`|`string[]`|否|事件源类型，详见附录A.63 事件源类型|
|`eventType`|`body`|`string`|否|事件类型，参考附录D|
|`eventLevels`|`body`|`string[]`|否|事件等级，1-低，2-中，3-高|
|`remark`|`body`|`string`|否|处理意见|
|`handleStatus`|`body`|`integer`|否|处理状态，0-未处理，1-已处理|
|`startTime`|`body`|`string`|是|开始时间,ISO8601时间|
|`endTime`|`body`|`string`|是|结束时间,ISO8601时间|
|`pageSize`|`body`|`integer`|是|分页大小|
|`pageNo`|`body`|`integer`|是|页码|

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码，0 – 成功，其他- 失败，参考附录E.other.2 事件订阅返回码说明|
|`msg`|`body`|`string`|否|返回描述|
|`data`|`body`|`object`|否|返回数据|
|`+pageNo`|`body`|`number`|否|页码|
|`+pageSize`|`body`|`number`|否|每页大小|
|`+total`|`body`|`number`|否|结果总数|
|`+list`|`body`|`object[]`|否|查询事件数据列表|
|`++eventLevel`|`body`|`number`|否|事件等级,1-低，2-中，3-高|
|`++eventLevelColor`|`body`|`string`|否|事件等级颜色|
|`++eventLevelValue`|`body`|`string`|否|事件等级value|
|`++eventRuleId`|`body`|`string`|否|事件规则id|
|`++eventTypeName`|`body`|`string`|否|事件类型名称|
|`++remark`|`body`|`string`|否|事件处理意见|
|`++handleStatus`|`body`|`number`|否|事件处理状态，0-未处理，1-已处理|
|`++id`|`body`|`string`|否|事件id|
|`++ruleName`|`body`|`string`|否|事件规则名称|
|`++startTime`|`body`|`string`|否|事件开始时间|
|`++endTime`|`body`|`string`|否|事件结束时间|
|`++eventLogSrcList`|`body`|`object[]`|否|事件记录列表|
|`+++ability`|`body`|`string`|否|事件分类，详见附录A.62 事件分类|
|`+++data`|`body`|`string`|否|事件扩展信息|
|`+++eventLogId`|`body`|`string`|否|事件id|
|`+++eventType`|`body`|`number`|否|事件类型，参考附录D|
|`+++eventTypeName`|`body`|`string`|否|事件类型名称|
|`+++extend1`|`body`|`string`|否|扩展字段1，暂不使用|
|`+++extend2`|`body`|`string`|否|扩展字段2，暂不使用|
|`+++id`|`body`|`string`|否|事件源id|
|`+++locationIndexCode`|`body`|`string`|否|所属位置，详见附录A.81 安装位置|
|`+++locationName`|`body`|`string`|否|所属位置名称|
|`+++regionIndexCode`|`body`|`string`|否|所属区域编号|
|`+++regionName`|`body`|`string`|否|所属区域名称|
|`+++resName`|`body`|`string`|否|事件源名称|
|`+++resIndexCode`|`body`|`string`|否|事件源编号|
|`+++resType`|`body`|`string`|否|事件源类型，详见附录A.63 事件源类型|
|`+++startTime`|`body`|`string`|否|事件开始时间|
