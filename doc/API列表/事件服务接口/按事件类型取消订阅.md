# 按事件类型取消订阅

> 接口说明
> 应用方指定事件类型取消订阅。

## 基本信息
|项目|值|
|---|---|
|接口版本|v1|
|适配版本|综合安防管理平台iSecure Center V1.0及以上版本|
|接口地址|`/api/eventService/v1/eventUnSubscriptionByEventTypes`|
|请求方法|`POST`|
|数据提交方式|application/json|

## 快速调用
```text
POST /api/eventService/v1/eventUnSubscriptionByEventTypes
```

## 请求参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`eventTypes`|`body`|`integer[]`|是|事件类型，参考附录D 事件列表|

## 请求参数示例
```json
{
  "eventTypes": [
    131329
  ]
}
```

## 返回参数
|参数|位置|类型|必填|说明|
|---|---|---|---|---|
|`code`|`body`|`string`|否|返回码， 0表示订阅成功，其它参考附录E.other.2 事件订阅返回码说明|
|`msg`|`body`|`string`|否|返回描述|

## 返回参数示例
```json
{
  "code": "0",
  "msg": "success"
}
```
